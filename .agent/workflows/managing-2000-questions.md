## 📋 Пример Скрипта "Всё за раз"

Создай `process-all-topics.sh`:

```bash
#!/bin/bash

TOPICS=("drivers" "documentation" "signs" "signals" "speed" "priority" "overtaking" "parking" "loading" "mechanics")

for i in "${!TOPICS[@]}"; do
  TOPIC_NUM=$((i + 1))
  TOPIC_NAME="${TOPICS[$i]}"
  
  echo "======================================"
  echo "Обработка Темы $TOPIC_NUM: $TOPIC_NAME"
  echo "======================================"
  
  # Объединяем батчи
  node scripts/merge-json.js \
    "data/topic-${TOPIC_NUM}-all.json" \
    data/raw/topic-${TOPIC_NUM}-*.json
  
  # Запускаем конвейер
  node scripts/pipeline.js \
    "data/topic-${TOPIC_NUM}-all.json" \
    --auto
  
  echo "✅ Тема $TOPIC_NUM завершена"
  echo ""
  
  # Пауза между темами (для rate limits)
  sleep 60
done

echo "🎉 Все темы обработаны!"
```

Запуск:
```bash
chmod +x process-all-topics.sh
./process-all-topics.sh
```
