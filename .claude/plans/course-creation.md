## Формат каждого материала

Каждый JSON файл:
```json
{
  "id": "subtopic-{code}",
  "code": "{code}",
  "topic_id": "topic-{N}",
  "title_ru": "...",
  "title_es": "...",
  "title_en": "...",
  "content_ru": "<div class=\"material-content-wrapper\">...</div>",
  "content_es": "<div class=\"material-content-wrapper\">...</div>",
  "content_en": "<div class=\"material-content-wrapper\">...</div>",
  "images": [],
  "order": N,
  "source_pdf": "https://www.dgt.es/..."
}
```

### Вау-элементы в HTML контенте
- **Аккордеоны** (`.hb-accordion-item`) — раскрывающиеся блоки для деталей
- **Info/Tip/Warning boxes** — цветные боксы для важной информации
- **Таблицы** (`.data-table`) — сравнительные таблицы
- **Signals grid** (`.signals-grid` + `.signal-card`) — карточки сигналов
- **Question blocks** — блоки самопроверки
- **Mnemonic boxes** — мнемонические правила для запоминания
- **Styled lists** — структурированные списки с иконками

### Качество контента
- Основан на официальном руководстве DGT Испании
- Адаптирован для русскоязычных студентов с пояснениями отличий от ПДД РФ
- Испанский — оригинальная терминология DGT
- Английский — для международных студентов
- Каждый материал 1500-3000 слов с практическими примерами
