# 🧪 Руководство по тестированию AdsGram рекламы

## 🎯 Где тестировать рекламу

### Вариант 1: DailyBonus (Восстановление Streak)

**Путь:** Daily Bonus → Восстановить streak через рекламу

1. Откройте приложение в Telegram
2. Перейдите в раздел **"Daily Bonus"** (Ежедневный бонус)
3. Если у вас сломан streak - появится кнопка **"Восстановить серию"**
4. В диалоге восстановления нажмите **"Смотреть рекламу (бесплатно)"**
5. Откроется модалка с рекламой

**Что проверить:**
- ✅ Модалка открывается
- ✅ Показывается кнопка "Смотреть рекламу"
- ✅ После нажатия идет загрузка (2 секунды в тестовом режиме)
- ✅ После "просмотра" показывается награда
- ✅ Streak восстанавливается

---

### Вариант 2: Shop → Получить монеты бесплатно

**Путь:** Shop → Монеты → "Получить монеты бесплатно"

1. Откройте приложение в Telegram
2. Перейдите в **Shop** (Магазин)
3. Выберите вкладку **"Монеты"**
4. Найдите кнопку **"Получить монеты бесплатно"** или **"Смотреть"** (с иконкой Video)
5. Нажмите на неё
6. Откроется модалка с рекламой

**Что проверить:**
- ✅ Кнопка видна (только для не-Premium пользователей)
- ✅ Модалка открывается
- ✅ После просмотра начисляются **20 монет**
- ✅ Баланс обновляется
- ✅ Показывается уведомление "✨ Монеты получены!"

---

## 🔧 Как работает тестовый режим

**Текущий статус:** Тестовый режим включен (`TEST_MODE_CONFIG.enabled = true`)

### Что происходит в тестовом режиме:

1. **Загрузка рекламы:**
   - Симулируется задержка 2 секунды
   - Реальная реклама не загружается
   - Статус меняется на "готово"

2. **Показ рекламы:**
   - Симулируется просмотр (3 секунды)
   - Никакого реального видео не показывается
   - После "просмотра" вызывается callback

3. **Начисление награды:**
   - Вызывается Edge Function `ad-reward` с вашими данными
   - **Реально начисляются монеты** ✅
   - **Реально логируется транзакция** ✅
   - **Реально работает rate limiting** ✅

**Вывод:** В тестовом режиме вы можете проверить ВСЮ логику, кроме реального показа видео!

## 📋 Пошаговая инструкция тестирования

### Шаг 1: Подготовка

1. **Откройте приложение в Telegram:**
   - Убедитесь, что используете реальный Telegram аккаунт
   - Приложение должно быть задеплоено или доступно через tunnel

2. **Проверьте, что вы НЕ Premium:**
   - Premium пользователи не видят рекламу
   - Если нужно - временно отключите Premium

3. **Откройте DevTools (опционально):**
   - В браузере: F12 или Cmd+Option+I
   - В Telegram Desktop: View → Developer Tools

---

### Шаг 2: Тест в DailyBonus

1. Откройте **Daily Bonus**
2. Если streak сломан:
   - Нажмите "Восстановить серию"
   - Выберите "Смотреть рекламу (бесплатно)"
3. Если streak не сломан:
   - Можно временно сломать в базе для теста
   - Или подождать, пока сломается естественно

**Что увидите:**
- Модалка "Rewarded Ad"
- Кнопка "Смотреть рекламу"
- При нажатии - загрузка 2 секунды
- Затем "просмотр" 3 секунды (в тестовом режиме)
- Показывается награда "Серию восстановлено!"
- Модалка закрывается

**В консоли (если открыта):**
```
[useRewardedAd] Loading ad...
[useRewardedAd] Ad loaded
[useRewardedAd] Showing ad...
[RewardedAdModal] Ad watched successfully
```

### Шаг 3: Тест в Shop

1. Откройте **Shop** → **Монеты**
2. Найдите кнопку **"Смотреть"** с иконкой Video
3. Нажмите на неё

**Что увидите:**
- Модалка открывается
- Кнопка "Смотреть рекламу"
- После просмотра:
  - Показывается "+20 монет"
  - Баланс увеличивается на 20
  - Toast: "✨ Монеты получены!"
  - Confetti анимация (если включена)

**Проверьте баланс:**
- До просмотра: запомните количество монет
- После просмотра: должно быть +20 монет

---

### Шаг 4: Проверка в базе данных

**Проверьте транзакции:**

```sql
SELECT * FROM transactions 
WHERE transaction_type = 'coins_earned_ad' 
ORDER BY created_at DESC 
LIMIT 5;
```

**Должно быть:**
- `transaction_type`: `coins_earned_ad`
- `amount`: `20` (или другое значение)
- `metadata`: JSON с данными о рекламе
- `user_id`: ваш profile ID

**Проверьте rate limiting:**

```sql
SELECT COUNT(*) as views_last_hour
FROM transactions 
WHERE transaction_type = 'coins_earned_ad' 
  AND user_id = 'YOUR_PROFILE_ID'
  AND created_at > NOW() - INTERVAL '1 hour';
```

Должно быть максимум 10 просмотров за час.

- ❌ SDK AdsGram не вызывается

**Но это нормально!** После получения SDK и интеграции всё заработает.

## 🔍 Отладка проблем

### Проблема: Модалка не открывается

**Проверьте:**
1. Вы не Premium? (Premium скрывает рекламу)
2. В консоли есть ошибки?
3. `useRewardedAd` hook правильно импортирован?

### Проблема: Монеты не начисляются

**Проверьте:**
1. Логи Edge Function - есть ошибки?
2. Транзакции в базе - создаются ли?
3. Правильный profileId передается?

**В консоли:**
```javascript
// Проверьте, что hook вызывается
console.log('[Test] useRewardedAd hook:', useRewardedAd());
```

### Проблема: Rate limiting срабатывает сразу

**Причина:** Возможно, уже было 10 просмотров за час

**Решение:** Подождите час или сбросьте в базе (только для теста):
```sql
DELETE FROM transactions 
WHERE transaction_type = 'coins_earned_ad' 
  AND user_id = 'YOUR_PROFILE_ID'
  AND created_at > NOW() - INTERVAL '1 hour';
```
