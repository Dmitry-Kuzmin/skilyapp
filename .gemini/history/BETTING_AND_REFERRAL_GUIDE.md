# Руководство по системам ставок и рефералов

## 🎰 Система ставок в дуэлях

### Как это работает

1. **Создание дуэли со ставкой**:
   - Откройте `/games/duel`
   - Выберите количество вопросов
   - Выберите тип ставки:
     - **Без ставки** - бесплатная дуэль
     - **Фиксированная** - 10, 50, 100 или 500 монет
     - **Произвольная** - введите любую сумму (до 10,000)
   - Нажмите "Создать за X монет"

2. **Присоединение к дуэли со ставкой**:
   - Введите код дуэли
   - Система проверит ваш баланс
   - Если монет достаточно - ставка спишется автоматически
   - Если недостаточно - увидите ошибку

3. **Результаты**:
   - **Победа**: Вы получаете `(ставка × 2) - 10%` комиссии
     - Пример: ставка 100 → банк 200 → выигрыш 180 (комиссия 20)
   - **Ничья**: Ставки переносятся на реванш
   - **Поражение**: Теряете свою ставку

### Формула выплат

```
Банк = Ставка хоста + Ставка оппонента
Комиссия = Банк × 10%
Выигрыш = Банк - Комиссия
```

### Пример

- Хост ставит: **100 монет**
- Оппонент ставит: **100 монет**
- Банк: **200 монет**
- Комиссия: **20 монет** (10%)
- Победитель получает: **180 монет**

### Реванш при ничьей

Если дуэль закончилась ничьей:
- Обе ставки (200 монет) сохраняются
- Появляется кнопка "Реванш"
- В реванше играют за накопленный банк
- Победитель реванша забирает всё

## 🎁 Реферальная система

### Как пригласить друга

1. **Получите свой код**:
   - Откройте `/referrals`
   - Скопируйте ваш реферальный код (например, `ABC123`)

2. **Поделитесь ссылкой** (в Telegram):
   - Нажмите "Поделиться"
   - Выберите чат или отправьте другу
   - Ссылка выглядит так: `https://t.me/ваш_бот/app?startapp=ref_ABC123`

3. **Друг регистрируется**:
   - Кликает на вашу ссылку
   - Приложение открывается с вашим реферальным кодом
   - При первом входе друг автоматически получает **+50 монет**

4. **Вы получаете награду**:
   - Когда друг заработает свои первые **50 монет**
   - Вам автоматически начисляется **+100 монет**
   - Без лимитов - приглашайте сколько угодно!

### Откуда друг может заработать 50 монет?

- Пройти тест
- Сыграть в игры
- Выполнить daily tasks
- Выиграть дуэль
- Streak бонусы

### Отслеживание рефералов

На странице `/referrals` вы видите:
- Ваш реферальный код
- Количество приглашенных друзей
- Заработано монет всего
- Список друзей с их прогрессом
- Статус наград (получена / в процессе)

## 🔗 Deep Links

### Формат ссылок

1. **Дуэль с кодом**:
   ```
   https://t.me/ваш_бот/app?startapp=duel_AB12
   ```
   - Автоматически присоединяет к дуэли `AB12`

2. **Реферальная ссылка**:
   ```
   https://t.me/ваш_бот/app?startapp=ref_ABC123
   ```
   - Регистрирует друга по коду `ABC123`
   - Начисляет +50 монет

### Настройка бота

Чтобы ссылки работали, в файле `.env` добавьте:

```bash
VITE_TELEGRAM_BOT_USERNAME=your_bot_name
```

Замените `your_bot_name` на имя вашего Telegram бота.

## 📊 База данных

### Новые таблицы

1. **duel_transactions** - история всех операций с монетами в дуэлях
   - Поля: `duel_id`, `user_id`, `amount`, `transaction_type`, `created_at`
   - Типы транзакций: `bet`, `win`, `refund`, `commission`, `rematch_carry`

2. **referrals** - связи реферер-реферал
   - Поля: `referrer_id`, `referred_id`, `reward_given`, `referred_earned_50`
   - Автоматическое отслеживание прогресса

### Новые функции

1. **process_duel_payout** - обработка выплат
2. **calculate_duel_commission** - расчет комиссии (10%)
3. **generate_referral_code** - генерация уникальных кодов
4. **check_referral_milestone** - проверка заработка 50 монет
5. **create_referral** - создание реферальной связи
