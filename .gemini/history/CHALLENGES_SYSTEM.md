# Система челленджей Duel Pass

## Обзор

Система челленджей автоматически отслеживает прогресс пользователя и награждает за выполнение различных действий.

## Типы челленджей

### 1. Ежедневные (Daily)
- **Длительность:** 1 день
- **Обновление:** Автоматически каждый день в 00:00 UTC
- **Примеры:**
  - Пройти 3 теста без ошибок
  - Набрать 100 SP за день
  - Пройти 5 тестов за день
  - Ответить на 50 вопросов за день

### 2. Еженедельные (Weekly)
- **Длительность:** 7 дней
- **Обновление:** Каждый понедельник
- **Примеры:**
  - Ответить на 150 вопросов за неделю
  - Сыграть 5 дуэлей за неделю

### 3. Сезонные (Season)
- **Длительность:** Весь сезон (30 дней)
- **Обновление:** Новые каждый сезон
- **Примеры:**
  - Пройти все игровые режимы
  - Держать streak 10 дней подряд

## Автоматическое отслеживание

Система автоматически отслеживает прогресс через Edge Function `season-challenges-track`:

- **Тесты:** `test_completed`, `test_perfect`
- **Дуэли:** `duel_win`, `duel_lose`
- **Входы:** `daily_login`
- **SP/Монеты:** Из metadata

## Автоматическое начисление наград

Когда челлендж завершен:
1. Edge Function `season-challenges-reward` вызывается автоматически
2. SP начисляются через `season-sp`, монеты — напрямую через `increment_profile_value` + запись в `transactions`
3. Прогресс отмечается как `reward_claimed: true`

## Автогенерация ежедневных челленджей

Edge Function `generate-daily-challenges`:
- Вызывается ежедневно (через cron или вручную)
- Деактивирует старые ежедневные челленджи
- Создает 2 случайных челленджа из шаблонов
- Устанавливает даты начала и окончания

## Настройка cron для автогенерации

Рекомендуется настроить cron job для автоматического вызова:

```bash
# Каждый день в 00:00 UTC
0 0 * * * curl -X POST https://YOUR_PROJECT.supabase.co/functions/v1/generate-daily-challenges \
  -H "Authorization: Bearer YOUR_SERVICE_ROLE_KEY"
```

Или использовать Supabase Cron Jobs (если доступно).
