# ✅ UI в стиле ChatGPT + Оценки - ГОТОВО!

## Что сделано

### 1. 🎨 Стиль ChatGPT

**Header:**
- Компактный заголовок (иконка + текст в одну строку)
- Без подзаголовка
- Минималистично

**Сообщения пользователя:**
- Закруглённый bubble (rounded-2xl)
- Справа
- bg-muted/80
- Max 85% ширины

**Сообщения AI:**
- Без карточек/рамок
- Просто текст слева
- Иконка Bot в круге (rounded-full)
- Расстояние между сообщениями (space-y-6)

**Footer:**
- Input + Send + Close в одну строку
- Кнопка закрытия (✕) справа
- Компактно (h-10)

### 2. 👍👎 Оценка качества

**Под каждым AI ответом:**
- Две кнопки: 👍 (ThumbsUp) и 👎 (ThumbsDown)
- Компактные (h-7, px-2)
- После клика:
  - Кнопка заполняется (fill-current)
  - Disabled (нельзя изменить оценку)
  - Toast: "✓ Спасибо за оценку!"

**Что сохраняется:**
- Вопрос и ответ AI
- Оценка (1 или -1)
- User ID
- Тема вопроса
- Модель которая ответила
- Timestamp

### 3. 📊 Таблица для аналитики

```sql
CREATE TABLE ai_feedback (
  id UUID,
  user_id UUID,
  question TEXT,
  ai_response TEXT,
  rating SMALLINT, -- 1 = 👍, -1 = 👎
  topic_number INTEGER,
  model_used TEXT,
  created_at TIMESTAMPTZ
);
```
