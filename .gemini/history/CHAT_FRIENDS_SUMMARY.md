## ✅ Что реализовано

### 1. База данных
- ✅ Таблица `telegram_chat_members` для хранения участников групп
- ✅ Функция `upsert_chat_member()` для добавления/обновления участников
- ✅ Функция `get_chat_friends()` для получения друзей из групп
- ✅ Индексы для оптимизации

### 2. Автоматическая синхронизация
- ✅ Бот автоматически синхронизирует участников при получении сообщений из групп
- ✅ Обновляет `last_seen_at` при каждом взаимодействии
- ✅ Работает для групп и супергрупп

### 3. Edge Functions
- ✅ `duel-pass-leaderboard` обновлена (поддержка друзей из чатов)
- ✅ `telegram-chat-sync` создана (для ручной синхронизации)
- ✅ `telegram-bot` обновлён (автосинхронизация)

### 4. Лидерборд
- ✅ Фильтр "Друзья" теперь включает друзей из групп
- ✅ Объединяет все источники: referrals + дуэли + группы

---

## 🎯 Как это работает

### Сценарий:

1. **Пользователь A и B в одной Telegram группе**
2. **Оба используют приложение** (авторизованы через Telegram)
3. **Оба взаимодействуют с ботом в группе** (отправляют `/start` или любое сообщение)
4. **Бот автоматически сохраняет их в `telegram_chat_members`**
5. **В лидерборде они видят друг друга в фильтре "Друзья"**

### Логика:

```
Пользователь A → Группа X
Пользователь B → Группа X
→ A и B друзья (общая группа)
```

## 📊 Текущие источники друзей

1. **Реферальная система** (`referrals`)
   - Те, кого пригласил
   - Тот, кто пригласил

2. **Дуэли** (`duel_players`)
   - Те, с кем играл в завершённых дуэлях

3. **Telegram группы** (`telegram_chat_members`) ✨ НОВОЕ
   - Те, кто в тех же группах
   - Автоматически синхронизируется
