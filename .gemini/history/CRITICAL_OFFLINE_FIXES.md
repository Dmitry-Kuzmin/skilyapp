# Критические исправления для Offline режима

## 🔴 Проблемы

### 1. Service Worker отключен
- ❌ Service Worker закомментирован в `main.tsx` (строка 32: `&& false`)
- ❌ Нет кэширования данных теста
- ❌ Нет offline режима для тестов

### 2. Нет локального сохранения ответов
- ❌ Ответы хранятся только в state
- ❌ При обрыве связи ответы теряются
- ❌ Нет синхронизации при восстановлении связи

### 3. Нет прелоадинга следующего вопроса
- ⚠️ Прелоадится только первые 3 изображения
- ❌ Нет прелоадинга следующего вопроса при переключении
- ❌ Белые моргания при переключении

### 4. Нет Supabase Image Transformations
- ❌ Используется только `getPublicUrl()` без параметров
- ❌ На мобильные прилетают огромные изображения
- ❌ Тратится лишний трафик

## ✅ Решения

### 1. Включить и улучшить Service Worker
- ✅ Включить Service Worker
- ✅ Кэшировать данные теста (вопросы, ответы)
- ✅ Добавить offline режим для тестов

### 2. Локальное сохранение ответов
- ✅ Сохранять ответы в IndexedDB/localStorage
- ✅ Синхронизировать при восстановлении связи
- ✅ Показывать статус синхронизации

### 3. Прелоадинг следующего вопроса
- ✅ Прелоадить изображение следующего вопроса
- ✅ Кэшировать изображения в памяти
- ✅ Убрать белые моргания

### 4. Supabase Image Transformations
- ✅ Добавить параметры трансформации в `getImageUrl()`
- ✅ Автоматически определять размер экрана
- ✅ Оптимизировать для мобильных устройств
