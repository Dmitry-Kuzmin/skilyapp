# 🎯 Daily Bonus: 7-дневный цикл (Финальный план)

## ✅ Решение принято

**7-дневный циклический streak** — оптимальное решение для retention и UX.

---

## 🎨 Концепция системы

### Две системы streak:

1. **Недельный цикл (1-7 дней)** — для ежедневных наград
   - Повторяется каждую неделю
   - Награды: XP, монеты, бусты, бейджи
   - Психологически легкий цикл

2. **Общий streak (1, 2, 3... 365+)** — для престижа
   - Никогда не сбрасывается (только при пропуске)
   - Используется для титулов, рамок, рангов
   - Вехи: 30, 60, 90, 180, 365 дней

---

## 📊 Баланс наград (7-дневный цикл)

### Ежедневные награды:

| День | XP | Монеты | Boost | Badge | Описание |
|------|----|--------|-------|-------|----------|
| 1 | 10 | 0 | ❌ | ❌ | Первый шаг |
| 2 | 15 | 0 | ❌ | ❌ | Продолжаем |
| 3 | 20 | 0 | ❌ | ❌ | Набираем темп |
| 4 | 0 | 0 | ✅ | ❌ | Boost день! |
| 5 | 30 | 0 | ❌ | ❌ | Почти неделя |
| 6 | 0 | 20 | ❌ | ❌ | День покупок |
| 7 | 50 | 10 | ❌ | ✅ | 🎉 Недельный герой! |

**Итого за неделю:**
- XP: 125
- Монеты: 30
- Boost: 1
- Badge: 1 (каждую неделю)

**За месяц (4 недели):**
- XP: 500
- Монеты: 120
- Boost: 4
- Badge: 4

---

## 📊 Общий streak (статистика)

### Показ общего streak:

Общий streak показывается для статистики и престижа, **без наград**:

- **В компоненте DailyRewards:**
  - Основной счетчик: "X дней" (общий streak)
  - Подпись: "Неделя Y, день Z" (текущая неделя)

- **В профиле (опционально):**
  - "Общий streak: X дней"
  - "Завершено недель: Y"

**Важно:** 
- Общий streak **не сбрасывается** при пропуске дня (только недельный цикл)
- Награды выдаются только по недельному циклу (дни 1-7)
- Долгосрочная мотивация = Duel Pass (30 дней сезон)

---

## 🔗 Интеграция с Duel Pass

### Синергия систем:

#### День 4 (Boost день):
- **Без Premium:** Обычный boost
- **С Premium/Duel Pass:** Усиленный boost (+50% эффективности)

#### День 7 (Недельный герой):
- **Без Premium:** +10 монет, +50 XP
- **С Premium/Duel Pass:** +30 монет, +50 XP, +1 SP бонус

#### День 7 (Скин-фрагмент):
- Каждую неделю: +1 скин-фрагмент
- 7 фрагментов = 1 полный скин
- Можно ускорить через Duel Pass Premium

## 🎉 UX: День 7 как событие

### Визуальные эффекты:

1. **Анимация при открытии:**
   - Плавное появление золотого свечения
   - Пульсирующая рамка вокруг кнопки "Получить"
   - Легкое дрожание иконки награды

2. **Конфетти:**
   - Мини-конфетти при клике на "Получить"
   - 2-3 секунды анимации
   - Не нагружает производительность

3. **Текст:**
   - "🎉 Неделя завершена!"
   - "Ты завершил неделю! Начинается новая!"
   - "Неделя X завершена! Общий streak: Y дней"

4. **Звук:**
   - Специальный звук успеха (более торжественный)
   - Отличается от обычного звука награды

## 💾 Структура данных

```sql

  last_claimed_date: DATE,
  total_claims: INTEGER,
  created_at: TIMESTAMP,
  updated_at: TIMESTAMP
}
```

### daily_bonus_def (только 7 дней):
```sql
{
  day_number: INTEGER (1-7),
  reward: JSONB {
    xp: INTEGER,
    coins: INTEGER,
    boost: BOOLEAN,
    badge: STRING | NULL
  },
  description: TEXT
}
```

## 🎮 Логика работы

### Вычисление награды:

```typescript
// Недельный день (1-7)
const weekDay = currentStreak % 7 || 7;

// Номер недели
const weekNumber = Math.ceil(currentStreak / 7);

// Награда из daily_bonus_def
const reward = dailyBonusDef.find(r => r.day_number === weekDay);
```

### Обновление streak:

```typescript
// Если вчера был вход - увеличиваем streak
if (lastClaimedDate === yesterday) {
  newStreak = currentStreak + 1;
} else {
  // Пропущен день - сбрасываем на 1
  newStreak = 1;
}
```
