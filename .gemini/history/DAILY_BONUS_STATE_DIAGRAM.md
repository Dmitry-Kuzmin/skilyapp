# 🔄 Диаграмма состояний: Daily Bonus (7-дневный цикл)

## 📊 Основной поток состояний

```
┌─────────────────────────────────────────────────────────────────┐
│                    DAILY BONUS STATE MACHINE                     │
└─────────────────────────────────────────────────────────────────┘

                    ┌──────────────┐
                    │  INITIAL     │
                    │  (День 0)    │
                    │ streak = 0   │
                    └──────┬───────┘
                           │
                           │ Первый вход
                           │ (last_claimed_date === null)
                           ▼
                    ┌──────────────┐
                    │  DAY 1       │
                    │  streak = 1  │
                    │  weekDay = 1 │
                    └──────┬───────┘
                           │
                           │ Ежедневный вход
                           │ (last_claimed_date === yesterday)
                           ▼
                    ┌──────────────┐
                    │  DAY 2-6     │
                    │  streak++    │
                    │  weekDay++   │
                    └──────┬───────┘
                           │
                           │ Продолжение streak
                           ▼
                    ┌──────────────┐
                    │  DAY 7       │
                    │  streak++    │
                    │  weekDay = 7 │
                    │  🎉 EVENT    │
                    └──────┬───────┘
                           │
                           │ Следующий день
                           │ (новая неделя)
                           ▼
                    ┌──────────────┐
                    │  DAY 1       │
                    │  streak++    │
                    │  weekDay = 1 │
                    │  (NEW WEEK)  │
                    └──────┬───────┘
                           │
                           └─────────► Цикл повторяется
```

## 🔧 Логика вычислений

### Week Day (день недели):
```typescript
const weekDay = currentStreak % 7 || 7;
// Если 0, то 7 (кратно 7)
```

### Week Number (номер недели):
```typescript
const weekNumber = Math.ceil(currentStreak / 7);
```

### Reward Day (день награды):
```typescript
const rewardDay = weekDay; // Совпадает с weekDay
```

### Milestone Check (проверка вехи):
```typescript
const milestones = [30, 60, 90, 180, 365];
const newMilestone = milestones.find(m => 
  currentStreak >= m && !milestonesClaimed.includes(m)
);
```

## 🎨 Визуальные состояния UI

### Обычный день (1-6):
- Стандартная карточка
- Обычная кнопка "Получить бонус"
- Нет анимации

### День 7 (событие):
- Золотое свечение
- Пульсирующая рамка
- Кнопка "🎉 Завершить неделю!"
- Конфетти при получении

### Новая неделя (день 1 после дня 7):
- Плашка "✨ Следующая неделя началась!"
- Временно (2-3 секунды)

### Веха:
- Специальное уведомление
- Дополнительные награды
- Отображение в профиле
