## 🎯 Ключевые решения

### 1. Две системы streak:
- **Недельный цикл (1-7 дней)** — для ежедневных наград
- **Общий streak (1, 2, 3... 365+)** — для престижа и вех

### 2. Циклический расчет:
```typescript
const weekDay = currentStreak % 7 || 7;
const weekNumber = Math.ceil(currentStreak / 7);
```

### 3. Вехи общего streak:
- 30 дней: Бейдж + 50 монет
- 60 дней: Титул + 100 монет
- 90 дней: Рамка + 150 монет
- 180 дней: Титул + 200 монет
- 365 дней: Бейдж + Титул + Рамка + 500 монет

### 4. Интеграция с Duel Pass:
- День 4: Усиленный boost для Premium
- День 7: Бонусные монеты/SP для Premium

### 5. День 7 как событие:
- Анимация (золотое свечение, пульсирующая рамка)
- Конфетти
- Специальное уведомление

## 📊 Баланс наград

### За неделю:
- XP: 125
- Монеты: 30 (или 60 с Premium)
- Boost: 1
- Badge: 1

### За месяц (4 недели):
- XP: 500
- Монеты: 120 (или 240 с Premium)
- Boost: 4
- Badge: 4

## 📈 Метрики успеха

1. **Retention:**
   - % пользователей, которые возвращаются на день 2
   - % пользователей, которые доходят до дня 7
   - % пользователей, которые начинают новую неделю

2. **Engagement:**
   - Средний streak пользователей
   - Количество завершенных недель
   - Количество достигнутых вех

3. **Экономика:**
   - Баланс наград (не создает инфляцию)
   - Конверсия в Premium/Duel Pass
