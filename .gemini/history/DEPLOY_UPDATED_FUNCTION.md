## 🔍 После деплоя

1. **Попробуйте синхронизировать снова**
2. **Проверьте логи функции:**
   ```
   https://supabase.com/dashboard/project/yffjnqegeiorunyvcxkn/functions/sync-google-sheets/logs
   ```

3. **Ищите новые сообщения:**
   - `Row 1 first 3 columns:` - показывает первые 3 колонки для первой строки
   - `Row 2 first 3 columns:` - показывает первые 3 колонки для второй строки
   - `Row 3 first 3 columns:` - показывает первые 3 колонки для третьей строки
   - `Missing source_id for row X:` - детальная информация о проблемной строке

4. **Проверьте:**
   - `col0` - что находится в первой колонке
   - `col0_trimmed` - что получается после trim()
   - `col0_length` - длина первой колонки
   - `columnsCount` - сколько колонок распознано

## 📊 Что искать в логах

### Если `col0` пустой или `undefined`:
- Проблема в парсинге CSV
- Возможно, первая колонка пустая в экспорте Google Sheets
- Возможно, есть пустые строки между данными

### Если `col0_length` = 0:
- Первая колонка пустая после парсинга
- Возможно, есть только пробелы в ячейке

### Если `columnsCount` = 0:
- Строка пустая
- Нужно удалить пустые строки из таблицы

### Если `col0` содержит правильное значение:
- Проблема в проверке `source_id`
- Возможно, есть скрытые символы

## 🔧 Возможные решения

### Если `col0` пустой:
1. Проверьте, что в Google Sheets колонка A заполнена
2. Убедитесь, что нет пустых строк между данными
3. Попробуйте пересоздать `source_id` с помощью формулы: `="GS-" & ROW()-1`

### Если `col0_length` = 0:
1. Используйте формулу `=TRIM(A2)` для очистки пробелов
2. Примените ко всем ячейкам в колонке A
3. Скопируйте результаты и вставьте как "Только значения"

### Если `columnsCount` = 0:
1. Удалите все пустые строки из таблицы
2. Убедитесь, что данные идут подряд без пропусков

## 📞 Если проблема не решена

1. **Скопируйте логи** из функции
2. **Проверьте CSV экспорт вручную:**
   - File → Download → Comma Separated Values (.csv)
   - Откройте CSV файл в текстовом редакторе
   - Проверьте, что первая колонка заполнена

3. **Проверьте формат данных в Google Sheets:**
   - Убедитесь, что `source_id` - это текст (не число)
   - Убедитесь, что нет лишних пробелов
