# 🎨 Hero Section - Финальная полировка

## ✅ Что изменено

### **1. Электрический синий вместо фиолетового** ⚡

**Было:**
```tsx
// Фиолетовый градиент (indigo-400 → violet-400 → indigo-600)
bg-gradient-to-b from-indigo-400 via-violet-400 to-indigo-600
```

**Стало:**
```tsx
// Электрический синий градиент (blue-500 → sky-400 → cyan-300)
bg-gradient-to-r from-blue-500 via-sky-400 to-cyan-300
```

**Эффект:** Текст теперь горит **ярким синим неоном** в фирменных цветах бренда

---

### **2. Компактная верстка** 📐

#### **Расширили контейнер H2:**
- Было: `max-w-2xl` (слишком узко, текст разваливается на 4-5 строк)
- Стало: `max-w-3xl` (текст легко ложится в 2 строки)

#### **Уменьшили вертикальные отступы:**
- Badge → H1: `mb-6 sm:mb-8` → `mb-5 sm:mb-6` ✂️
- H1 → H2: `mb-5 md:mb-8` → `mb-4 md:mb-6` ✂️
- H2 → CTA: `mb-10 md:mb-12` → `mb-8 md:mb-10` ✂️
- CTA gap: `gap-6` → `gap-5` ✂️
- Section padding: `pb-10 md:pb-16` → `pb-8 md:pb-12` ✂️

**Результат:** Hero занимает **на 20-25% меньше места** по вертикали

---

### **3. Премиум-эффекты** ✨

#### **Синее свечение за текстом:**
```tsx
<div className="absolute inset-0 blur-3xl opacity-30 bg-gradient-to-r from-blue-500 via-sky-400 to-cyan-300"></div>
```
- **Эффект:** Текст "подсвечивает" стену за собой, как неоновая вывеска

#### **Плотный кернинг:**
- Было: `tracking-tight md:tracking-tighter`
- Стало: `tracking-tighter` (всегда плотно)
- **Эффект:** Заголовок выглядит более "сбитым" и дорогим

#### **Оптимизировали цвета:**
- Badge: `bg-indigo-500/10` → `bg-blue-500/10`
- Badge pulse: `bg-indigo-400` → `bg-sky-400`
- Текст badge: `text-indigo-300` → `text-blue-300`
