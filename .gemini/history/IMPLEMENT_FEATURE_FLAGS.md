# 🚦 Реализация Feature Flags - Пошаговая инструкция

**Цель:** Возможность выключить дуэли за 5 секунд без пересборки

---

## 📋 План действий

### 1. Применить миграцию (5 минут)

**Файл:** `supabase/migrations/20250101000001_app_config_feature_flags_safe.sql`

1. Supabase Dashboard → SQL Editor
2. Скопируйте SQL из файла
3. Выполните

**Проверка:**
```sql
SELECT * FROM app_config;
```

### 2. Добавить флаг для дуэлей (2 минуты)

Выполните в SQL Editor:

```sql
INSERT INTO app_config (key, value, description) VALUES
  ('duels_enabled', 'true'::jsonb, 'Включить функцию Дуэли')
ON CONFLICT (key) DO NOTHING;
```

### 3. Использовать в коде (15 минут)

#### 3.1. Обновить `Duel.tsx`

Добавить проверку флага в начале компонента.

#### 3.2. Обновить `useDuelRealtime.ts`

Добавить проверку флага `duel_realtime`.

#### 3.3. Обновить `AIWidget.tsx`

Добавить проверку флага `ai_chat_enabled`.

### 4. Создать админку (30 минут)

Создать компонент для управления флагами в админке.

---

**Готово!** После выполнения можно будет выключить дуэли за 5 секунд. ✅
