## 🐛 Возможные проблемы и решения

### Проблема: "Не удалось загрузить рейтинг"
**Решение:**
1. Проверь логи Edge Function в Supabase Dashboard
2. Убедись, что функция задеплоена
3. Проверь, что в таблице `profiles` есть пользователи с `duel_pass_level IS NOT NULL`

### Проблема: Фильтр "Друзья" не работает
**Решение:**
1. Убедись, что в таблице `referrals` есть записи для текущего пользователя
2. Проверь, что `referrer_id` или `referred_id` совпадает с `profileId`

### Проблема: Фильтр "Страна" не работает
**Решение:**
1. Убедись, что в таблице `profiles` есть `language_code`
2. Проверь, что `user.language_code` не NULL

### Проблема: Блок "Моё место" не загружается
**Решение:**
1. Проверь логи Edge Function
2. Убедись, что `profileId` не NULL
3. Проверь SQL функцию напрямую в SQL Editor

### Проблема: Пагинация не работает
**Решение:**
1. Проверь, что `pagination.total_pages > 1`
2. Убедись, что в ответе Edge Function есть поле `pagination`
