## 🐛 Исправление подсказок

### Было:
```typescript
if (messages.length === 1) {
  generateSmartSuggestions();
}
```
**Проблема:** Условие не срабатывало правильно

### Стало:
```typescript
if (!customPrompt && messages.length <= 1) {
  setTimeout(() => generateSmartSuggestions(), 1000);
}
```
**Решение:**
- Проверяем, что это первое объяснение (`!customPrompt`)
- Используем `<=` вместо `===`
- Добавили задержку 1 секунду для плавности

## 🎯 Компоненты Markdown

### Заголовки
- `<h1>` — крупный жирный (text-lg)
- `<h2>` — средний жирный (text-base)
- `<h3>` — мелкий жирный (text-sm)

### Выделение
- `<strong>` — жирный **цветом primary**
- `<em>` — курсив

### Списки
- `<ul>` — маркированные с точками
- `<ol>` — нумерованные
- Отступы и пробелы между элементами

### Абзацы
- `<p>` — отступы сверху и снизу
- Автоматический перенос строк
