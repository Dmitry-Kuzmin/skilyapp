# 🚀 Monetag для Telegram Mini Apps - Анализ

## 📊 Ключевые факты из официальной документации

### ✅ Что предлагает Monetag для TMA:

1. **Rewarded Interstitial** формат
   - Высокий CPM и CTR
   - Выглядит как сообщение в чате (неинтрузивно)
   - Настраиваемые параметры: размещение, таймауты, частота, награды

2. **Три способа интеграции:**
   - **SDK** - для разработчиков (предпочтительный вариант)
   - **Smartlink** - минимальная интеграция (просто ссылка)
   - **Programmatic** - для крупных приложений с кастомной интеграцией

3. **Преимущества Monetag:**
   - **500+ мини-приложений** уже монетизируются
   - **4000+ прямых рекламодателей** со всего мира
   - **80% рекламодателей** - web2 офферы (не привязаны к Telegram экосистеме)
   - **Неограниченные бюджеты** рекламодателей
   - **Персональный менеджер** для крупных издателей
   - **Приоритетная программа** с бонусами
   - **Реферальная программа**

4. **CPM:**
   - **Средний CPM начинается от $2**
   - Может значительно увеличиться после оптимизации алгоритма (несколько дней)
   - Зависит от GEO, платформы (iOS/Android), размещения, частоты
   - При высоком качестве трафика - можно договориться об особых условиях

5. **Требования:**
   - **Нет требований** к количеству пользователей
   - Можно монетизировать приложение любого размера

6. **Выплаты:**
   - **Минимум $5** (зависит от способа выплаты)
   - **CPM модель** - не зависит от сезонности, частоты показов или кликов
   - Доход растет вместе с аудиторией

7. **Качество рекламы:**
   - Только качественные рекламодатели из:
     * Investment & Trading
     * iGaming (только лицензированные бренды)
     * Financial
     * Mobile Apps
     * E-commerce
     * Utilities & Software
   - Можно отключить любую рекламу или группу реклам по запросу

8. **Статистика:**
   - Обновляется в течение нескольких часов
   - Рекомендуется смотреть финальную статистику в конце дня (EST)
   - Не в реальном времени (из-за задержек трекеров рекламодателей)

## 🔄 Сравнение: Monetag vs AdsGram для Telegram Mini Apps

| Параметр | **AdsGram** | **Monetag** |
|----------|-------------|-------------|
| **Формат** | Rewarded Video | Rewarded Interstitial (как сообщение в чате) |
| **CPM (старт)** | ~$1.5-4 (РФ), ~$8-15 (Tier-1) | От $2 (динамический, может расти) |
| **Рекламодатели** | Telegram-ориентированные | 80% web2 офферы (не привязаны к Telegram) |
| **Количество рекламодателей** | Среднее | 4000+ прямых рекламодателей |
| **Интеграция** | SDK (один способ) | SDK / Smartlink / Programmatic (3 способа) |
| **Требования к трафику** | Нет | Нет |
| **Минимальная выплата** | Не указано | $5 |
| **Персональный менеджер** | Не указано | Да (для крупных издателей) |
| **Реферальная программа** | Не указано | Да |
| **Качество рекламы** | Хорошее | Высокое (только лицензированные бренды) |
| **Настройка** | Базовая | Расширенная (размещение, таймауты, частота) |

## 💡 Рекомендация: Использовать ОБЕ платформы!

### Почему гибридный подход лучше:

1. **Максимизация дохода:**
   - AdsGram: для Telegram-ориентированной аудитории
   - Monetag: для web2 офферов (более широкий охват)

2. **Разные форматы:**
   - AdsGram: Rewarded Video (классический формат)
   - Monetag: Rewarded Interstitial (выглядит как сообщение в чате)

3. **Fallback и конкуренция:**
   - Если одна платформа не загрузила рекламу - показываем другую
   - Конкуренция между платформами = лучшие цены

4. **Разные рекламодатели:**
   - AdsGram: Telegram-экосистема
   - Monetag: глобальные web2 бренды

## 🎯 Стратегия интеграции

### Вариант 1: Приоритет AdsGram (текущий)
```
1. Пытаемся показать AdsGram Rewarded Video
2. Если не загрузилось → показываем Monetag Rewarded Interstitial
3. Если обе не загрузились → показываем сообщение "Реклама временно недоступна"
```

### Вариант 2: Приоритет Monetag
```
1. Пытаемся показать Monetag Rewarded Interstitial
2. Если не загрузилось → показываем AdsGram Rewarded Video
3. Если обе не загрузились → показываем сообщение "Реклама временно недоступна"
```

### Вариант 3: Ротация (рекомендуется)
```
1. Случайно выбираем платформу (50/50)
2. Если выбранная не загрузилась → показываем другую
3. Это обеспечивает равномерное распределение и тестирование обеих
```

## 📋 План интеграции Monetag

### Шаг 1: Регистрация (уже сделано ✅)
- Аккаунт создан на publishers.monetag.com
- Осталось завершить регистрацию

### Шаг 2: Добавить Telegram Mini App
1. В панели Monetag найти раздел "Telegram Mini Apps"
2. Добавить приложение (указать URL и название)
3. Получить API ключи и SDK

### Шаг 3: Интеграция SDK
1. Добавить Monetag SDK в `index.html`
2. Создать `src/lib/monetag.ts` (аналогично `adsgram.ts`)
3. Обновить `useRewardedAd` для поддержки обеих платформ

### Шаг 4: Настройка формата
- Выбрать Rewarded Interstitial
- Настроить размещение, частоту, таймауты
- Настроить награды за просмотр

## 🔧 Техническая реализация

### Архитектура:

```typescript
// src/lib/ad-provider.ts
export type AdProvider = 'adsgram' | 'monetag';

export function getAdProvider(): AdProvider {
  // Ротация или приоритет
  return Math.random() > 0.5 ? 'adsgram' : 'monetag';
}

// src/lib/monetag.ts
export function initMonetag() { ... }
export function showMonetagRewardedInterstitial() { ... }

// src/hooks/useRewardedAd.ts
export function useRewardedAd() {
  const showAd = async () => {
    const provider = getAdProvider();
    
    try {
      if (provider === 'adsgram') {
        return await showAdsGramRewardedVideo();
      } else {
        return await showMonetagRewardedInterstitial();
      }
    } catch (error) {
      // Fallback на другую платформу
      if (provider === 'adsgram') {
        return await showMonetagRewardedInterstitial();
      } else {
        return await showAdsGramRewardedVideo();
      }
    }
  };
}
```
