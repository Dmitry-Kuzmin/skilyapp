## 🎯 Приоритетные задачи

### Приоритет 1: Оптимизация изображений

**Проблема:**
- Изображения могут быть не оптимизированы (PNG/JPEG вместо WebP)
- Нет правильных размеров для мобильных устройств
- LCP изображение может быть слишком большим

**Действия:**
1. Проверить все изображения в Dashboard
2. Конвертировать в WebP формат
3. Добавить `srcset` для адаптивных изображений
4. Оптимизировать размер LCP изображения

### Приоритет 2: Оптимизация шрифтов

**Проблема:**
- Шрифты могут блокировать рендеринг
- Нет `font-display: swap`
- Нет preload для критичных шрифтов

**Действия:**
1. Проверить используемые шрифты
2. Добавить `font-display: swap` в CSS
3. Preload критичных шрифтов в `<head>`
4. Использовать woff2 формат

### Приоритет 3: Уменьшение unused JavaScript/CSS

**Проблема:**
- PageSpeed показывает unused JavaScript (350 KiB)
- Unused CSS (45 KiB)

**Действия:**
1. Проверить bundle analysis
2. Удалить неиспользуемый код
3. Оптимизировать импорты
4. Проверить Tailwind purge

### Приоритет 4: Оптимизация порядка загрузки

**Проблема:**
- Ресурсы могут загружаться в неправильном порядке
- Критичные ресурсы могут загружаться поздно

**Действия:**
1. Проверить Network Waterfall
2. Оптимизировать preload/prefetch
3. Убедиться, что критичные ресурсы загружаются первыми
