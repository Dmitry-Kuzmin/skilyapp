## 📋 Что делать дальше

### 1. Применить миграции в Supabase

**Через Supabase Dashboard:**

1. Открой: https://supabase.com/dashboard
2. Выбери проект
3. SQL Editor → New query
4. Примени миграции в порядке:
   - `20251218000001_create_countries_table.sql`
   - `20251218000002_create_pdd_russia_tables.sql`
   - `20251218000003_create_pdd_russia_storage_bucket.sql`
5. Нажми Run

**Проверка:**
- Table Editor → должны появиться таблицы `countries`, `pdd_russia_questions`, `pdd_russia_answers`, etc.
- Storage → должен появиться bucket `pdd_russia`

---

### 2. Скачать и импортировать данные

```bash
# 1. Скачай репозиторий
git clone https://github.com/etspring/pdd_russia.git /tmp/pdd_russia

# 2. Настрой .env (если ещё не настроен)
echo "SUPABASE_SERVICE_KEY=your-service-key" >> .env

# 3. Запусти импорт
npm run import:pdd-russia /tmp/pdd_russia
```

**Ожидаемый результат:**
- ~800 вопросов импортировано
- ~150 знаков импортировано
- ~200 штрафов импортировано
- Изображения загружены в Storage

## 🌍 Добавление новых стран

**См. подробное руководство:** `ADD_NEW_COUNTRY_GUIDE.md`

**Краткий процесс:**

1. Найди источник данных (GitHub, официальный сайт)
2. Создай таблицы (скопируй структуру России, адаптируй)
3. Напиши скрипт импорта (скопируй `import-pdd-russia.js`, адаптируй)
4. Добавь в админку (страница + меню + роут)
5. Добавь фронтенд для пользователей

**Примеры стран для добавления:**
- Украина (ПДД України)
- Беларусь (ПДД Беларуси)
- Германия (StVO)
- Франция (Code de la route)
