---

## 🎯 Проблемы Текущего Дизайна

### ❌ Что Не Так Сейчас:

1. **Разнобой Цветов:**
   - Зеленые, синие, желтые, фиолетовые карточки
   - Нет единой палитры
   - Выглядит хаотично

2. **Перегруженность:**
   - Слишком много элементов сразу
   - Нет четкой иерархии
   - Информация "кричит"

3. **Inconsistent Spacing:**
   - Разные отступы между элементами
   - Нет rhythm

4. **Старомодные Градиенты:**
   - Слишком яркие
   - Отвлекают от контента

---

## ✨ Принципы Нового Дизайна

### 1. Единая Цветовая Палитра

**Primary:** Indigo/Blue (#6366f1)  
**Success:** Subtle Green (#10b981)  
**Warning:** Subtle Amber (#f59e0b)  
**Neutral:** Slate shades

**НЕТ:** Красных, фиолетовых, оранжевых карточек  
**ДА:** Subtle accents, один primary цвет

### 2. Minimalism

**Меньше = Лучше:**
- Убрать лишние иконки
- Убрать декоративные элементы
- Фокус на данных, не на украшениях

### 3. Clear Hierarchy

```
H1: Заголовок страницы (36px, bold)
H2: Секции (24px, semibold)
H3: Карточки (18px, medium)
Body: Данные (14-16px, regular)
```

### 4. Consistent Cards

**Все карточки:**
- Одинаковый border-radius (12px)
- Одинаковый padding (24px)
- Одинаковая тень (subtle)
- Один стиль border

### 5. Modern Aesthetics

- Glassmorphism (легкий blur)
- Subtle gradients (не яркие!)
- Smooth animations
- Proper whitespace

## 🎨 Новый Дизайн (Концепт)

### Color Palette:

```css
Background: #0a0e1a (очень темный синий, не черный)
Cards: #151923 (темно-серый с легким синим оттенком)
Border: #1e293b (subtle)
Primary: #6366f1 (индиго)
```
