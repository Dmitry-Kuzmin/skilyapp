# 🚀 Быстрый старт - Получить Access Token

## Самый простой способ:

### Вариант 1: Через терминал (РЕКОМЕНДУЕТСЯ)
```bash
export PATH="$HOME/.local/bin:$PATH"
supabase login
```
Откроется браузер → авторизуйтесь → вернитесь в терминал → скажите мне "готово"

### Вариант 2: Через браузер
1. Откройте: https://supabase.com/dashboard/account/tokens
2. Нажмите "Generate new token"
3. Скопируйте токен
4. Сообщите мне токен

## ⚠️ НЕ путайте:
- ❌ Project Settings → API (это для Service Role Key)
- ✅ Account Settings → Access Tokens (это для Access Token)
