# 🧪 Инструкция по тестированию новой функциональности ПДД

## 🎯 Seamless UX-архитектура (Ultra Modern)

### Философия: "Не спрашивать дважды"

Если пользователь выбрал страну на лендинге, он **НЕ должен** видеть экран выбора после входа. Переход должен быть бесшовным.

---

## 📋 Поток пользователя

### 1. **Лендинг → Авторизация → Приложение**

#### Шаг А: Лендинг (Витрина)

**Авто-определение (Magic):**
- Пользователь заходит на сайт
- Система определяет IP (например, Испания)
- Сайт открывается с заголовком: *"Получи права в Испании (DGT)"*
- В хедере справа: селектор `[ 🇪🇸 ES ▾ ]`

**Ручной выбор (Если авто ошиблось):**
- Пользователь кликает `[ 🇪🇸 ES ]` → меняет на `[ 🇷🇺 RU ]`
- Сайт перезагружается: *"Сдай экзамен в ГИБДД РФ"*
- **ВАЖНО:** Выбор сохраняется в `localStorage` как `preselected_country: 'ru'`

#### Шаг Б: Переход в приложение (The Handshake)

Пользователь жмет **"Начать учиться"**.

**Сценарий 1: Страна уже выбрана (из Лендинга)**
- Приложение считывает `preselected_country` из `localStorage`
- **ПРОПУСКАЕТ** экран выбора страны (`/learn`)
- Сразу открывает дашборд с билетами выбранной страны (`/learn/russia`)
- *Результат:* Магия. Выбрал РФ на сайте → зашел → сразу вижу билеты РФ

**Сценарий 2: Страна НЕ выбрана (Прямая ссылка / Чистый браузер)**
- Приложение видит, что `preselected_country` пусто
- **ПОКАЗЫВАЕТ** экран выбора (`/learn`)
- Пользователь выбирает → сохраняем в `localStorage`

---

### 2. **Умный переключатель контекста (Внутри приложения)**

**Место:** В хедере дашборда (над приветствием, в строке "чипсов")

**Вид:** Одна слитная кнопка-пилюля с разделителем

```
[ 🇷🇺 Россия  |  🚗 B (Легковая) ▾ ]
```

**Интеракция:**
- При клике → снизу выезжает **Bottom Sheet** "Настройки обучения"
- Внутри два блока:
  1. **Выбор страны ПДД**
  2. **Выбор категории прав** (зависит от страны!)

**Логика зависимостей:**
- Выбираешь "Россия" → показываются категории РФ (A, B, C, D, BE, CE, DE, M)
- Выбираешь "Испания" → показываются категории DGT (A, A1, A2, B, B1, C, C1, D, D1, E, AM)
- Выбираешь "США" → показываются категории DMV (Class A, B, C, D, M)

## 🔧 Технические детали

### localStorage ключи:
- `preselected_country` - страна, выбранная на лендинге (`'russia' | 'spain'`)
- `pdd_selected_country` - страна, выбранная в приложении
- `pdd_selected_category` - категория прав (`'B' | 'A' | 'C'` и т.д.)

### Компоненты:
- `LearnCountrySelector` - экран выбора страны (показывается только если нет `preselected_country`)
- `ContextSwitcher` - умный переключатель в хедере дашборда
- `ContextSettingsSheet` - Bottom Sheet для выбора страны и категории

### Типы:
- `CountryCode` - `'russia' | 'spain' | 'ukraine' | 'belarus'`
- `LicenseCategory` - зависит от страны (см. `src/types/pdd.ts`)

## 🚀 Быстрый старт для тестирования

### Тест 1: Seamless переход с лендинга
1. Открой лендинг (или симулируй выбор страны)
2. Установи в консоли: `localStorage.setItem('preselected_country', 'russia')`
3. Перейди на `/learn` → должен автоматически редиректить на `/learn/russia`
4. **Результат:** Экран выбора пропущен, сразу дашборд

### Тест 2: Прямой вход (без выбора на лендинге)
1. Очисти `localStorage`: `localStorage.removeItem('preselected_country')`
2. Перейди на `/learn`
3. **Результат:** Показывается экран выбора страны

### Тест 3: Умный переключатель
1. Открой `/dashboard` или `/learn/russia`
2. Найди кнопку `[ 🇷🇺 Россия | 🚗 B ▾ ]` в хедере
3. Кликни → должен открыться Bottom Sheet
4. Измени страну → категории должны обновиться
5. Выбери категорию → нажми "Применить"
6. **Результат:** UI обновился, выбор сохранен

### Тест 4: Экзамен
1. Открой `/learn/russia`
2. Нажми "Экзамен ПДД"
3. Тестируй логику "5 за 1"
