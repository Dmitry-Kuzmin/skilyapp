# 🎯 Рекламные платформы для веб-браузера

## ⚠️ Важное отличие от мобильных

**Rewarded Video** на веб-браузерах имеет ограничения:
- Меньше платформ поддерживают этот формат
- Пользователи веба менее привычны к такому формату
- eCPM обычно ниже, чем в мобильных приложениях

---

## 🏆 Топ платформы для веб-браузера (по приоритету)

### 1. **Google AdSense / Google Ad Manager** ⭐⭐⭐⭐⭐
**Лучший выбор для глобальной аудитории (Tier-1, Tier-2)**

#### ✅ Плюсы:
- **Самая большая сеть** - максимальный fill rate
- **Высокий eCPM** для Tier-1 рынков ($8-20 за 1000 показов для display)
- **Rewarded Video поддерживается** через Ad Manager
- **Простая интеграция** - готовый SDK
- **Надежные выплаты** - Google Pay, банковский перевод
- **Аналитика мирового класса**

#### ❌ Минусы:
- **Строгая модерация** - может занять 1-2 недели
- **Требует трафик** - рекомендуют 100+ посетителей/день для одобрения
- **Минимальная сумма выплаты**: $100
- **В РФ работает ограниченно** (нужен VPN или прокси)

#### 💰 eCPM:
- **Tier-1** (США, Германия, UK): $8-20
- **Tier-2** (Испания, Франция): $3-8
- **РФ**: Не работает напрямую

#### 🔗 Документация:
- https://support.google.com/adsense
- https://developers.google.com/ad-manager/mobile-ads/sdk/rewarded-ads

---

### 2. **Яндекс.Директ / Яндекс RTB (РСЯ)** ⭐⭐⭐⭐⭐
**Лучший выбор для РФ и СНГ**

#### ✅ Плюсы:
- **Топ для России** - 90%+ российского трафика
- **Хорошие цены** для РФ ($1-4 eCPM для display, до $10 для video)
- **Rewarded Video поддерживается** через Яндекс Video Ads
- **Быстрая модерация** - 1-3 дня
- **Низкий порог** - можно подаваться с 10 униками (раньше требовали 500)
- **Выплаты в рублях** - удобно для РФ

#### ❌ Минусы:
- **Работает только для РФ/СНГ** - не подходит для глобальной аудитории
- **eCPM ниже** для международного трафика (если есть)

#### 💰 eCPM:
- **РФ**: $1-4 (display), $5-10 (video)
- **СНГ**: $0.5-2

#### 🔗 Документация:
- https://yandex.ru/support/direct/
- https://yandex.ru/support/direct/statistics/rtb.html

---

### 3. **Monetag** (ранее PropellerAds) ⭐⭐⭐⭐⭐
**Отличная платформа для веба И Telegram Mini Apps!**

#### ✅ Плюсы:
- **Поддерживает Rewarded Video/Interstitial** для веба И Telegram Mini Apps
- **Telegram Mini Apps**: Rewarded Interstitial (выглядит как сообщение в чате)
- **4000+ прямых рекламодателей** - огромная сеть
- **80% web2 офферы** - не привязаны к Telegram экосистеме (больше бюджетов)
- **Быстрая модерация** - 1-2 дня
- **Низкий порог** - можно подаваться с 0 трафиком
- **Минимальная выплата**: всего $5 (один из самых низких)
- **Глобальная аудитория** - работает везде
- **Гибкие форматы** - Video, Pop-under, Push-уведомления, In-page Push, Smart Links, Rewarded Interstitial
- **Еженедельные выплаты** - быстрее чем у конкурентов
- **Множество способов выплат** - PayPal, Payoneer, Skrill, WebMoney, банковский перевод
- **Персональный менеджер** для крупных издателей
- **Реферальная программа**
- **500+ Telegram Mini Apps** уже монетизируются

#### ❌ Минусы:
- **eCPM ниже** Google AdSense для Tier-1 ($2-8 для веба)
- **Для Telegram Mini Apps**: CPM начинается от $2 (может расти после оптимизации)
- **Меньше заполняемость** чем у Google
- **Популярные форматы** - Pop/Push (могут раздражать пользователей, если не настроить правильно)
- **Для неанглоязычного контента** выплаты могут быть ниже

#### 💰 eCPM:
- **Веб, Tier-1**: $2-8
- **Веб, Tier-2**: $1-3
- **Веб, РФ**: $0.5-2
- **Telegram Mini Apps**: от $2 (динамический, может значительно увеличиться)

#### 📊 Особенности:
- **Переименована** с PropellerAds в Monetag (декабрь 2022)
- **Фокус на веб-монетизацию** и высокотехнологичные решения
- **Популярность**: особенно популярна в США, Индии и России
- **Telegram Mini Apps**: Rewarded Interstitial формат (неинтрузивный, как сообщение в чате)
- **Три способа интеграции**: SDK, Smartlink, Programmatic

#### 🔗 Документация:
- https://monetag.com/ (основной сайт)
- https://monetag.com/publishers/ (для издателей)
- https://monetag.com/blog/telegram-mini-app-monetization-faq/ (Telegram Mini Apps FAQ)
- https://help.monetag.com/ (помощь)

### 4. **Media.net** ⭐⭐⭐
**Альтернатива для международной аудитории**

#### ✅ Плюсы:
- **Контекстная реклама** - высокий eCPM
- **Поддержка Video Ads** (но не классический Rewarded Video)
- **Глобальная сеть**
- **Быстрая интеграция**

#### ❌ Минусы:
- **Нет классического Rewarded Video** - только Video Ads
- **Меньше fill rate** чем Google
- **eCPM ниже** Google для Tier-1

#### 💰 eCPM:
- **Tier-1**: $3-10
- **Tier-2**: $1-5

## 🎯 Рекомендация для твоего проекта

### Стратегия по рынкам:

#### 🌍 **Глобальная аудитория (Tier-1, Tier-2):**
1. **Google Ad Manager** (Rewarded Video)
   - Лучший eCPM и fill rate
   - Нужно дождаться одобрения (100+ юзеров/день)

2. **Fallback: Monetag** (пока нет одобрения Google)
   - Можно подключить сразу
   - Поддерживает Rewarded Video
   - Низкий порог выплат ($5)

#### 🇷🇺 **Россия и СНГ:**
1. **Яндекс.Директ / Яндекс RTB**
   - Обязательно для РФ
   - Лучшие цены и заполняемость

## 📋 План действий

### Шаг 1: Определить приоритет рынка
- Если **главный рынок - РФ**: Яндекс.Директ
- Если **главный рынок - глобальный**: Google Ad Manager

### Шаг 2: Подключить платформу
1. Регистрация на платформе
2. Получение API ключей / Unit IDs
3. Интеграция SDK в код
4. Модерация приложения

### Шаг 3: Адаптировать код

Нужно будет:
1. Создать аналогичный `src/lib/web-ad-platform.ts` (как `adsgram.ts`)
2. Обновить `useRewardedAd` для поддержки веб-платформ
3. Добавить условие: `if (isTelegramMiniApp()) AdsGram else WebPlatform`

---

## 💡 Альтернативный подход (гибридный)

Если Rewarded Video на вебе сложно:
1. **Telegram Mini App** → AdsGram (Rewarded Video) ✅
2. **Веб-браузер** → Google AdSense (Display/Banner/Native)
   - Монетизируем веб через классическую рекламу
   - Пользователи веба более привычны к баннерам
   - Меньше технических сложностей

## 🔧 Техническая реализация

### Пример архитектуры:

```typescript
// src/lib/ad-provider.ts
export function getAdProvider() {
  if (isTelegramMiniApp()) {
    return 'adsgram'; // Telegram Mini App
  } else {
    return 'google-ads' || 'yandex'; // Web Browser
  }
}
```

### Структура кода:
```
src/lib/
  ├── adsgram.ts          # AdsGram SDK (Telegram)
  ├── google-ads.ts       # Google Ad Manager SDK (Web)
  └── yandex-ads.ts       # Яндекс RTB SDK (Web, РФ)
```
