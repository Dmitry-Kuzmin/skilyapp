# 🚀 Как задеплоить Edge Function (2 способа)

## ✅ Способ 1: Через Supabase Dashboard (ПРОЩЕ - рекомендую!)

### Шаг 1: Открой Supabase Dashboard
1. Перейди на: https://supabase.com/dashboard
2. Выбери проект **sdadim-dgt-prep**

### Шаг 2: Открой Edge Functions
1. В меню слева найди **"Edge Functions"**
2. Нажми на него

### Шаг 3: Открой функцию ai-chat
1. Найди функцию **"ai-chat"** в списке
2. Нажми на неё

### Шаг 4: Обнови код функции
1. Нажми кнопку **"Edit Function"** или **"View Code"**
2. Скопируй содержимое файла `supabase/functions/ai-chat/index.ts` из проекта
3. Вставь код в редактор
4. Нажми **"Deploy"** или **"Save"**

### Шаг 5: Проверь деплой
1. После деплоя должно появиться сообщение об успехе
2. Проверь логи: нажми **"Logs"** - там должны быть сообщения `[AI Chat]`

---

## ✅ Способ 2: Через CLI (если нужно)

### Шаг 1: Установи Supabase CLI

**На macOS (через Homebrew):**
```bash
brew install supabase/tap/supabase
```

**Или через npm:**
```bash
npm install -g supabase
```

**Или через прямую загрузку:**
1. Перейди на: https://github.com/supabase/cli/releases
2. Скачай версию для macOS
3. Распакуй и добавь в PATH

### Шаг 2: Авторизуйся в Supabase
```bash
supabase login
```

### Шаг 3: Свяжи проект
```bash
supabase link --project-ref ваш-project-ref
```

Project ref можно найти в настройках проекта в Supabase Dashboard.

### Шаг 4: Задеплой функцию
```bash
supabase functions deploy ai-chat
```

## 🎯 Рекомендация

**Используй Способ 1 (через Dashboard)** - он проще и не требует установки CLI!

---

## ❓ Проблемы?

### "Function not found"
- Убедись, что функция `ai-chat` существует
- Если нет - создай новую функцию через Dashboard

### "Deploy failed"
- Проверь, что код правильно скопирован
- Посмотри ошибки в логах
- Убедись, что секрет `GROQ_API_KEY` добавлен

### "Can't edit function"
- Некоторые проекты требуют обновления через CLI
- В таком случае установи CLI (Способ 2)

---

## ✅ После деплоя

1. Проверь работу AI помощника в приложении
2. Посмотри логи в Supabase Dashboard
3. Должно быть: `[AI Chat] Trying Groq API...` или `[AI Chat] ✅ Groq API success`
