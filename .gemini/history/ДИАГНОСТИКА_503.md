# 🔍 Диагностика ошибки 503

## Что проверить

### 1. Проверь, что функция задеплоена

1. Открой Supabase Dashboard
2. Перейди в **Edge Functions** → **ai-chat**
3. Проверь статус функции - должно быть **"Active"** или **"Deployed"**
4. Если статус другой - функция не задеплоена

### 2. Проверь логи функции

1. В функции **ai-chat** нажми **"Logs"**
2. Попробуй задать вопрос в приложении
3. Посмотри, что появляется в логах:

**Если видишь:**
- `[AI Chat] Request received` - функция работает
- `[AI Chat] Groq API key not found` - секрет не добавлен
- `[AI Chat] ❌ No AI provider available` - оба API недоступны
- Ничего не появляется - функция не запускается (проблема с деплоем)

### 3. Проверь секреты

1. Перейди в **Settings** → **Edge Functions** → **Secrets**
2. Убедись, что есть:
   - `GROQ_API_KEY` (обязательно)
   - `LOVABLE_API_KEY` (опционально, для fallback)

### 4. Проверь код функции

1. Открой функцию **ai-chat** в Dashboard
2. Проверь, что код начинается с:
   ```typescript
   import { createClient } from 'https://esm.sh/@supabase/supabase-js@2';
   ```
3. Проверь, что используется `Deno.serve`, а не `serve`

## Возможные причины 503

### Причина 1: Функция не задеплоена
**Решение:** Задеплой функцию заново

### Причина 2: Секрет GROQ_API_KEY не добавлен
**Решение:** Добавь секрет `GROQ_API_KEY` в Supabase Secrets

### Причина 3: Синтаксическая ошибка в коде
**Решение:** Проверь код на ошибки, попробуй задеплоить заново

### Причина 4: Функция падает при выполнении
**Решение:** Проверь логи - там должна быть ошибка

## Быстрое решение

1. **Убедись, что секрет добавлен:**
   - Settings → Edge Functions → Secrets
   - Должен быть `GROQ_API_KEY`

2. **Перезадеплой функцию:**
   - Edge Functions → ai-chat → Redeploy
   - Или скопируй код заново и задеплой

3. **Проверь логи:**
   - После деплоя задай вопрос
   - Посмотри логи - должно быть видно, что происходит

## Если ничего не помогает

Попробуй создать функцию заново:
1. Удали старую функцию (если возможно)
2. Создай новую функцию с названием `ai-chat`
3. Скопируй код из `supabase/functions/ai-chat/index.ts`
4. Задеплой
