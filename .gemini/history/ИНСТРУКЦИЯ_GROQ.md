# 🚀 Быстрая инструкция: Настройка Groq AI

## Что нужно сделать (3 простых шага):

---

### ✅ ШАГ 1: Получить API ключ Groq

1. Перейди на: **https://console.groq.com**
2. Зарегистрируйся (Google/GitHub/Email)
3. Открой **"API Keys"** в меню слева
4. Нажми **"Create API Key"**
5. **СКОПИРУЙ ключ** (он начинается с `gsk_`)
   - ⚠️ Сохрани его сразу - показывается только раз!

---

### ✅ ШАГ 2: Добавить ключ в Supabase

1. Открой: **https://supabase.com/dashboard**
2. Выбери проект **sdadim-dgt-prep**
3. Перейди: **Settings** → **Edge Functions** → **Secrets**
4. Нажми **"Add a new secret"**
5. Заполни:
   - **Name**: `GROQ_API_KEY`
   - **Value**: вставь ключ из шага 1
6. Нажми **"Save"**

---

### ✅ ШАГ 3: Задеплоить функцию

**Вариант А (через CLI):**
```bash
supabase functions deploy ai-chat
```

**Вариант Б (через Dashboard):**
1. Открой **Edge Functions** → **ai-chat**
2. Нажми **"Redeploy"**

---

## ✅ Готово!

Теперь AI помощник будет:
- ⚡ Работать быстрее (< 1 сек)
- 💰 Бесплатно (14,400 запросов/день)
- 🇷🇺 Лучше понимать русский язык
- 🔄 Автоматически переключаться на Lovable API, если Groq недоступен

---

## 🧪 Проверка

1. Открой приложение
2. Задай вопрос AI помощнику
3. Должен ответить быстро!
4. Проверь логи в Supabase: **Edge Functions** → **ai-chat** → **Logs**

В логах должно быть:
```
[AI Chat] Trying Groq API...
[AI Chat] ✅ Groq API success
```

---

## ❓ Проблемы?

**"AI service temporarily unavailable"**
- Проверь, что ключ добавлен правильно
- Название должно быть точно `GROQ_API_KEY`

**Медленно работает**
- Groq должен быть быстрым (< 1 сек)
- Проверь интернет-соединение
- Посмотри логи в Supabase

**Не работает вообще**
- Убедись, что функция задеплоена
- Проверь, что ключ активен в Groq Console
- Система автоматически переключится на Lovable API

---

**Всё готово! 🎉**
