# 📚 ИНСТРУКЦИЯ: Загрузка PDF учебников в Supabase

## Шаг 1: Подготовка

### 1.1. Создай папку для PDF
```bash
mkdir -p data/pdf
```

### 1.2. Положи PDF файлы
Переименуй и положи файлы в `data/pdf/`:
- `tema1.pdf` — Общие положения
- `tema2.pdf` — Водитель
- `tema3.pdf` — Скорость
- `tema4.pdf` — Знаки
- ...и так далее

**Важно:** Название должно содержать номер темы (`tema1`, `tema2`, и т.д.)

---

## Шаг 2: Установка зависимостей

Установи библиотеку для работы с PDF:

```bash
npm install pdf-parse
```

---

## Шаг 3: Применение миграции

### Через Supabase Dashboard:
1. Открой Supabase Dashboard
2. Перейди в **SQL Editor**
3. Скопируй содержимое файла `supabase/migrations/20251110_create_dgt_knowledge.sql`
4. Вставь в редактор
5. Нажми **"Run"**

### Или через CLI:
```bash
supabase db push
```

---

## Шаг 4: Получение Service Key

Для загрузки данных нужен service key:

1. Открой Supabase Dashboard → Settings → API
2. Скопируй **service_role key** (secret)
3. Создай файл `.env` в корне проекта:
   ```
   SUPABASE_SERVICE_KEY=твой_service_key
   ```

⚠️ **ВАЖНО**: Не коммить `.env` в Git!

---

## Шаг 5: Запуск скрипта обработки

```bash
node scripts/process-pdf-to-db.js
```

### Что происходит:
1. Скрипт читает PDF файлы из `data/pdf/`
2. Извлекает текст из каждого PDF
3. Разбивает на разделы
4. Извлекает ключевые слова
5. Загружает в таблицу `dgt_knowledge`

### Ожидаемый вывод:
```
🚀 Начинаем обработку PDF учебников DGT

📚 Найдено PDF файлов: 10

📄 Обработка: tema1.pdf
✅ Извлечено 15234 символов
✅ Создано 8 разделов
✅ Загружено 8 разделов из 8

📄 Обработка: tema2.pdf
...

✅ ГОТОВО! Загружено 75 разделов в базу данных
```

## Проверка

### Проверь, что данные загрузились:

1. Supabase Dashboard → Table Editor → dgt_knowledge
2. Должны быть записи с контентом из PDF
3. Проверь поля: topic_number, section_title, content

### Протестируй поиск:

В SQL Editor:
```sql
SELECT * FROM search_dgt_knowledge('обгон', NULL, 3);
```

Должны вернуться разделы про обгон.

## Возможные проблемы

### Проблема: "Cannot find module 'pdf-parse'"
**Решение:**
```bash
npm install pdf-parse
```

### Проблема: "SUPABASE_SERVICE_KEY not found"
**Решение:**
1. Получи service key в Supabase Dashboard
2. Создай файл `.env` с ключом

### Проблема: PDF не читается
**Решение:**
- Проверь, что PDF не зашифрован паролем
- Попробуй конвертировать PDF → текст вручную
- Используй онлайн-конвертер и сохрани как .txt

### Проблема: Разделы не создаются
**Решение:**
- Проверь структуру PDF (есть ли заголовки)
- Отредактируй функцию `splitIntoSections` под свой формат

## Альтернатива: Ручная загрузка

Если скрипт не работает, можно загрузить вручную:

1. Конвертируй PDF → текст (онлайн или через `pdftotext`)
2. Открой Supabase Dashboard → Table Editor → dgt_knowledge
3. Нажми **"Insert row"**
4. Заполни поля:
   - topic_number: 1
   - section_title: "Название раздела"
   - content: вставь текст из PDF
   - keywords: ['обгон', 'скорость']
5. Сохрани
