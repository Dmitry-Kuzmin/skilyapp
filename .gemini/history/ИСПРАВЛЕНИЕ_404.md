# 🔧 Исправление ошибки 404 Gemini

## Проблема
```
ERROR: Gemini API error: 404
models/gemini-1.5-flash-latest is not found
```

**Причина:** Google изменил названия моделей. Старое название больше не работает.

---

## ✅ Решение

Обновил файл **`AI_CHAT_FINAL.txt`**

**Изменение:**
- ❌ Старое: `gemini-1.5-flash-latest`
- ✅ Новое: `gemini-2.0-flash-exp`

---

## Что делать

### 1. Открой файл `AI_CHAT_FINAL.txt`

### 2. Скопируй ВЕСЬ обновлённый код
Ctrl+A → Ctrl+C (весь файл)

### 3. Вставь в Supabase
**Supabase** → **Edge Functions** → **ai-chat** → **Code**

- Выдели весь код (Ctrl+A)
- Вставь новый (Ctrl+V)
- Проверь строку 95: должно быть `gemini-2.0-flash-exp`

### 4. Deploy updates

Нажми **"Deploy updates"**
