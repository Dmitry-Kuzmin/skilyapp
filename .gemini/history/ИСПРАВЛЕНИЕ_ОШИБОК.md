## 🎯 Что с Rate Limit?

### Вариант 1: Подожди до завтра

**Groq лимит:**
- 14,400 запросов/день
- Обновляется в 00:00 (по времени Groq)
- Завтра всё заработает

### Вариант 2: Создай новый Groq ключ

1. Открой: https://console.groq.com/keys
2. Создай новый API ключ
3. **Supabase** → **Secrets** → обнови `GROQ_API_KEY`
4. Redeploy функцию
