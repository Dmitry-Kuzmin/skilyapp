## 2. 🖼️ Увеличение изображений исправлено

**Проблема:** Картинка обрезалась и выглядела странно.

**Что исправлено:**

### Было:
```css
max-w-[95vw] max-h-[95vh] w-auto h-auto
// Ограничение контейнера
max-w-full max-h-[95vh] object-contain
// Ограничение изображения
```
❌ Изображение обрезалось, не центрировалось

### Стало:
```css
w-screen h-screen // Диалог на весь экран
bg-black/95 // Тёмный фон
padding: 1rem // Отступы от краёв

max-w-full max-h-full // Изображение не выходит за экран
w-auto h-auto // Сохраняет пропорции
object-contain // Вписывается без обрезки
imageRendering: 'high-quality' // Лучшее качество
```

✅ **Результат:**
- Изображение на весь экран
- Сохраняет пропорции
- Не растягивается (нет пикселей!)
- Не обрезается
- Центрируется
- Кнопка закрытия (оранжевая, как на скрине)
