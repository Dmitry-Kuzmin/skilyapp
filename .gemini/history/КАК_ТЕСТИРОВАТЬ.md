# 🧪 Как протестировать AI помощника

## ✅ Шаг 1: Запусти приложение

Открой терминал в папке проекта и выполни:

```bash
npm run dev
```

Дождись сообщения:
```
➜  Local:   http://localhost:8080/
➜  Network: http://[::]:8080/
```

---

## ✅ Шаг 2: Открой приложение в браузере

1. Открой браузер (Chrome, Safari, Firefox)
2. Перейди по адресу: **http://localhost:8080**
3. Войди в приложение (если нужно авторизоваться)

---

## ✅ Шаг 3: Найди AI помощника

AI помощник находится на **главной странице** (Dashboard):

1. После входа в приложение ты попадёшь на главную страницу
2. **Вверху страницы** увидишь виджет AI помощника:
   - Поле для ввода вопроса
   - Кнопка отправки
   - Иконка с ботом 🤖

3. Или можешь использовать **готовые темы**:
   - 🚦 Знаки
   - 🚗 Манёвры
   - 📝 Экзамен
   - ⚠️ Ситуации

---

## ✅ Шаг 4: Задай вопрос AI помощнику

### Примеры вопросов для тестирования:

1. **"Объясни основные дорожные знаки"**
2. **"Как правильно выполнять обгон?"**
3. **"Что нужно знать для экзамена DGT?"**
4. **"Что делать при ДТП?"**
5. **"Какие штрафы за парковку на пешеходном переходе?"**

### Что должно произойти:

1. ✅ **Быстрый ответ** (< 2 секунды)
2. ✅ **Ответ на русском языке** (если задал вопрос на русском)
3. ✅ **Полезная информация** по ПДД
4. ✅ **Кнопки действий**:
   - 🚀 Пройти тест
   - 📖 Открыть урок
   - 💡 Объясни проще

---

## ✅ Шаг 5: Проверь логи в Supabase

Чтобы убедиться, что используется Groq:

1. Открой **Supabase Dashboard**: https://supabase.com/dashboard
2. Выбери проект **sdadim-dgt-prep**
3. Перейди в **Edge Functions** → **ai-chat**
4. Нажми **"Logs"** или **"View Logs"**
5. Должно быть видно:
   ```
   [AI Chat] Trying Groq API...
   [AI Chat] ✅ Groq API success
   ```

Если видишь:
- `[AI Chat] Groq API key not found` → проверь, что секрет `GROQ_API_KEY` добавлен
- `[AI Chat] Trying Lovable API (Gemini)...` → Groq недоступен, используется fallback
- `[AI Chat] ❌ No AI provider available` → нет доступных API ключей

## ✅ Шаг 6: Проверь работу в реальном времени

### Тест 1: Быстрота ответа
- Задай вопрос
- Ответ должен прийти **менее чем за 2 секунды**
- Если медленно → проверь интернет или логи

### Тест 2: Качество ответа
- Задай вопрос про ПДД
- Ответ должен быть **понятным и полезным**
- Должен содержать **практические советы**

### Тест 3: Множественные вопросы
- Задай несколько вопросов подряд
- Каждый ответ должен приходить быстро
- Контекст должен сохраняться

### Тест 4: Разные типы вопросов
- Вопросы про знаки
- Вопросы про манёвры
- Вопросы про экзамен
- Все должны работать корректно

## ❓ Проблемы при тестировании

### Проблема: "AI service temporarily unavailable"

**Решение:**
1. Проверь, что секрет `GROQ_API_KEY` добавлен в Supabase Secrets
2. Название должно быть точно `GROQ_API_KEY` (без пробелов)
3. Проверь, что ключ активен в Groq Console
4. Перезапусти функцию (redeploy)

### Проблема: Медленные ответы

**Решение:**
1. Groq должен работать очень быстро (< 1 сек)
2. Если медленно - проверь интернет-соединение
3. Посмотри логи в Supabase Dashboard
4. Проверь, что используется Groq (не Lovable API)

### Проблема: Ответы на английском

**Решение:**
1. Задавай вопросы на русском языке
2. AI должен отвечать на том же языке
3. Если не работает - проверь системный промпт в коде

### Проблема: Ошибка при отправке вопроса

**Решение:**
1. Проверь консоль браузера (F12 → Console)
2. Посмотри ошибки в логах Supabase
3. Убедись, что функция задеплоена
4. Проверь, что пользователь авторизован
