# 🎯 Компактные справочники DGT - НОВЫЙ ПОДХОД!

## Проблема старого подхода

**RAG с полными текстами:**
- ❌ ~9,000 токенов на запрос
- ❌ Всего 11 запросов/день
- ❌ Лимит исчерпается за час тестирования
- ❌ Медленный поиск по длинным текстам

---

## ✅ Новый подход: Компактные справочники

**Вместо длинных текстов → структурированные карточки:**

```
Estrechamiento:
• Правило: R-5/R-6 → первый → иерархия
• Штраф: 200€ + 4 балла
• Знаки: R-5 (уступи), R-6 (приоритет)
• Экзамен: автобус > легковая с прицепом
• Пример: узкий мост, автобус первый
```

**Преимущества:**
- ✅ **~300 токенов** вместо 5000 (в 15 раз меньше!)
- ✅ **~300 запросов/день** вместо 11
- ✅ Все ключевые факты сохранены
- ✅ Быстрый поиск
- ✅ Структурированные данные (JSON)

---

## 🗂️ Структура справочника

### Поля карточки правила:

```typescript
{
  keyword: "estrechamiento",          // ID правила
  keyword_es: "estrechamiento",      // Испанский термин
  keyword_ru: "сужение",             // Русский термин
  topic_number: 1,                    // Тема 1-10
  
  rule_summary: "Краткая суть...",    // 1 предложение
  
  details: {                          // JSON с цифрами
    "штраф": "200€",
    "баллы": 4,
    "знаки": ["R-5", "R-6"]
  },
  
  signs: ["R-5", "R-6"],             // Знаки
  terms: ["paso estrecho"],          // Термины для поиска
  
  exam_tips: [                        // Ловушки на экзамене
    "Автобус > легковая с прицепом",
    "Если нет знаков → кто первый"
  ],
  
  common_mistakes: [                  // Частые ошибки
    "Путают с rotonda"
  ],
  
  related_rules: [                    // Связанные правила
    "prioridad", 
    "jerarquia_vehiculos"
  ],
  
  practical_example: "Узкий мост..."  // Короткий пример
}
```

**Старый формат (полный текст):**
```

→ ~5000 токенов
× 2 раздела = 10,000 токенов

```
Компактное правило: 50-70 слов

× 2 правила = 600 токенов
