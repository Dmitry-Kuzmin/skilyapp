# 🎯 Лайфхак с темами из картинок - ГОТОВО!

## Твоя идея реализована!

**Теперь AI автоматически определяет тему по названию картинки!**

### Примеры:
- `123.jpeg` → **Тема 1**
- `239.jpeg` → **Тема 2**
- `345-something.jpg` → **Тема 3**
- `456_test.png` → **Тема 4**

---

## 🔍 Как это работает

### 1. Frontend извлекает номер темы
```typescript
const extractTopicFromImage = (imageUrl: string): number | null => {
  const filename = imageUrl.split('/').pop(); // "239.jpeg"
  const match = filename.match(/^(\d)/);      // Первая цифра: "2"
  return match ? parseInt(match[1], 10) : null; // → 2
};
```

**Логирование:**
```
[AI Dialog] Extracted topic 2 from image: 239.jpeg
```

### 2. Номер темы отправляется в Edge Function
```typescript
body: JSON.stringify({
  messages: newMessages,
  topicNumber: 2 // ← Из названия картинки!
})
```

### 3. Edge Function ищет в конкретной теме
```typescript
const knowledge = await searchDGTKnowledge(
  "estrechamiento prioridad", 
  2 // ← Ищет только в теме 2!
);
```

**Логирование:**
```
[AI Chat] 🎯 Topic number provided: 2 - will search in specific topic
[AI Chat] Search query extracted: estrechamiento prioridad señal
[AI Chat] ✅ Found 5 relevant sections from DGT manuals
```

### 4. SQL ищет в конкретной теме
```sql
WHERE 
  (topic_num IS NULL OR dk.topic_number = 2) -- Фильтр по теме!
  AND (content matches search terms)
```
