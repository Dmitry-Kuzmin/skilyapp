# 🔧 Исправление: Модель Groq обновлена

## Проблема

Модель `llama-3.1-70b-versatile` была **деактивирована** Groq и больше не поддерживается.

**Ошибка в логах:**
```
ERROR [AI Chat] Groq API error: 400 {"error":{"message":"The model llama-3.1-70b-versatile has been decommissioned and is no longer supported."}}
```

## Решение

✅ **Код обновлён!** Модель заменена на актуальную: `llama-3.3-70b-versatile`

## Что нужно сделать

### Шаг 1: Обнови код в Supabase Dashboard

1. **Ты находишься в правильном месте!** ✅
   - Supabase Dashboard → Edge Functions → ai-chat → Code

2. **Обнови код:**
   - В редакторе кода найди строку:
     ```typescript
     model: 'llama-3.1-70b-versatile',
     ```
   - Замени на:
     ```typescript
     model: 'llama-3.3-70b-versatile',
     ```
   
   **ИЛИ** скопируй весь обновлённый код из файла `supabase/functions/ai-chat/index.ts`

3. **Задеплой изменения:**
   - Нажми кнопку **"Deploy updates"** (зелёная кнопка внизу справа)
   - Дождись сообщения об успешном деплое

### Шаг 2: Проверь работу

1. Перезагрузи страницу приложения
2. Задай вопрос AI помощнику
3. Должен прийти ответ без ошибок

### Шаг 3: Проверь логи

1. В Supabase Dashboard перейди в **Logs** функции
2. Должно быть видно:
   ```
   [AI Chat] Trying Groq API...
   [AI Chat] ✅ Groq API success
   ```
