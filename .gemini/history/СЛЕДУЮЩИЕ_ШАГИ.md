## ЧТО ДЕЛАТЬ ДАЛЬШЕ

### Вариант А: У меня ЕСТЬ PDF файлы

#### 1. Положи PDF в папку `data/pdf/`

**Где папка:**
```
/Users/dimka/Desktop/Sdadim/sdadim-dgt-prep/data/pdf/
```

**Открыть в Finder:**
```bash
open /Users/dimka/Desktop/Sdadim/sdadim-dgt-prep/data/pdf
```

**Положи файлы:**
- `tema1.pdf`
- `tema2.pdf`
- ...
- `tema10.pdf`

#### 2. Установи библиотеку для PDF

```bash
npm install pdf-parse
```

#### 3. Получи Service Key

1. **Supabase Dashboard** → **Settings** → **API**
2. Скопируй **service_role key** (secret, не anon!)
3. Создай файл `.env` в корне проекта:
   ```
   SUPABASE_SERVICE_KEY=твой_service_key
   ```

#### 4. Запусти обработку PDF

```bash
npm run process-pdf
```

Должно вывести:
```
✅ ГОТОВО! Загружено 75 разделов в базу данных
```

#### 5. Обнови Edge Function

**Supabase Dashboard** → **Edge Functions** → **ai-chat** → **Code**

Скопируй код из файла: `AI_CHAT_WITH_KNOWLEDGE.txt`

Нажми **"Deploy updates"**

#### 6. Проверь работу

В логах должно быть:
```
[AI Chat] Searching DGT knowledge for: ...
[AI Chat] Found 3 relevant sections
[AI Chat] Using knowledge from DGT manuals
```

### Вариант Б: У меня НЕТ PDF (загружу потом)
