- ✅ AI определяет язык своего ответа

- ✅ Работает для русского и испанского

**Логика определения:**
```typescript
const detectLanguage = (text: string): 'ru' | 'es' => {
  const hasCyrillic = /[а-яА-ЯёЁ]/.test(text);
  const hasSpanishChars = /[áéíóúñÁÉÍÓÚÑ¿¡]/.test(text);
  const hasSpanishWords = /(autopista|autovía|carretera)/i.test(text);
  
  if (hasCyrillic) return 'ru';
  if (hasSpanishChars || hasSpanishWords) return 'es';
  return 'ru'; // По умолчанию
};
```

### 3. 📝 Улучшенные промпты

**Для русского:**
- Генерирует 3 подсказки (не 4)

**Для испанского:**
- Генерирует 3 подсказки
- Максимум 6 слов
- Примеры: "¿Cuál es la multa?", "¿Dónde se aplica?", "¿Regla básica?"

## 🔧 Технические детали

### Функция определения языка
- Проверяет кириллицу (`[а-яА-ЯёЁ]`)
- Проверяет испанские символы (`[áéíóúñ]`)
- Проверяет испанские слова (`autopista`, `autovía`, etc.)
- Возвращает `'ru'` или `'es'`

### Два промпта для генерации
- `promptRu` — для русских подсказок
- `promptEs` — для испанских подсказок
- AI получает соответствующий промпт в зависимости от языка

### Парсинг ответа
```typescript
const suggestions = fullResponse
```
