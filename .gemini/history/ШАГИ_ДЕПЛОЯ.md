# 🚀 ПОШАГОВАЯ ИНСТРУКЦИЯ: Задеплоить функцию ai-chat

## ✅ ШАГ 1: Открой Supabase Dashboard

1. Перейди на: **https://supabase.com/dashboard**
2. Войди в свой аккаунт
3. Выбери проект **sdadim-dgt-prep**

---

## ✅ ШАГ 2: Открой Edge Functions

1. В меню слева найди **"Edge Functions"** (иконка молнии ⚡ или код `</>`)
2. Нажми на него
3. Должен открыться список функций

---

## ✅ ШАГ 3: Найди или создай функцию ai-chat

### Вариант А: Функция уже существует

1. Найди функцию **"ai-chat"** в списке
2. Нажми на неё
3. Переходи к ШАГУ 4

### Вариант Б: Функции нет (нужно создать)

1. Нажми кнопку **"Create Function"** или **"New Function"** или **"+"**
2. Название функции: `ai-chat`
3. Нажми **"Create"** или **"Continue"**
4. Переходи к ШАГУ 4

---

## ✅ ШАГ 4: Скопируй и вставь код

1. **Открой файл** `AI_CHAT_CODE.txt` в проекте (или `supabase/functions/ai-chat/index.ts`)
2. **Выдели весь код** (Ctrl+A или Cmd+A)
3. **Скопируй** (Ctrl+C или Cmd+C)
4. **В Supabase Dashboard:**
   - Найди редактор кода функции
   - Выдели весь старый код
   - **Вставь новый код** (Ctrl+V или Cmd+V)
5. **Проверь**, что код вставился полностью

---

## ✅ ШАГ 5: Задеплой функцию

1. Найди кнопку **"Deploy"** или **"Save"** или **"Deploy Function"**
2. Нажми на неё
3. Дождись сообщения об успехе (обычно несколько секунд)
4. Должно появиться: **"Function deployed successfully"** или подобное

---

## ✅ ШАГ 6: Проверь секрет GROQ_API_KEY

1. Перейди в **Settings** → **Edge Functions** → **Secrets**
2. Убедись, что есть секрет с названием **`GROQ_API_KEY`**
3. Если его нет - добавь:
   - Нажми **"Add a new secret"**
   - **Name**: `GROQ_API_KEY`
   - **Value**: вставь ключ из Groq
   - Нажми **"Save"**

---

## ✅ ШАГ 7: Проверь работу

1. **Открой приложение** (локально или на сервере)
2. **Задай вопрос AI помощнику**
3. **Должен ответить быстро** (< 2 секунды)
4. **Проверь логи:**
   - В Supabase Dashboard → Edge Functions → ai-chat
   - Нажми **"Logs"** или **"View Logs"**
   - Должно быть: `[AI Chat] Trying Groq API...` и `[AI Chat] ✅ Groq API success`

---

## ❓ Проблемы и решения

### Проблема: "Function not found" или функции нет в списке

**Решение:**
- Создай новую функцию (Вариант Б в ШАГЕ 3)
- Название должно быть точно `ai-chat`

### Проблема: "Deploy failed" или ошибка при деплое

**Решение:**
1. Проверь, что код полностью скопирован
2. Убедись, что нет лишних символов в начале/конце
3. Посмотри ошибки в консоли браузера (F12 → Console)
4. Попробуй скопировать код заново

### Проблема: "Can't find Edit button" или нельзя редактировать

**Решение:**
- Некоторые проекты требуют деплоя через CLI
- В этом случае используй альтернативный способ (см. ниже)

### Проблема: "AI service temporarily unavailable"

**Решение:**
1. Проверь, что секрет `GROQ_API_KEY` добавлен правильно
2. Название должно быть точно `GROQ_API_KEY` (без пробелов)
3. Убедись, что ключ активен в Groq Console
4. Перезапусти функцию (redeploy)

### Проблема: Медленные ответы или не работает

**Решение:**
1. Проверь логи в Supabase Dashboard
2. Убедись, что Groq API ключ правильный
3. Проверь интернет-соединение
4. Система автоматически переключится на Lovable API, если Groq недоступен

## 🎯 Альтернативный способ: Установка CLI

Если Dashboard не работает, установи CLI:

### Шаг 1: Установи Homebrew (если нет)

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

### Шаг 2: Установи Supabase CLI

```bash
brew install supabase/tap/supabase
```

### Шаг 3: Авторизуйся

```bash
supabase login
```

### Шаг 4: Свяжи проект

1. Найди **Project Ref** в настройках проекта в Supabase Dashboard
2. Выполни:
```bash
supabase link --project-ref ваш-project-ref
```

### Шаг 5: Задеплой функцию

```bash
supabase functions deploy ai-chat
```
