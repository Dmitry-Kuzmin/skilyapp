# Анализ системы достижений Zutobi и рекомендации

## 📊 Анализ достижений из Zutobi

### Существующие достижения Zutobi:

1. **Prodigy** - получить 95%+ на категорийном тесте (+100 очков)
2. **Question Master** - получить 100% на главе (+100 очков)
3. **Devotion** - учиться 2 дня подряд (+100 очков)
4. **Exam Master** - пройти финальный тест 1 раз (+100 очков)
5. **Explorer** - завершить 30% дорожной карты (+100 очков)
6. **Influencer** - поделиться с 1 другом (+200 очков)
7. **Extra Test Master** - завершить 3 дополнительных теста (+100 очков)
8. **Unstoppable** - завершить 2 урока подряд без ошибок (+100 очков)

### Дополнительные системы Zutobi:
- **Vehicles** - коллекция транспорта, разблокируется за очки
- **Questions** - статистика по вопросам (правильность с первой попытки)
- **Leaderboard** - таблица лидеров
