**Текущий URL (с опечаткой):**
```
https://yffjnqegeiorunyvcxkn.supabase.co/functions/v1/ad-reward?userid=[userld]&reward_type=coins&amount=20
```

**Проблема:** `[userld]` вместо `[userId]` ❌

**Правильный URL должен быть:**
```
https://yffjnqegeiorunyvcxkn.supabase.co/functions/v1/ad-reward?userid=[userId]&reward_type=coins&amount=20
```

**Как исправить:**
1. Нажми кнопку **"Редактировать"** (Edit) на странице блока
2. Найди поле **"Reward URL"**
3. Измени `[userld]` на `[userId]`
4. Сохрани изменения

**Важно:** Если URL неправильный, AdsGram не сможет передать ID пользователя, и награды не будут начисляться!
