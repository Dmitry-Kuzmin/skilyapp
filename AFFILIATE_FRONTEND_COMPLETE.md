### Шаг 2: Обновить Типы Supabase (опционально)

TypeScript ругается на новые RPC функции - это нормально. Есть 2 варианта:

#### Вариант А: Игнорировать ошибки (быстро)
Добавить `// @ts-ignore` перед вызовами RPC:

```typescript
// @ts-ignore - новая RPC функция из миграции
const { data, error } = await supabase.rpc('get_partner_funnel_stats', {
  p_partner_id: partnerId,
  p_days: days
});
```

#### Вариант Б: Обновить типы (правильно)
```bash
# Сгенерировать новые типы из Supabase
npx supabase gen types typescript --project-ref YOUR_PROJECT_REF > src/integrations/supabase/types.ts
```

## 🐛 Известные Проблемы

### 1. TypeScript Ошибки (24 ошибки)
**Причина:** Новые RPC функции не в типах Supabase  
**Решение:** См. "Шаг 2" выше (игнорировать или обновить типы)  
**Критичность:** Низкая (не влияет на работу)

### 2. Нет данных в воронке
**Причина:** Еще не было трафика по партнерским ссылкам  
**Решение:** Эмулировать тестовые данные (SQL выше) или дождаться реального трафика
