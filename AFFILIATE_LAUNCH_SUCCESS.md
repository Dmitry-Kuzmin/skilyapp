## 🎯 KPI для Мониторинга

**Отслеживай еженедельно:**

```sql
-- Dashboard партнерской программы
SELECT 
  COUNT(*) as total_partners,
  COUNT(*) FILTER (WHERE registration_status = 'approved') as approved_partners,
  SUM(total_link_activations) as total_activations,
  SUM(accumulated_commission) as total_commission,
  ROUND(AVG(accumulated_commission) FILTER (WHERE accumulated_commission > 0), 2) as avg_commission_per_partner
FROM partners;

-- Топ-5 партнеров
SELECT 
  p.name,
  p.partner_code,
  p.total_link_activations,
  COUNT(*) FILTER (WHERE pc.event_type = 'purchase') as purchases,
  SUM(pc.commission_amount) as commission
FROM partners p
LEFT JOIN partner_conversions pc ON p.id = pc.partner_id
WHERE p.registration_status = 'approved'
GROUP BY p.id
ORDER BY commission DESC NULLS LAST
LIMIT 5;
```

**Цели на 3 месяца:**
- Активных партнеров: **50+**
- Конверсия клик → покупка: **>5%**
- LTV партнера: **>€500**
- Fraud rate: **<2%**
