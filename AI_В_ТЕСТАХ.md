# ✅ AI-объяснения в тестах (режим практики)

## Что добавлено

В режиме **практики** (не экзамен) после ответа на вопрос:
1. Показывается стандартное объяснение
2. Под ним — кнопка **"Спросить AI: объясни по-другому"**
3. При клике на кнопку открывается диалог с AI-объяснением

## Как работает

### 1. Автоматическое объяснение
При открытии диалога AI автоматически:
- Анализирует вопрос
- Видит правильный и выбранный ответы
- Видит официальное объяснение
- Видит тему вопроса
- Разжёвывает тему простым языком с примерами

### 2. Дополнительные кнопки
Пользователь может уточнить:
- **💡 Проще** — объяснить ещё проще
- **🚗 Пример** — привести пример из жизни
- **🧠 Как запомнить** — совет по запоминанию

### 3. Контекст
AI получает:
- Текст вопроса
- Правильный ответ
- Ответ пользователя
- Результат (правильно/неправильно)
- Официальное объяснение
- Тему вопроса
- Информацию о наличии изображения

## Как протестировать

1. Запусти приложение: `npm run dev`
2. Открой браузер: http://localhost:8080
3. Перейди в **Тесты** → выбери тему
4. Выбери режим **"Практика"** (не экзамен)
5. Ответь на вопрос
6. После ответа появится объяснение
7. Под объяснением — кнопка **"Спросить AI: объясни по-другому"**
8. Нажми на кнопку
9. Откроется диалог с AI-объяснением
10. Попробуй кнопки: "Проще", "Пример", "Как запомнить"

## Преимущества

1. **Контекстное обучение** — AI видит весь контекст вопроса
2. **Персонализация** — может объяснить проще или подробнее
3. **Реальные примеры** — приводит примеры из жизни
4. **Интерактивность** — пользователь может уточнять
5. **Не мешает экзамену** — работает только в практике
