## 🚀 Быстрый Старт (3 Способа)

### Способ 1: Использовать Админку (Для Тебя) ⭐

**ШАГ 1: Создать автошколу через админку**

1. Открой `/admin/partners`
2. Нажми кнопку **"Добавить партнера"**
3. Заполни форму:
   ```
   Имя партнера: AutoEscuela Madrid
   Email: autoschool@example.com
   Тип партнера: Бартер (или Revenue Share)
   ```
4. Нажми **"Создать партнера"**

**ШАГ 2: Изменить тип на autoschool (SQL, один раз)**

```sql
-- Изменить тип созданного партнера
UPDATE partners
SET 
  partner_type = 'autoschool',
  instructor_mode_enabled = true
WHERE email = 'autoschool@example.com';
```

**ШАГ 3: Одобрить партнера**

1. В `/admin/partners` найди созданного партнера
2. Нажми **"Одобрить"**
3. ✅ Premium автоматически активируется!

**ШАГ 4: Войти как автошкола**

1. Зарегистрируй пользователя с email `autoschool@example.com`
2. Войди под ним
3. Открой `/partner-dashboard`
4. Увидишь вкладку **"Мои Студенты"** 🎓

---

### Способ 2: Форма Регистрации (Для Партнеров)

**Если автошкола регистрируется сама:**

1. Автошкола открывает `/partners`
2. Заполняет форму регистрации
3. Ты одобряешь в `/admin/partners`
4. **Вручную меняешь** тип на `autoschool` (SQL выше)
5. Автошкола заходит в кабинет

---

### Способ 3: Быстрое Создание (SQL) - Твой Аккаунт

**Самый быстрый для теста:**

```sql
-- Использовать свой существующий аккаунт
-- 1. Узнать свой user_id
SELECT id, email FROM auth.users WHERE email = 'твой@email.com';

-- 2. Создать автошколу с твоим user_id
INSERT INTO partners (
  name,
  email,
  user_id,
  partner_type,
  partner_code,
  registration_status,
  status,
  instructor_mode_enabled
) VALUES (
  'Моя Тестовая Автошкола',
  'твой@email.com',
  'ТВОЙ-USER-ID', -- 👈 Вставь свой UUID
  'autoschool',
  'MYAUTO',
  'approved',
  'active',
  true
) RETURNING id, is_partner_premium;

-- 3. Готово! Открой /partner-dashboard
-- Увидишь вкладку "Мои Студенты"
```

## 🎓 Как Добавлять Студентов (Через UI!)

### После Входа в Кабинет Автошколы:

**ШАГ 1:** Открой `/partner-dashboard`

**ШАГ 2:** Перейди на вкладку **"Мои Студенты"**

**ШАГ 3:** Нажми кнопку **"Добавить студента"** (вверху справа) ✨

**ШАГ 4:** Заполни форму:
```
Email студента: student@example.com
Группа: Группа А
Дата зачисления: [сегодня]
Планируемый экзамен: [через месяц]
Заметки: Хороший студент, занимается регулярно
```

**ШАГ 5:** Нажми **"Добавить студента"**

**✅ Готово!** Студент появится в списке с его прогрессом! 🎉

## 💡 Важно: Студент Должен Быть Зарегистрирован

**Форма добавления студента работает так:**

1. Вводишь **email студента**
2. Система ищет пользователя с таким email в БД
3. Если найден → добавляет в автошколу
4. Если НЕ найден → показывает ошибку

**Это значит:**
- ✅ Студент **сначала** регистрируется в Skily (через приложение)
- ✅ **Потом** автошкола добавляет его по email

**Зачем так:**
- Автошкола видит **реальный** прогресс из приложения
- Не нужно вручную вводить статистику
- Всё автоматически синхронизируется!

## 🎯 Полный Flow (Как Это Работает)

```
1. СТУДЕНТ
   Скачивает Skily → Регистрируется (email: ivan@example.com)
   ↓
2. АВТОШКОЛА
   Открывает /partner-dashboard → "Мои Студенты"
   ↓
3. АВТОШКОЛА
   Нажимает "Добавить студента" → Вводит: ivan@example.com
   ↓
4. СИСТЕМА
   Находит Ивана в БД → Связывает с автошколой
   ↓
5. АВТОШКОЛА
   Видит в кабинете:
   • Сколько тестов Иван решил
   • Какая у него точность
   • Готов ли к экзамену ✅
   ↓
6. ИВАН
   Решает тесты в приложении
   ↓
7. АВТОШКОЛА
   Видит прогресс В РЕАЛЬНОМ ВРЕМЕНИ! (обновляется при перезагрузке)
```
