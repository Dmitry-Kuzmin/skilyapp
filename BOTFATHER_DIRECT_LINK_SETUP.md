# 🔗 Настройка Direct Link в BotFather

## 📋 Инструкция по заполнению формы

### Шаг 1: Откройте BotFather

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте команду `/newlink` или выберите существующего бота `@sdadimtutbot`
3. Выберите опцию "Create Direct Link" / "Создать прямую ссылку"

---

## 📝 Заполнение формы

### 1. **Enter URL** (URL приложения)

**Введите:**
```
https://skilyapp.com
```

**Или если используете другой домен:**
- `https://skily.app` (если настроен)
- `https://your-vercel-app.vercel.app` (если деплой на Vercel)

⚠️ **Важно:** 
- Это должен быть **реальный HTTPS URL** вашего приложения
- URL должен быть доступен в интернете
- Не используйте `localhost` или временные туннели для продакшена

---

### 2. **Title** (Заголовок)

**Введите:**
```
Skily - Подготовка к экзамену на права
```

**Или на английском:**
```
Skily - Driver's License Exam Prep
```

**Или на испанском:**
```
Skily - Preparación para el Examen de Conducir
```

---

### 3. **Description** (Описание)

**Русский вариант:**
```
Приложение для подготовки к экзамену на водительские права. Тренируйся на реальных билетах, проходи тесты и готовься к экзамену в ГИБДД.
```

**Английский вариант:**
```
Driver's license exam preparation app. Practice with real exam questions, take tests and prepare for your DMV exam.
```

**Испанский вариант:**
```
Aplicación de preparación para el examen de conducir. Practica con preguntas reales del examen, realiza tests y prepárate para tu examen de tráfico.
```

---

### 4. **Set Photo or GIF** (Обложка)

**Рекомендации:**
- **Размер:** 640x360 пикселей (для фото)
- **Или GIF:** 320x180, 640x360 или 960x540 пикселей
- **Формат:** JPG, PNG или GIF
- **Содержание:** Логотип приложения, скриншот интерфейса или промо-изображение

**Где взять изображение:**
- Создайте скриншот главного экрана приложения
- Используйте логотип Skily
- Или создайте промо-баннер с текстом "Skily - Подготовка к экзамену"

**Если нет изображения:**
- Можно пропустить этот шаг (нажать "Skip" / "Пропустить")
- Telegram создаст превью автоматически на основе Title и Description

---

### 5. **Link** (Ссылка)

**Предлагается автоматически:**
```
t.me/sdadimtutbot/directlink
```

✅ **Оставьте как есть** - это правильный формат!

**Или можно изменить на:**
- `t.me/sdadimtutbot/app` (если хотите использовать `/app`)
- `t.me/sdadimtutbot/start` (если хотите использовать `/start`)

⚠️ **Важно:** 
- Ссылка должна быть уникальной для вашего бота
- Формат: `t.me/BOT_USERNAME/LINK_NAME`
- `LINK_NAME` может быть любым (например: `app`, `start`, `directlink`, `open`)

---

## ✅ После заполнения

1. **Нажмите кнопку "Create"** / **"Создать"**
2. BotFather подтвердит создание ссылки
3. Вы получите финальную ссылку вида: `https://t.me/sdadimtutbot/directlink`

---

## 🔗 Использование Direct Link

### Где можно использовать:

1. **В рекламе:**
   - Прямая ссылка на приложение
   - Открывается сразу в Telegram Web App

2. **В партнерских ссылках:**
   - Можно комбинировать с параметрами: `https://t.me/sdadimtutbot/directlink?startapp=PARTNER_CODE`

3. **В социальных сетях:**
   - Поделиться ссылкой в постах
   - Добавить в профиль/био

4. **В AdsGram:**
   - Использовать как "Telegram direct link" при регистрации

---

## 📱 Примеры использования

### Простая ссылка:
```
https://t.me/sdadimtutbot/directlink
```

### С параметром для партнеров:
```
https://t.me/sdadimtutbot/directlink?startapp=PARTNER_CODE
```

### С параметром для рефералов:
```
https://t.me/sdadimtutbot/directlink?startapp=ref_REFERRAL_CODE
```

---

## ⚠️ Важные моменты

1. **URL должен быть доступен:**
   - Проверьте, что `https://skilyapp.com` открывается в браузере
   - Если используете Vercel - убедитесь, что деплой активен

2. **Обложка не обязательна:**
   - Можно пропустить, если нет подходящего изображения
   - Telegram создаст превью автоматически

3. **Ссылка уникальна:**
   - Каждый бот может иметь несколько Direct Links
   - Название ссылки (`directlink`) должно быть уникальным для бота

4. **Можно изменить позже:**
   - Через BotFather можно редактировать Direct Link
   - Команда: `/mybots` → выбрать бота → "Links" → выбрать ссылку → "Edit"

7. Измените нужные поля
