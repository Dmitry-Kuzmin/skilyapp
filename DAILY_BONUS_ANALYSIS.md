## 🔴 Выявленные проблемы

### 1. **Отсутствие прогрессии**
- День 1: 10 XP
- День 2: 15 XP (+5)
- День 3: 20 XP (+5)
- День 4: 0 XP (❌ **скачок вниз**)
- День 5: 30 XP (+30)
- День 6: 0 XP (❌ **снова скачок**)
- День 7: 50 XP (+50)

**Проблема:** Нет логичной прогрессии, пользователь не видит роста наград.

### 2. **День 4 - только Boost, без XP/монет**
- Пользователь получает только Boost
- Нет видимого прогресса (0 XP, 0 монет)
- Может разочаровать пользователя, который ожидает награду

**Проблема:** Пользователь может не понять ценность Boost и почувствовать, что "ничего не получил".

### 3. **День 6 дает больше монет, чем день 7**
- День 6: 20 монет
- День 7: 10 монет (❌ **меньше!**)

**Проблема:** Финальный день недели должен быть самым ценным, но дает меньше монет.

### 4. **Несбалансированное распределение**
- Первые 3 дня: только XP
- День 4: только Boost
- День 5: только XP
- День 6: только монеты
- День 7: XP + монеты (но меньше монет чем день 6)

**Проблема:** Нет комбинации наград в большинстве дней, что делает систему скучной.

### 5. **Низкая мотивация для финального дня**
- День 7 должен быть кульминацией недели
- Но дает меньше монет, чем день 6
- Только 10 монет за завершение недели - недостаточно впечатляюще

## 🎯 Критерии хорошей системы ежедневных наград

1. **Прогрессия** - награды должны расти к концу недели
2. **Баланс** - комбинация XP и монет в большинстве дней
3. **Мотивация** - финальный день должен быть самым ценным
4. **Понятность** - пользователь должен видеть логику роста
5. **Ценность Boost** - Boost должен дополнять, а не заменять награды
