# 📝 Тексты для UI: Daily Bonus (7-дневный цикл)

## 🎨 Компонент DailyRewards

### Заголовок
**Основной:**
```
Ежедневная серия
```

**Подзаголовок:**
```
Неделя {weekNumber}
```
Где `weekNumber = Math.ceil(currentStreak / 7)`

**Примеры:**
- Streak 1-7: "Неделя 1"
- Streak 8-14: "Неделя 2"
- Streak 15-21: "Неделя 3"

---

### Счетчик streak

**Основной счетчик:**
```
{currentStreak} дней
```

**Подпись (опционально):**
```
Текущая неделя: день {weekDay}
```
Где `weekDay = currentStreak % 7 || 7`

**Примеры:**
- Streak 1: "1 день" / "Текущая неделя: день 1"
- Streak 7: "7 дней" / "Текущая неделя: день 7"
- Streak 8: "8 дней" / "Текущая неделя: день 1"
- Streak 14: "14 дней" / "Текущая неделя: день 7"

---

## 📱 Сообщения в компоненте

### Индикатор недели
**Над шкалой прогресса:**
```
Неделя {weekNumber}
```

**Примеры:**
- "Неделя 1"
- "Неделя 2"
- "Неделя 3"

---

### Индикатор новой недели
**При завершении дня 7 (следующий день):**
```
✨ Следующая неделя началась!
```

**Где показывать:**
- Временно (2-3 секунды) при открытии компонента
- Если `currentStreak % 7 === 1` и `currentStreak > 7`

---

### Описание награды
**Под счетчиком (опционально):**
```
Сегодня: {description}
```

**Примеры:**
- День 1: "Сегодня: Первый шаг"
- День 4: "Сегодня: Boost день!"
- День 7: "Сегодня: Недельный герой!"
