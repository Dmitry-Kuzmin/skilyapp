# Автоматический деплой Edge Functions

## Быстрая настройка (один раз)

Для автоматического деплоя Edge Functions нужно один раз авторизоваться в Supabase.

### Вариант 1: Авторизация через CLI (рекомендуется)

```bash
npx supabase login
```

После авторизации токен будет сохранен автоматически, и деплой будет работать автоматически.

### Вариант 2: Использование токена из переменной окружения

1. Получите токен доступа: https://supabase.com/dashboard/account/tokens
2. Установите переменную окружения:

```bash
export SUPABASE_ACCESS_TOKEN="your-token-here"
```

Или добавьте в `~/.zshrc` или `~/.bashrc`:
```bash
echo 'export SUPABASE_ACCESS_TOKEN="your-token-here"' >> ~/.zshrc
source ~/.zshrc
```

## Деплой функции

После настройки авторизации, деплой выполняется автоматически:

```bash
npm run supabase:deploy:duel-manager
```

Скрипт автоматически:
- ✅ Проверяет наличие Supabase CLI через npx
- ✅ Использует токен из переменной окружения или сохраненную авторизацию
- ✅ Деплоит функцию `duel-manager`
- ✅ Показывает ссылку на логи

## Автоматический деплой

После первоначальной настройки, все последующие деплои будут выполняться автоматически при запуске команды `npm run supabase:deploy:duel-manager`.
