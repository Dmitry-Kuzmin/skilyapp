## 🚨 Потенциальные проблемы

### 1. Long Task при загрузке
**Проблема:** Обнаружены блокирующие задачи > 50ms (3 случая)  
**Время:** ~400ms, ~600ms, ~800ms после загрузки  
**Влияние:** Может вызывать задержки UI  
**Рекомендация:** 
- Проверить тяжелые вычисления в `useDashboardData`
- Оптимизировать инициализацию компонентов дашборда
- Использовать `useMemo` и `useCallback` где возможно
- Рассмотреть виртуализацию для больших списков

### 2. Super RPC не работает
**Проблема:** Super RPC функция возвращает ошибку  
**Влияние:** Используется fallback на regular RPC (работает)  
**Рекомендация:**
- Проверить функцию `get_dashboard_super` в Supabase
- Убедиться что функция существует и доступна
- Проверить права доступа (RLS policies)

### 3. Внешние скрипты
**Проблема:** Загружаются несколько внешних SDK (Adsgram, Monetag)  
**Влияние:** Увеличивают время загрузки  
**Рекомендация:**
- Проверить необходимость всех SDK в dev режиме
- Рассмотреть lazy loading для некритичных SDK
