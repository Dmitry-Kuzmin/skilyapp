# 🔧 Исправление ошибки "Invalid API Key provided"

## Проблема:
Stripe говорит: `Invalid API Key provided: sk_test_...`

Это означает, что:
- ✅ Секрет теперь доступен функции (`hasStripeSecret: true`)
- ❌ Но Stripe не принимает этот ключ

## Возможные причины:

### 1. Неправильный или неполный ключ
- Ключ может быть обрезан при копировании
- Ключ может быть из другого аккаунта Stripe
- Ключ может быть устаревшим

### 2. Ключ из неправильного режима
- Используете `sk_live_...` в тестовом режиме Stripe
- Или наоборот

## Решение:

### Шаг 1: Получите правильный ключ из Stripe

1. Откройте Stripe Dashboard: https://dashboard.stripe.com/test/apikeys
2. Убедитесь, что вы в **Test Mode** (переключатель вверху справа)
3. Найдите раздел **"Standard keys"**
4. Найдите **"Secret key"** (НЕ "Publishable key")
5. Нажмите **"Reveal test key"** или **"Reveal"**
6. Скопируйте **ВЕСЬ** ключ (он должен быть очень длинным, ~100+ символов)
7. Ключ должен начинаться с `sk_test_...`

### Шаг 2: Обновите секрет в Supabase

1. Откройте: https://supabase.com/dashboard/project/yffjnqegeiorunyvcxkn/settings/functions
2. Найдите секрет `STRIPE_SECRET_KEY`
3. Нажмите на три точки → **"Edit"** или **"Update"**
4. Вставьте **ПОЛНЫЙ** ключ из Stripe (убедитесь, что скопировали весь ключ)
5. Нажмите **"Save"**
6. Подождите 30 секунд

### Шаг 3: Проверьте формат ключа

Правильный формат:
- Начинается с `sk_test_` (для теста) или `sk_live_` (для продакшена)
- Длина примерно 100+ символов
- Не содержит пробелов или переносов строк

Неправильный формат:
- Начинается с `pk_test_` (это publishable key, не secret key!)
- Слишком короткий
- Содержит пробелы или переносы строк

### Шаг 4: Попробуйте снова

После обновления секрета:
1. Подождите 30 секунд
2. Обновите страницу приложения (F5)
3. Попробуйте снова нажать кнопку покупки

---

## Проверка:

После обновления ключа проверьте логи:
```
https://supabase.com/dashboard/project/yffjnqegeiorunyvcxkn/functions/purchase-create/logs
```

Если ключ правильный, ошибка должна исчезнуть и должно открыться окно Stripe Checkout.

---

## Если все еще не работает:

1. Убедитесь, что используете **Secret key**, а не **Publishable key**
2. Убедитесь, что вы в правильном режиме Stripe (Test/Live)
3. Попробуйте создать новый ключ в Stripe Dashboard:
   - Stripe Dashboard → Developers → API keys
   - Нажмите "Create secret key"
   - Скопируйте новый ключ
   - Обновите секрет в Supabase
