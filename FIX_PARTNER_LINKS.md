### ШАГ 2: Перезапустить Browser (Важно!)

**Для обновления Service Worker:**

1. **Закрой ВСЕ вкладки** `localhost:8080`
2. **Открой Chrome DevTools** (F12)
3. **Application → Service Workers**
4. Нажми **"Unregister"** для всех Service Workers
5. **Закрой и открой Chrome заново**

**Или проще:**

```

Очистить данные → Кэшированные изображения и файлы → Очистить
```
