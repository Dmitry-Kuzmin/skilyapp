# 🔑 Настройка своего Gemini API (бесплатно)

## Преимущества

- ✅ **Бесплатно**: 15 запросов в минуту, 1500 в день
- ✅ **Лучше для русского**: почти нет орфографических ошибок
- ✅ **Естественный язык**: более живые ответы
- ✅ **Независимость от Lovable**: свой API

---

## Шаг 1: Получить API ключ

1. Перейди на: **https://aistudio.google.com/apikey**
2. Войди через Google аккаунт
3. Нажми **"Create API Key"**
4. Выбери проект или создай новый
5. **Скопируй ключ** (начинается с `AIza...`)

⚠️ Сохрани ключ сразу!

---

## Шаг 2: Добавить ключ в Supabase

1. **Supabase Dashboard**: https://supabase.com/dashboard
2. Выбери проект **sdadim-dgt-prep**
3. **Settings** → **Edge Functions** → **Secrets**
4. Нажми **"Add a new secret"**
5. Заполни:
   - **Name**: `GEMINI_API_KEY`
   - **Value**: вставь ключ из шага 1
6. Нажми **"Save"**

---

## Шаг 3: Обновить Edge Function

**Supabase Dashboard** → **Edge Functions** → **ai-chat** → **Code**

**Скопируй код из файла:** `AI_CHAT_CODE.txt` (уже обновлён)

**Нажми** "Deploy updates"

---

## Шаг 4: Проверить работу

1. Перезагрузи приложение
2. Задай вопрос AI
3. В логах должно быть:
```
[AI Chat] Trying Google Gemini API...
[AI Chat] ✅ Gemini API success
```

4. Ответ должен быть:
   - ✅ Грамотный (почти без ошибок)
   - ✅ Естественный язык
   - ✅ Про Испанию

---

## Лимиты Gemini (бесплатно)

- ✅ **15 запросов в минуту** (RPM)
- ✅ **1,500 запросов в день** (RPD)
- ✅ **1 млн токенов в день**
- ✅ Этого хватит для сотен пользователей!

---

## Если Gemini недоступен

Система автоматически переключится на **Groq** (Mixtral).
