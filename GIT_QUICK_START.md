# ⚡ Git: Быстрая шпаргалка

## 🔄 Перед началом работы (ОБЯЗАТЕЛЬНО!)

```bash
gsync
# или вручную:
git fetch origin && git pull origin main
```

## 💾 После работы (ОБЯЗАТЕЛЬНО!)

```bash
git add .
git commit -m "feat: описание изменений"
git push origin main
```

## 📋 Полный цикл работы

```bash
# 1. Синхронизация перед началом
gsync

# 2. Работаешь...

# 3. Сохранение после работы
git add .
git commit -m "feat: что сделал"
git push origin main
```

## 🎯 Алиасы (уже настроены)

- `gsync` - синхронизация с удалённой веткой
- `gcheck` - проверка статуса
- `gpush "сообщение"` - коммит и пуш одной командой

## ⚠️ Если забыл синхронизироваться

```bash
# Если уже начал работать, но забыл синхронизироваться:
git stash  # сохраняет твои изменения
gsync      # синхронизируется
git stash pop  # возвращает твои изменения
```

## 🔧 Настройки (уже применены)

- ✅ Автоматический rebase при pull
- ✅ Автоматическое удаление удалённых веток
- ✅ Автоматическое сохранение незакоммиченных изменений

---

**Главное:** `gsync` перед работой, `git push` после работы! 🎯
