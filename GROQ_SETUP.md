# 📋 Инструкция по настройке Groq AI

## Шаг 1: Регистрация в Groq

1. Перейди на сайт: https://console.groq.com
2. Нажми кнопку **"Sign Up"** или **"Get Started"**
3. Зарегистрируйся через Google, GitHub или email
4. Подтверди email (если нужно)

## Шаг 2: Получение API ключа

1. После входа в консоль, перейди в раздел **"API Keys"** (в меню слева или в настройках)
2. Нажми кнопку **"Create API Key"**
3. Придумай название для ключа (например: "Sdadim DGT App")
4. Скопируй созданный API ключ (он выглядит так: `gsk_xxxxxxxxxxxxx`)
5. ⚠️ **ВАЖНО**: Сохрани ключ сразу, он показывается только один раз!

## Шаг 3: Добавление ключа в Supabase

1. Открой проект в Supabase: https://supabase.com/dashboard
2. Выбери свой проект
3. Перейди в **Settings** → **Edge Functions** → **Secrets**
4. Нажми **"Add a new secret"**
5. Заполни:
   - **Name**: `GROQ_API_KEY`
   - **Value**: вставь скопированный API ключ из Groq
6. Нажми **"Save"**

## Шаг 4: Проверка работы

1. После добавления ключа, код автоматически начнёт использовать Groq
2. Если Groq недоступен, система автоматически переключится на Lovable API (fallback)
3. Проверь работу AI помощника в приложении

## Лимиты Groq (бесплатно)

- ✅ **14,400 запросов в день** - это очень много!
- ✅ Очень быстрые ответы (< 1 секунда)
- ✅ Поддержка streaming
- ✅ Отличная поддержка русского языка

## Что делать если закончился лимит?

Если лимит Groq закончился, система автоматически переключится на Lovable API (если он настроен).
Ты можешь также добавить Gemini API как дополнительный вариант (инструкция в GEMINI_SETUP.md).

## Модели Groq

В коде используется модель `llama-3.1-70b-versatile` - это одна из лучших моделей.
Другие доступные модели:
- `llama-3.1-70b-versatile` (рекомендуется) - лучшая для русского
- `mixtral-8x7b-32768` - быстрая и качественная
- `llama-3.1-8b-instant` - самая быстрая, но менее умная

## Проблемы?

Если что-то не работает:
1. Проверь, что API ключ правильно добавлен в Supabase
2. Проверь название переменной: должно быть `GROQ_API_KEY`
3. Посмотри логи Edge Function в Supabase Dashboard
4. Убедись, что ключ активен в Groq Console
