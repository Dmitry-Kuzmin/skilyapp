# 🔑 Как получить Access Token для Supabase CLI

## Способ 1: Через браузер (самый простой)

### Шаг 1: Откройте Account Settings
1. В Supabase Dashboard нажмите на **ваш аватар/имя** в **правом верхнем углу**
2. В выпадающем меню выберите **"Account Settings"** или **"Account"**

Или откройте напрямую:
👉 **https://supabase.com/dashboard/account/tokens**

### Шаг 2: Создайте новый токен
1. На странице "Access Tokens" нажмите **"Generate new token"**
2. Введите название токена (например, "My Project Token")
3. Нажмите **"Generate token"**
4. ⚠️ **ВАЖНО**: Скопируйте токен сразу! Он показывается только один раз!

### Шаг 3: Используйте токен
После получения токена, сообщите мне, и я применю его автоматически.

---

## Способ 2: Через терминал (еще проще!)

Просто выполните в терминале:

```bash
export PATH="$HOME/.local/bin:$PATH"
supabase login
```

1. Откроется браузер автоматически
2. Авторизуйтесь в Supabase
3. Вернитесь в терминал - токен сохранится автоматически
4. Сообщите мне "готово", и я продолжу!

---

## 📍 Где находится в Dashboard:

```
Supabase Dashboard
└── Правый верхний угол (ваш аватар)
    └── Account Settings / Account
        └── Access Tokens (вкладка)
            └── Generate new token
```

---

## ⚠️ Важно:

- **Access Token** - это для Supabase CLI (для работы с проектами)
- **Service Role Key** - это для API (находится в Project Settings → API)
- Для нашей задачи нужен именно **Access Token**
