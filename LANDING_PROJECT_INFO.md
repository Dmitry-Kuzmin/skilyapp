# 📚 Информация о проекте Skilyapp для разработки лендинга

## 🎯 О проекте

### Основная информация
- **Название:** Skilyapp
- **Домен:** skilyapp.com
- **Тип:** Web-приложение + Telegram Mini App
- **Назначение:** Подготовка к экзамену DGT (испанские водительские права)
- **Целевая аудитория:** Русскоязычные жители Испании

### Технологический стек
- **Frontend:** React 18 + TypeScript
- **Styling:** Tailwind CSS
- **UI Components:** shadcn/ui (Radix UI)
- **Routing:** React Router v7
- **Backend:** Supabase (PostgreSQL, Edge Functions)
- **Платежи:** Stripe + Telegram Stars
- **Deployment:** Vercel
- **Build:** Vite

---

## 🎮 Основные функции приложения

### 1. Тесты DGT

#### Типы тестов
- **Последовательные тесты** - по темам и подтемам
- **Экзаменационные тесты** - симуляция реального экзамена (30 вопросов, 30 минут)
- **DGT тесты** - официальные вопросы DGT (4012+ вопросов)
  - Категория A1 (мотоциклы): 899 вопросов
  - Категория B (легковые авто): 2947 вопросов
  - Категория D (автобусы): 166 вопросов

#### Особенности
- Таймер 30 минут для экзаменационных тестов
- 90% для успешной сдачи
- Изображения к вопросам
- Подробные объяснения ответов
- Статистика по результатам
- Конфетти при успешной сдаче

#### Настройки теста
- Voice over (озвучка вопросов)
- Answer popularity (процент выбора ответов)
- Режим множественного выбора
- Адаптивные тесты
