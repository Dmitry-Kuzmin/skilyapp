## 🐛 Известные ограничения

1. **Фильтр "Друзья":**
   - Работает только если есть записи в таблице `referrals`
   - Если друзей нет - показывается пустой список

2. **Фильтр "Страна":**
   - Использует `language_code` из профиля
   - Если `language_code` NULL - фильтр не работает

3. **Блок "Моё место":**
   - Показывается только если пользователь НЕ в топе
   - Требует `profileId` для работы
