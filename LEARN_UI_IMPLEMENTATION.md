## 💡 Особенности реализации

### Умное определение страны
- Приоритет 1: Сохраненный выбор пользователя
- Приоритет 2: IP геолокация (через ipapi.co)
- Приоритет 3: Язык браузера/Telegram
- Fallback: Первая доступная страна

### Адаптеры данных
Все данные преобразуются к единому формату `UniversalQuestion`, что позволяет:
- Переиспользовать `TestSession` без изменений
- Легко добавлять новые страны
- Изолировать логику стран

### Премиум UI
- Градиенты и тени
- Плавные анимации (framer-motion)
- Адаптивный дизайн
- Индикаторы прогресса
- Hover эффекты
