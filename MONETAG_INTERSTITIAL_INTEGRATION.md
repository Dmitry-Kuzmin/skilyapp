# 🎯 Monetag Native Banner (Interstitial) - Интеграция

## 📋 Что это:
- **Формат:** Native Banner (Interstitial)
- **Zone ID:** `10323437`
- **Тип:** Полноэкранная реклама, появляется между переходами

---

## ✅ Рекомендуемые места для показа:

### 🥇 Приоритет #1: После закрытия результатов теста/дуэли
**Место:** При возврате на дашборд после просмотра результатов

**Почему:**
- Пользователь уже завершил действие
- Переход уже происходит (не мешает текущему действию)
- Естественный момент для рекламы

**Где интегрировать:**
- `TestResults.tsx` - кнопка "Вернуться на главную"
- `DuelResult.tsx` - кнопка "Вернуться в меню"

---

### 🥈 Приоритет #2: При переходе между разделами
**Места:**
- Tests → Games
- Games → Duel
- Dashboard → Tests (опционально)

**Почему:**
- Переход между разделами - естественное место для Interstitial
- Не прерывает текущее действие
- Показывается только при навигации

---

### 🥉 Приоритет #3: При первом входе (опционально)
**Место:** При открытии приложения впервые в сессии

**Почему:**
- Один раз за сессию
- Не раздражает частыми показами
- Но может быть менее эффективно (пользователь еще не вовлечен)

---

## 🚀 План интеграции:

### Шаг 1: Создан хук `useInterstitialBanner`
- ✅ Расположение: `src/hooks/useInterstitialBanner.ts`
- ✅ Проверка Premium статуса
- ✅ Проверка Telegram Mini App
- ✅ Один раз за сессию (sessionStorage)

### Шаг 2: Интеграция в TestResults
- Добавить при нажатии "Вернуться на главную"
- Показывать с небольшой задержкой (300ms)

### Шаг 3: Интеграция в DuelResult
- Добавить при нажатии "Вернуться в меню"
- Показывать с небольшой задержкой (300ms)

### Шаг 4: Опционально - в навигации
- При переходе между основными разделами
- Через роутер или компонент навигации

---

## 💡 Рекомендация:

**Стартовать с приоритета #1** (после результатов):
1. Это самый безопасный вариант (UX-wise)
2. Пользователь уже завершил действие
3. Реклама не мешает основному flow

**Добавить приоритет #2** (между разделами) можно позже, если нужен больший охват.

---

## 🔧 Технические детали:

### Zone ID:
- `10323437` - Native Banner (Interstitial)

### SessionStorage ключ:
- `monetag_interstitial_shown` - помечает, что баннер уже показан

### Задержка:
- По умолчанию 300ms (настраивается)
- Чтобы переход был плавным

---

## ⚠️ Важно:

- Interstitial показывается **один раз за сессию**
- НЕ показывается Premium пользователям
- НЕ показывается в Telegram Mini App
- Показывается только в веб-версии
