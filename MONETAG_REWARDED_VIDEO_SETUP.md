# 🎯 Monetag Rewarded Video для веб-версии - Инструкция

## ✅ Текущая ситуация:

У тебя уже есть **3 зоны** в Monetag:
1. **10323509** - Epic tag - Native Banner (Interstitial)
2. **10323437** - Skillful tag - Native Banner (Interstitial) ← **Используем этот!**
3. **10323401** - Talented tag - Vignette Banner

## 💡 Решение:

**Native Banner (Interstitial)** в Monetag может работать как **Rewarded Video** для веб-версии!

Я обновил код, чтобы использовать **Zone ID: 10323437** (Skillful tag).

2. ✅ Создан файл `src/lib/monetag.ts` для работы с Monetag
3. ✅ Обновлен `src/hooks/useRewardedAd.ts` для поддержки веб-версии

## 📋 Что нужно проверить:

### 1. Настройки зоны в Monetag:

1. Открой зону **10323437** (Skillful tag) в Monetag
2. Нажми **"<> Get tag"**
3. Проверь, что там есть функция `show_10323437()`
4. Если есть - всё готово! ✅

### 2. Если нужен отдельный Rewarded Video блок:

Если ты хочешь создать **специальный блок** только для Rewarded Video:

1. В боковом меню Monetag нажми **"Telegram Mini Apps"** (даже для веб-версии)
2. Нажми **"+ Create new"**
3. Укажи URL: `https://skilyapp.com`
4. Выбери формат **"Rewarded Interstitial"** (если доступен)
5. Получи новый Zone ID
6. Обнови код с новым Zone ID

---

## 🧪 Как протестировать:

1. Открой приложение в **веб-браузере** (НЕ в Telegram)
2. Перейди на страницу с дуэлями
3. Попробуй открыть слот OVERCLOCK (с иконкой Play)
4. Должна открыться модалка с кнопкой "Смотреть видео"
5. Нажми "Смотреть видео"
6. Должна показаться реклама Monetag

---

## ⚠️ Если не работает:

1. Проверь консоль браузера (F12) - там должны быть логи от Monetag
2. Проверь, что SDK загрузился: `window.show_10323437` должно существовать
3. Если ошибка - пришли мне скриншот консоли

---

## 📝 Примечание:

**Native Banner (Interstitial)** в Monetag работает немного иначе, чем классический Rewarded Video:
- Может показываться как полноэкранный баннер
- Пользователь может закрыть его (в отличие от AdsGram)
- Но это нормально для веб-версии!
