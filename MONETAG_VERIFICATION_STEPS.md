# 🔧 Шаги для успешной верификации Monetag

## ⚠️ Проблема решена

PWA service worker переименован с `sw.js` на `pwa-sw.js`, чтобы освободить `sw.js` для Monetag верификации.

---

## 📋 Что нужно сделать СЕЙЧАС:

### Шаг 1: Дождись нового деплоя
- Изменения закоммичены и запушены
- Деплой должен автоматически начаться на Vercel
- Подожди ~1-2 минуты пока деплой завершится

### Шаг 2: Проверь доступность файла
После деплоя проверь, что файл доступен:
```bash
curl https://skilyapp.com/sw.js
```

Должен вернуться:
```javascript
self.options = {
    "domain": "5gvci.com",
    "zoneId": 10323310
}
self.lary = ""
importScripts('https://5gvci.com/act/files/service-worker.min.js?r=sw')
```

### Шаг 3: Верификация в Monetag
1. Вернись в панель Monetag
2. Нажми кнопку **"Verify"**
3. Должно пройти успешно! ✅

---

## 🔍 Если всё ещё не работает:

### Проверь кэш
- Очисти кэш браузера (Ctrl+Shift+R или Cmd+Shift+R)
- Или попробуй в режиме инкогнито
- Или подожди 5-10 минут (кэш CDN/Vercel)

### Проверь файл напрямую
Открой в браузере: `https://skilyapp.com/sw.js`
- Должен показать код верификации (не PWA service worker)

### Свяжись с поддержкой Monetag
Если файл правильный, но верификация не проходит:
- Напиши в поддержку Monetag
- Объясни, что файл доступен по URL
- Попроси проверить вручную
