# 👀 Чеклист мониторинга в час "Ч"

**Критично:** Во время запуска рекламы следите за метриками!

---

## 📊 Что мониторить

### 1. Active Connections (КРИТИЧНО!)

**Где:** Supabase Dashboard → **Database** → **Health** → **Active Connections**

**Что делать:**

| Connections | Статус | Действие |
|-------------|--------|----------|
| **0-30** | ✅ Норма | Продолжайте работу |
| **30-40** | ⚠️ Внимание | Следите внимательнее |
| **40-50** | 🚨 **КРИТИЧНО** | **СРОЧНО апгрейд на Pro!** |
| **50-60** | 🔴 **ПАДЕНИЕ** | Проект может упасть в любой момент |

**Алерт:** Настройте уведомление при > 40 connections

---

### 2. Edge Function Invocations

**Где:** Supabase Dashboard → **Edge Functions** → **Usage**

**Лимиты:**
- Free план: 500K/месяц
- Pro план: 2M/месяц

**Что делать:**
- Следите за дневным использованием
- При > 15K/день на Free плане → апгрейд

---

### 3. Real-time Connections

**Где:** Supabase Dashboard → **Database** → **Health** → **Real-time**

**Лимиты:**
- Free план: 200 connections
- Pro план: 500 connections

**Что делать:**
- Следите за активными подписками
- При > 150 connections → апгрейд

---

### 4. API Requests

**Где:** Supabase Dashboard → **API** → **Usage**

**Лимиты:**
- Free план: 50K/месяц
- Pro план: 500K/месяц

**Что делать:**
- Следите за дневным использованием
- При > 1.5K/день на Free плане → апгрейд

---

## 🚨 Критические алерты

### Настройте уведомления:

1. **Supabase Dashboard** → **Settings** → **Alerts**
2. Добавьте алерты на:
   - ✅ Database Connections > 40
   - ✅ Edge Function Invocations > 15K/день
   - ✅ Real-time Connections > 150
   - ✅ API Requests > 1.5K/день

---

## 📈 План действий при росте

### Сценарий 1: Active Connections > 40

**Действие:**
1. 🚨 **НЕМЕДЛЕННО** апгрейд на Pro план
2. Проверьте, что Connection Pooling работает
3. Отключите Spend Cap (на Pro плане можно)

### Сценарий 2: Edge Functions > 15K/день

**Действие:**
1. ⚠️ Планируйте апгрейд на Pro
2. Оптимизируйте вызовы функций
3. Используйте кэширование где возможно

### Сценарий 3: Real-time > 150

**Действие:**
1. ⚠️ Планируйте апгрейд на Pro
2. Отключите ненужные real-time подписки
3. Используйте Feature Flags для отключения real-time

---

## ✅ Итоговый чеклист

**Перед запуском рекламы:**
- [ ] Откройте Dashboard → Database → Health
- [ ] Настройте алерты на критические метрики
- [ ] Подготовьте план апгрейда на Pro (если нужно)

**Во время запуска:**
- [ ] Следите за Active Connections каждые 5 минут
- [ ] Проверяйте Edge Function invocations
- [ ] Мониторьте Real-time connections

**При росте:**
- [ ] При 40+ connections → апгрейд на Pro
- [ ] При превышении лимитов → оптимизация или апгрейд
