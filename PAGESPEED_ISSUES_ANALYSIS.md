## 🔍 Возможные проблемы

### 1. Проблемы с деплоем
**Симптомы:**
- Старая версия в production
- Изменения не применились
- Кэш браузера/CDN

**Проверка:**
- Проверить версию в production (console.log, версия bundle)
- Проверить, что изменения задеплоены
- Очистить кэш и перепроверить

### 2. Проблемы с preload/modulepreload
**Симптомы:**
- JS chunks не загружаются с правильным приоритетом
- FCP/LCP ухудшились
- Warnings в консоли о preload

**Возможные причины:**
- Неправильные пути в preload ссылках
- Конфликт между статическими и динамическими preload
- Проблемы с CORS/crossorigin

### 3. Проблемы с bundle splitting
**Симптомы:**
- Большой initial bundle
- Медленная загрузка
- Unused JavaScript увеличился

**Возможные причины:**
- Изменения в vite.config.ts не применились
- Проблемы с manualChunks
- Конфликты зависимостей

### 4. Проблемы с изображениями
**Симптомы:**
- LCP ухудшился
- Изображения не загружаются с правильным приоритетом
- Проблемы с fetchPriority

**Возможные причины:**
- Изменения в Dashboard.tsx не применились
- Проблемы с внешними изображениями (noise.svg)
- Кэш браузера

### 5. Проблемы с CSS
**Симптомы:**
- Unused CSS увеличился
- Render-blocking CSS
- Большой CSS bundle

**Возможные причины:**
- Изменения в index.css не применились
- Проблемы с Tailwind purge
- Динамические классы не попадают в purge
