## ⚠️ Почему Score только 64 (а не 90)?

### Главный тормоз: `index.js` (670KB)

**Проблема:**
- Первые **4.6 секунды** экран пустой (или скелетон)
- На **4.6с** появляется контент (FCP/LCP)
- Основной код (`index.js`) всё ещё грузится долго

### Opportunities из PageSpeed:

1. **Reduce unused JavaScript — Potential savings 350 KiB**
   - Нужно найти и убрать неиспользуемый код
   - Проверить tree-shaking

2. **Reduce unused CSS — Potential savings 45 KiB**
   - Проверить PurgeCSS/Tailwind
   - Убедиться что неиспользуемые классы удаляются

3. **Serve images in next-gen formats** (если есть картинки)
   - Конвертировать в WebP
   - Оптимизировать размер

### Что ещё нужно улучшить:

1. **Speed Index: 6.6s** - без изменений
   - Возможно, проблема в загрузке изображений
   - Нужно оптимизировать LCP элемент

2. **FCP/LCP: ~4.6s** - всё ещё долго
   - Нужно оптимизировать `index.js` (670KB → ~400-500KB)
   - Lazy-load больше компонентов
