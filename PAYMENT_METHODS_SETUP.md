# 💳 Настройка методов оплаты без autónomo

## 📋 Ситуация

В Испании Stripe требует регистрации как autónomo (самозанятый) для приема платежей. До оформления документов используются альтернативные методы.

## ✅ Решение

### 1. **Telegram Stars** (Основной метод) ⭐

**Статус:** ✅ Работает без регистрации  
**Платформа:** Только Telegram Mini App  
**Комиссия:** ~30% (встроена в цены Telegram)  
**Интеграция:** ✅ Уже реализована

**Преимущества:**
- Не требует юридической регистрации
- Нативная интеграция в Telegram
- Мгновенные платежи
- Уже работает в приложении

**Использование:**
- Автоматически показывается в Telegram Mini App
- Кнопка оплаты появляется для всех пакетов (Premium, монеты)

---

### 2. **PayPal** (Альтернатива для веб)

**Статус:** ⚠️ В разработке  
**Платформа:** Веб-версия (не Telegram)  
**Комиссия:** ~3.4% + €0.35  
**Требования:** Можно использовать без autónomo на начальном этапе

**Преимущества:**
- Работает в обычном браузере
- Не требует autónomo (на начальном этапе)
- Широко используется в Испании

**⚠️ Важно:**
- Для коммерческой деятельности в Испании может потребоваться регистрация
- Рекомендуется проконсультироваться с налоговым консультантом

---

### 3. **Stripe** (Отключен до оформления autónomo)

**Статус:** ❌ Отключен  
**Причина:** Требует регистрации autónomo в Испании  
**Когда включить:** После оформления документов

**Как включить:**
1. Оформить autónomo в Испании
2. Открыть `src/lib/payment-config.ts`
3. Изменить `stripeEnabled: true`
4. Настроить секреты в Supabase Dashboard

---

## 🔧 Текущая конфигурация

Файл: `src/lib/payment-config.ts`

```typescript
export const PAYMENT_CONFIG: PaymentConfig = {
  stripeEnabled: false,      // ❌ Отключено до autónomo
  paypalEnabled: true,       // ✅ Можно использовать
  telegramStarsEnabled: true, // ✅ Работает
};
```

---

## 📱 Как это работает

### В Telegram Mini App:
1. **Telegram Stars** — основной метод (приоритет)
2. **PayPal** — альтернатива (если включен)
3. **Stripe** — скрыт (отключен в конфиге)

### В веб-версии:
1. **PayPal** — основной метод (если включен)
2. **Stripe** — скрыт (отключен в конфиге)
3. **Telegram Stars** — недоступен (только в Telegram)

## 🚀 Следующие шаги

### Вариант 1: Использовать только Telegram Stars (рекомендуется)
- ✅ Уже работает
- ✅ Не требует регистрации
- ✅ Нативная интеграция
- ⚠️ Ограничение: только для пользователей Telegram

### Вариант 2: Добавить PayPal для веб-версии
1. Зарегистрироваться в PayPal Business
2. Получить API ключи
3. Создать Edge Function для PayPal
4. Обновить UI для показа PayPal кнопок

### Вариант 3: Оформить autónomo и включить Stripe
1. Получить NIE (если нет)
2. Зарегистрироваться в Seguridad Social как autónomo
3. Открыть банковский счет
4. Включить Stripe в конфиге
5. Настроить секреты

---

## 📝 Примечания

- **Telegram Stars** — лучший вариант для начала, не требует документов
- **PayPal** — можно использовать временно, но лучше проконсультироваться с юристом
- **Stripe** — включить после оформления autónomo

## ⚖️ Юридические вопросы

**⚠️ ВАЖНО:** Информация в этом документе не является юридической консультацией. 

Для коммерческой деятельности в Испании рекомендуется:
1. Проконсультироваться с налоговым консультантом (asesor fiscal)
2. Оформить autónomo при достижении определенного дохода
3. Вести учет всех транзакций для налоговой отчетности

**Льготы для новых autónomo:**
- Сниженные взносы в первые 2 года
- Льготный режим для доходов до €12,000/год
