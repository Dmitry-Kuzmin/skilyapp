## 🚀 Быстрый старт с Retool

### Шаг 1: Создай аккаунт
1. Зайди на https://retool.com
2. Создай бесплатный аккаунт

### Шаг 2: Подключи Supabase
1. Retool → Resources → Create new → Database → PostgreSQL
2. Вставь connection string из Supabase:
   ```
   postgresql://postgres:[PASSWORD]@[HOST]:5432/postgres
   ```
3. Тест подключения → Save

### Шаг 3: Создай админку
1. Create new app → Blank app
2. Добавь компоненты:
   - **Table** → подключи к `pdd_russia_questions`
   - **Form** → для редактирования вопроса
   - **Button** → для сохранения
3. Настрой фильтры (билет, категория, тема)

### Результат:
✅ Готовая админка за 30 минут без единой строчки кода!
