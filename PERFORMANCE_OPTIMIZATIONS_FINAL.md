## 📈 Как проверить производительность

### В dev режиме
1. Откройте приложение
2. В правом нижнем углу появится кнопка Performance Monitor
3. Нажмите на неё для просмотра метрик в реальном времени

### В production
1. Откройте DevTools → Performance
2. Запишите профиль производительности
3. Проверьте метрики Web Vitals
4. Анализируйте Long Tasks

### Bundle анализ
```bash
npm run build:analyze
```
Откроется визуализация bundle size в браузере.
