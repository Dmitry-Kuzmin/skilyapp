# 📊 ИНСТРУКЦИЯ: Тестирование Performance Tab

**Цель:** Получить скриншот Performance Tab с доказательством, что Long Tasks < 50ms

---

## ⚠️ ТЕКУЩАЯ СИТУАЦИЯ:

В production есть ошибка с динамическим импортом `DuelPassLeaderboardModal`, которая **не связана** с изменениями в `NotificationsPanel`. Это мешает тестированию.

**Статус:** Ожидаем деплой исправлений.

---

## 🧪 КАК ПРОВЕРИТЬ (после деплоя):

### Шаг 1: Открой DevTools
1. Открой https://skilyapp.com/dashboard
2. Нажми `F12` или `Cmd+Option+I` (Mac)
3. Перейди на вкладку **Performance**

### Шаг 2: Настрой CPU slowdown
1. В Performance Tab нажми на иконку **шестеренки** (⚙️) в правом верхнем углу
2. Выбери **CPU: 4x slowdown**
3. Это симулирует медленный CPU для более точного тестирования

### Шаг 3: Запиши профиль
1. Нажми кнопку **Record** (круглая красная кнопка) или `Cmd+E` (Mac)
2. **Открой панель уведомлений** (иконка колокольчика в правом верхнем углу)
3. **Активно скролли** список уведомлений вверх-вниз в течение **5 секунд**
4. Нажми **Stop** (кнопка Stop или `Cmd+E`)

### Шаг 4: Проверь результаты
1. В профиле найди полосу **"Main"** (главный поток)
2. Проверь:
   - ✅ **Нет красных флажков** (Long Tasks)
   - ✅ **Scripting time < 50ms** при открытии панели
   - ✅ **FPS стабильно 60** при скролле
   - ✅ **Нет блокирующих задач** > 50ms

### Шаг 5: Сделай скриншот
1. Увеличь масштаб профиля (колесико мыши или `Cmd/Ctrl + Plus`)
2. Сделай скриншот полосы **"Main"**
3. Убедись, что видно:
   - Временную шкалу
   - Полосу "Main"
   - Отсутствие красных флажков

## 📝 ПРИМЕЧАНИЯ:

1. **Виртуализация работает только для списков > 10 элементов**
   - Если у тебя < 10 уведомлений, виртуализация не активируется
   - Это нормально - для малых списков она не нужна

2. **CPU 4x slowdown критичен**
   - Без него Long Tasks могут быть не видны
   - Это симулирует медленный CPU (например, мобильный)

3. **Активный скролл важен**
   - Просто открыть панель недостаточно
   - Нужно активно скроллить, чтобы увидеть производительность виртуализации
