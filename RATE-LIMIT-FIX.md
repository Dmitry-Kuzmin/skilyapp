# ✅ ПРОБЛЕМА РЕШЕНА: Rate Limit

## 🔍 Что было:

### Ошибки в логах:
- ⚠️ "Ошибка, повтор через 3с..."
- 429 TooManyRequests
- 503 ServiceUnavailable

### Причина:
**gemini-2.0-flash-exp** — исчерпан лимит!
- RPM: **10/10** (Free Tier)
- Каждая картинка = 2 запроса (Vision + Imagen)
- 10 запросов / 2 = **только 5 картинок в минуту**
- Задержка 3 сек = слишком быстро!

## ✅ Что исправлено:

### 1. Модель изменена:
**БЫЛО:**
```javascript
gemini-2.0-flash-exp  // Экспериментальная, лимит 10 RPM
```

**СТАЛО:**
```javascript
gemini-1.5-flash      // Стабильная, лимит выше
```

### 2. Задержки увеличены:
**БЫЛО:**
```javascript
delayBetweenImages: 3000  // 3 секунды
```

**СТАЛО:**
```javascript
delayBetweenImages: 7000  // 7 секунд (безопасно для Free Tier)
delayBetweenVisionAndImagen: 3000  // 3 сек между вызовами
```

### 3. Математика:
- 1 картинка = Vision (1 запрос) + Imagen (1 запрос) = **2 запроса**
- Задержка 7 сек между картинками
- **60 сек / 7 сек = ~8 картинок в минуту**
- 8 картинок × 2 запроса = **16 API calls/минуту**
- Безопасный запас для лимита 10-15 RPM ✅
