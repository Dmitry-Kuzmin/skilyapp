## 🎨 Design System

### Цветовая Палитра (Строгая!):

```css
/* Backgrounds */
#0a0e1a  → Main background
#151923  → Cards
#1a1f2e  → Hover state

/* Borders */
#1e293b  → Default border
#2d3748  → Hover border

/* Text */
#f1f5f9  → Primary text (white)
#94a3b8  → Secondary text
#64748b  → Tertiary text

/* Accent (Minimal!) */
#6366f1  → Primary (indigo) - ТОЛЬКО для кнопок и акцентов
#10b981  → Success (зеленый) - ТОЛЬКО для success states
#f59e0b  → Warning (янтарный) - ТОЛЬКО для warning states
```

**Правило:** НЕТ других цветов! Только эти!


```javascript

```

**4. Обнови лендинг:**
