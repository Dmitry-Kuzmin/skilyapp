# 🔧 Пошаговая установка Rollbar

## ⚠️ ВАЖНО: Сейчас вы на странице создания нового проекта

**НЕ создавайте новый проект!** Нам нужно установить интеграцию в существующий проект `sdadim-dgt-prep`.

---

## 📋 Шаг 1: Вернуться к проекту

1. **Закройте** текущую страницу "New Project" (или просто перейдите назад)
2. Откройте ваш проект: **sdadim-dgt-prep**
3. Перейдите в настройки проекта

---

## 📋 Шаг 2: Найти раздел Integrations

**Вариант A: Через навигацию проекта**
1. Откройте проект **sdadim-dgt-prep** в Vercel
2. В верхней навигации найдите вкладку **"Integrations"**
3. Нажмите на неё

**Вариант B: Через Settings**
1. Откройте проект **sdadim-dgt-prep**
2. Перейдите в **"Settings"** (в верхней навигации)
3. В левом боковом меню найдите **"Integrations"**
4. Нажмите на неё

---

## 📋 Шаг 3: Установить Rollbar

1. На странице Integrations нажмите **"Browse Marketplace"** (если видите "No Integrations Installed")
2. Найдите **"Rollbar"** в списке интеграций
3. Нажмите на **"Rollbar"**
4. Нажмите кнопку **"Add Integration"** или **"Install"**
5. Следуйте инструкциям на экране:
   - Подтвердите установку
   - Выберите проект **sdadim-dgt-prep** (если предложат)
   - Примите разрешения

---

## 📋 Шаг 4: Получить API токен Rollbar

После установки интеграции:

1. В Vercel вы получите переменную окружения `ROLLBAR_ACCESS_TOKEN`
2. **ИЛИ** откройте Rollbar Dashboard напрямую:
   - Перейдите на https://rollbar.com
   - Войдите (или создайте аккаунт, если нужно)
   - Создайте новый проект (если ещё нет)
   - Перейдите в **Settings** → **Projects** → выберите проект
   - Скопируйте **"Server-side access token"** (post_server_item)

---

## 📋 Шаг 5: Добавить токен в Vercel Environment Variables

1. В Vercel откройте проект **sdadim-dgt-prep**
2. Перейдите в **Settings** → **Environment Variables**
3. Добавьте новую переменную:
   - **Name**: `VITE_ROLLBAR_ACCESS_TOKEN`
   - **Value**: вставьте токен из шага 4
   - **Environment**: выберите **Production**, **Preview**, **Development** (все три)
4. Нажмите **"Save"**

---

## 📋 Шаг 6: Установить пакет Rollbar

После того, как я обновлю код, выполни в терминале:

```bash
npm install rollbar
```

## 🧪 Проверка работы

После деплоя:

1. Откройте Rollbar Dashboard: https://rollbar.com
2. Перейдите в ваш проект
3. В разделе **"Items"** (или **"Errors"**) должны появиться ошибки из production

**Чтобы протестировать:**
- Создайте тестовую ошибку (например, в консоли браузера: `throw new Error('Test error')`)
- Ошибка должна появиться в Rollbar в течение нескольких секунд
