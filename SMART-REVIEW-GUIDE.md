# 🎯 Smart Review System - Готово!

## Как это работает:

### 📊 Процесс генерации с паузами:

```
[Запуск] → Генерирует 10 картинок
           ↓
       [ПАУЗА] ⏸️
           ↓
   "10 готовы к проверке!"
           ↓
   Ты проверяешь ИЛИ не проверяешь
           ↓
   [Автопродолжение через 10 мин]
           ↓
   Генерирует следующие 10
           ↓
   [Повтор]
```

## 🚀 Команды для запуска:

### **Режим 1: С паузами (рекомендуется)**
```bash
node scripts/generate-images-batch.js
```
- Генерит пачками по 10
- После каждых 10 → **ПАУЗА**
- Ждет проверки ИЛИ 10 минут
- Продолжает автоматически

### **Режим 2: Без пауз (batch)**
```bash
node scripts/generate-images-batch.js --no-pause
```
- Генерит все подряд
- Проверишь потом сразу все

## 🎨 Процесс проверки:

### Что происходит после пачки:

1. **Консоль показывает:**
   ```
   ============================================================
   🎨 Пачка из 10 картинок сгенерирована!
   📊 Всего готово к проверке: 10
   ============================================================
   
   ⏸️  ПАУЗА ДЛЯ ПРОВЕРКИ
   
   Варианты:
     1. Открой Dashboard (http://localhost:3030)
     2. Перейди на таб "Валидация"
     3. Проверь новые картинки (10 шт.)
   
   Генерация продолжится автоматически через 10 минут
   или сразу после проверки.
   ```

2. **Ты открываешь Dashboard:**
   - Видишь: "10 новых картинок готовы"
   - Переходишь на "Валидация"
   - Быстро проверяешь (Y/N/R)

3. **После проверки:**
   - Нажимаешь "Продолжить генерацию"
   - Генерация продолжается сразу

4. **Если не проверил:**
   - Через 10 минут генерация продолжится сама
   - Проверишь позже (накопится больше)

## 📁 Файлы системы:

- `data/generated-images/` — сгенерированные картинки
- `data/generated-images/originals/` — оригиналы
- `data/review-queue.json` — очередь на проверку
- `data/image-gen-checkpoint.json` — прогресс
- `data/continue-generation.flag` — флаг продолжения (создается автоматически)

## 🎯 Сценарии использования:

### **Сценарий 1: Активная работа**
```
15:00 → Запустил генерацию
15:05 → 10 готово, проверил за 2 мин
15:07 → Продолжил, генерит следующие 10
15:12 → 20 готово, проверил
...repeat
```

### **Сценарий 2: Фоновая генерация**
```
15:00 → Запустил генерацию
15:05 → 10 готово, УШЕЛ
15:15 → Автопродолжение (таймаут)
15:20 → 20 готово, УШЕЛ
15:30 → Автопродолжение
16:00 → Вернулся, проверил все 60 разом
```

### **Сценарий 3: Batch без пауз**
```
15:00 → Запустил с --no-pause
16:00 → Все 100 готово
16:05 → Проверил все за раз
```

## ⚙️ Настройки (в CONFIG):

```javascript
batchSize: 10,              // Пачка по 10 картинок
pauseForReview: true,       // Пауза после пачки
autoContinueAfter: 600000,  // 10 минут таймаут
delayBetweenImages: 3000    // 3 сек между картинками
```

## 💡 Рекомендации:

1. **Первый раз:** Запусти на 10 картинках, проверь качество
2. **Если качество ОК:** Запусти на весь Topic-01 (17 тестов = ~450 картинок)
3. **Проверяй пачками:** Не копи больше 30-40, проверяй порциями
4. **Отклоняй сразу:** Плохие картинки = не сохраняются
5. **Регенерация:** Если почти хорошо, допиши промпт
