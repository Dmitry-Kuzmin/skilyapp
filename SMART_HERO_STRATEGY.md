# Smart Hero Strategy - Документация

## 🎯 **Концепция: Один сайт, две стратегии продаж**

Лендинг автоматически адаптирует **смысл** и **эмоцию** под целевую аудиторию в зависимости от выбранного языка.

## 🎨 **Ключевые отличия по языкам**

### **🇷🇺/🇬🇧 Для Экспатов:**
- ✅ Акцент на **"языковой барьер"**
- ✅ Слова: "уверенность", "переводы", "поддержка", "на русском/английском"
- ✅ Тон: спокойный, поддерживающий, защищающий

- ✅ Тон: энергичный, молодежный, современный

## 🛠️ **Техническая реализация**

### **Файл:** `src/translations/landing.ts`

Каждый язык имеет **разные тексты по смыслу**, а не просто переводы:

```typescript
// RU - для экспатов
hero: {
  titleTop: "Твоя уверенность за рулем",
  descriptionHighlight: "Забудь про страх экзамена и языковой барьер"
}

// ES - для местных
hero: {
  titleTop: "Tu teórico a la primera",
  descriptionHighlight: "Olvida los manuales aburridos"
}
```

## 📈 **Эффект**

### **Было (универсальный текст):**
```
"Сдай теоретический экзамен DGT с первого раза"
```
- ❌ Говорит ЧТО делает продукт
- ❌ Не учитывает боли аудитории
- ❌ Одинаково для всех

### **Стало (Smart Hero):**

**Для русского экспата:**
```
"Твоя уверенность за рулем начинается здесь
Забудь про страх экзамена и языковой барьер"
```
- ✅ Говорит КАК решает проблему
- ✅ Попадает в боль (языковой барьер)
- ✅ Создает эмоцию (уверенность)

**Для испанца:**
```
"Tu teórico a la primera. Sin dramas.
Olvida los manuales aburridos"
```
- ✅ Говорит о РЕЗУЛЬТАТЕ (с первого раза)
- ✅ Попадает в боль (скучные учебники)
- ✅ Создает эмоцию (легкость, драйв)

## 🚀 **Автоопределение языка**

При первом заходе на сайт система:

1. **Проверяет язык браузера** (`navigator.language`)
2. **Определяет тип аудитории:**
   - `ru`, `en`, `uk`, `by` → Экспаты (фокус на языковой барьер)
   - `es`, `ca` → Местные (фокус на скорость)
3. **Подставляет соответствующие тексты**
4. **Пользователь может переключить вручную**

## 🎓 **Лучшие практики**

### **DO ✅:**
- Адаптируй **смысл**, а не только слова
- Попадай в **реальную боль** аудитории
- Создавай нужную **эмоцию**
- Используй **разные слова-триггеры**

### **DON'T ❌:**
- Не делай просто перевод
- Не используй универсальные фразы
- Не игнорируй культурные особенности
- Не пиши одинаково для всех
