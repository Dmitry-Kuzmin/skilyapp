# 🚀 Настройка Upstash Redis для Rate Limiting

## 📝 Заполнение формы "Create Database"

### 1. **Name** (Имя базы данных)
```
sdadim-rate-limit
```
или
```
skilyapp-rate-limit
```

**Рекомендация:** Используйте короткое, понятное имя, связанное с вашим проектом.

---

### 2. **Primary Region** (Основной регион)

**Выберите регион, ближайший к вашим пользователям:**

Для пользователей в **Испании/Европе:**
- ✅ **eu-west-1** (Ireland) - рекомендуется
- ✅ **eu-central-1** (Frankfurt)
- ✅ **eu-north-1** (Stockholm)

Для пользователей в **России/СНГ:**
- ✅ **eu-west-1** (Ireland) - ближайший
- ✅ **eu-central-1** (Frankfurt)

**Рекомендация:** Выберите **eu-west-1 (Ireland)** - хорошая задержка для Европы и России.

---

### 3. **Read Regions** (Регионы для чтения)

**Оставьте пустым** (не выбирайте ничего)

**Почему:**
- Read regions доступны только на платных планах
- Для rate limiting не нужны (мы только пишем и читаем счетчики)
- Бесплатный план не поддерживает read regions

---

### 4. **Eviction** (Вытеснение данных)

**Включите** (переключите в положение "ON")

**Почему:**
- Rate limiting использует временные ключи (TTL)
- Eviction автоматически удалит старые ключи
- Экономит память и предотвращает переполнение

---

### 5. **Payment Method** (Способ оплаты)

**Для бесплатного плана:**
- Можно пропустить (если выбираете Free план)
- Или добавить карту (не будет списаний на Free плане)

**Рекомендация:** Добавьте карту, чтобы не было проблем при переходе на платный план в будущем.

---

## ✅ Итоговые настройки

| Поле | Значение |
|------|----------|
| **Name** | `sdadim-rate-limit` |
| **Primary Region** | `eu-west-1` (Ireland) |
| **Read Regions** | *(оставить пустым)* |
| **Eviction** | ✅ **Включено (ON)** |
| **Plan** | Free (10K команд/день) |

---

## 🎯 После создания базы данных

1. **Скопируйте URL и Token:**
   - Откройте созданную базу данных
   - Перейдите в **Details** или **Settings**
   - Скопируйте:
     - `UPSTASH_REDIS_REST_URL`
     - `UPSTASH_REDIS_REST_TOKEN`

2. **Добавьте в Supabase Secrets:**
   - Supabase Dashboard → Settings → Edge Functions → Secrets
   - Добавьте:
     - `UPSTASH_REDIS_REST_URL` = ваш URL
     - `UPSTASH_REDIS_REST_TOKEN` = ваш токен

3. **Проверьте работу:**
   - См. `QUICK_START_PROTECTION.md` → Шаг 4

---

## 💡 Советы

- **Имя базы:** Можете использовать любое имя, главное чтобы было понятно
- **Регион:** Выбирайте ближайший к вашим пользователям
- **Eviction:** Обязательно включите для rate limiting
- **План:** Free плана достаточно для начала (10K команд/день)

---
