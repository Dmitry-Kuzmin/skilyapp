# 🔧 Настройка DNS записей для Zoho Mail в Vercel

## 📍 Ваш домен использует Vercel DNS

Домен `skilyapp.com` использует DNS серверы Vercel, поэтому все DNS записи настраиваются в Vercel Dashboard.

---

## 🚀 Пошаговая инструкция

### Шаг 1: Откройте Vercel Dashboard

1. Перейдите на [vercel.com](https://vercel.com)
2. Войдите в свой аккаунт
3. Выберите проект, к которому привязан домен `skilyapp.com`

### Шаг 2: Откройте настройки DNS

1. В меню проекта перейдите в **Settings**
2. Выберите **Domains** в боковом меню
3. Найдите домен `skilyapp.com` в списке
4. Нажмите на домен, чтобы открыть его настройки
5. Перейдите на вкладку **DNS Records** (или **DNS**)

### Шаг 3: Добавьте MX записи

Нужно добавить **3 MX записи**:

#### MX запись 1:
- **Тип:** `MX`
- **Имя:** `@` (или оставьте пустым)
- **Значение:** `mx.zoho.eu`
- **Приоритет:** `10`
- **TTL:** Оставьте по умолчанию или установите минимальный

#### MX запись 2:
- **Тип:** `MX`
- **Имя:** `@` (или оставьте пустым)
- **Значение:** `mx2.zoho.eu`
- **Приоритет:** `20`
- **TTL:** Оставьте по умолчанию или установите минимальный

#### MX запись 3:
- **Тип:** `MX`
- **Имя:** `@` (или оставьте пустым)
- **Значение:** `mx3.zoho.eu`
- **Приоритет:** `50`
- **TTL:** Оставьте по умолчанию или установите минимальный

**Как добавить:**
1. Нажмите кнопку **"Add Record"** или **"Add DNS Record"**
2. Выберите тип `MX`
3. Заполните поля согласно таблице выше
4. Нажмите **Save** или **Add**
5. Повторите для всех трех записей

### Шаг 4: Добавьте SPF запись (TXT)

- **Тип:** `TXT`
- **Имя:** `@` (или оставьте пустым)
- **Значение:** `v=spf1 include:zohomail.eu ~all`
- **TTL:** Оставьте по умолчанию или установите минимальный

**Как добавить:**
1. Нажмите **"Add Record"**
2. Выберите тип `TXT`
3. Заполните поля
4. Сохраните

## ⏱️ Время распространения DNS

- **Минимум:** 30 минут
- **Максимум:** до 24 часов (в зависимости от TTL)
- **Рекомендация:** Установите минимальный TTL для быстрого обновления

## 🔍 Проверка настроек

### 1. Проверка в Vercel:
- Убедитесь, что все 5 записей отображаются в списке DNS записей

### 2. Проверка через внешние инструменты:

**MX записи:**
```
https://mxtoolbox.com/SuperTool.aspx?action=mx%3askilyapp.com
```

**TXT записи (SPF и DKIM):**
```
https://mxtoolbox.com/SuperTool.aspx?action=txt%3askilyapp.com
```

**DKIM запись:**
```
https://mxtoolbox.com/SuperTool.aspx?action=txt%3azmail._domainkey.skilyapp.com
```

### 3. Проверка в Zoho:
1. Войдите в [Zoho Mail Admin Console](https://mailadmin.zoho.eu)
2. Перейдите в **Настройки** → **Сопоставление DNS**
3. Нажмите **"Проверить"** или **"Verify"**
4. Zoho проверит все записи и покажет статус

## 🐛 Решение проблем

### Проблема: Записи не отображаются после добавления

**Решение:**
- Обновите страницу в браузере
- Подождите 1-2 минуты и проверьте снова
- Убедитесь, что вы сохранили записи (нажали Save/Add)

### Проблема: Zoho не видит записи

**Решение:**
- Подождите 30-60 минут для распространения DNS
- Проверьте записи через mxtoolbox.com
- Убедитесь, что значения записей скопированы точно (без лишних пробелов)

### Проблема: Не могу найти DNS настройки в Vercel

**Решение:**
- Убедитесь, что домен добавлен в проект
- Перейдите: Settings → Domains → [ваш домен] → DNS Records
- Если не видите, возможно, домен управляется через другого провайдера
