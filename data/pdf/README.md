# 📚 PDF Учебники DGT

## Как загрузить учебники

Положи PDF файлы в эту папку с названиями:
- `tema1.pdf` — Тема 1
- `tema2.pdf` — Тема 2
- `tema3.pdf` — Тема 3
- ...
- `tema10.pdf` — Тема 10

## Затем запусти скрипт обработки:

```bash
npm run process-pdf
```

Скрипт автоматически:
1. Извлечёт текст из PDF
2. Разобьёт на разделы
3. Загрузит в Supabase таблицу `dgt_knowledge`

## Альтернатива

Если скрипт не работает, можно конвертировать PDF → текст вручную:
- Используй онлайн-конвертер (например: pdf2txt.org)
- Или утилиту `pdftotext`
- Затем загрузи через Supabase Dashboard
