## 📋 Как использовать

### Создание сезона

1. Нажми **"Создать сезон"** или **"Запланировать следующий"**
2. Заполни:
   - Номер сезона (автоматически следующий)
   - Название (RU, ES, EN)
   - Тему (зима, весна, лето, осень, специальный)
   - Даты начала и окончания (автозаполнение при планировании)
   - Описание
3. Нажми **"Сохранить"**

### Настройка призов

1. Открой карточку сезона
2. Нажми **"Управление"** в разделе "Призы лидерборда"
3. Выбери один из вариантов:

   **Вариант А: Быстрое создание по шаблонам**
   - Нажми **"Применить шаблон"** для нужной позиции (1, 2, 3, 4-10)
   - Система автоматически создаст все призы для этой позиции
   - Можно настроить после создания

   **Вариант Б: Ручное создание**
   - Нажми **"Добавить приз"** или **"Добавить"** для конкретной позиции
   - Выбери тип приза:
     - **Монеты** — укажи количество
     - **Скин/Бейдж/Рамка** — выбери из списка доступных
     - **Титул** — укажи ID титула
     - **Аура** — укажи тип ауры
   - Настрой параметры:
     - Эксклюзивный приз (да/нет)
     - Автоматически активировать (для скинов/рамок)
     - Автоматически показывать (для бейджей)
     - Описание (RU, ES)
   - Нажми **"Сохранить"**

### Редактирование призов

1. Открой редактор призов
2. Найди нужный приз в таблице
3. Нажми **✏️ Редактировать**
4. Измени параметры
5. Нажми **"Сохранить"**

### Удаление призов

1. Открой редактор призов
2. Найди нужный приз
3. Нажми **🗑️ Удалить**
4. Подтверди удаление

## 🤖 Автоматическое распределение призов

### Как это работает

1. **При завершении сезона:**
   - Триггер в БД автоматически логирует завершившийся сезон
   - Лог попадает в `cron_job_logs`

2. **При нажатии "Проверить сезоны":**
   - Система находит все завершившиеся сезоны
   - **Автоматически распределяет призы** для каждого сезона
   - Показывает результат (сколько обработано, успешно, ошибок)

3. **Статус в карточке сезона:**
   - ✅ **Зелёный** — призы распределены (показано количество)
   - 🟠 **Оранжевый** — сезон завершён, призы настроены, но не распределены
   - 🔴 **Красный** — призы не настроены

### Ручное распределение (если нужно)

1. Открой карточку завершившегося сезона
2. Нажми **"Распределить призы"**
3. Готово!

## 🎯 Типичный workflow

### Подготовка нового сезона

1. **Создай сезон:**
   - Нажми "Запланировать следующий"
   - Проверь даты (автоматически: следующий день после окончания текущего + 30 дней)
   - Заполни название и описание
   - Сохрани

2. **Настрой призы:**
   - Открой "Управление призами"
   - Нажми "Применить шаблон" для позиций 1, 2, 3, 4-10
   - При необходимости отредактируй призы
   - Готово!

3. **Активируй сезон:**
   - Сезон автоматически активен при создании
   - Если нужно деактивировать — нажми "Деактивировать"

### Завершение сезона

1. **Автоматически:**
   - Сезон завершается в указанную дату
   - Триггер логирует событие
   - При следующей проверке призы распределятся автоматически

2. **Вручную:**
   - Нажми "Проверить сезоны"
   - Система найдёт завершившиеся сезоны и распределит призы
   - Проверь статус в карточке сезона

## 💡 Полезные советы

### Шаблоны призов

Используй шаблоны для быстрого создания стандартных призов:
- **1 место** — чемпионский набор (скин, бейдж, рамка, титул, аура, 500 монет)
- **2 место** — серебряный набор (350 монет)
- **3 место** — бронзовый набор (250 монет)
- **4-10 места** — элитный набор (100 монет)

После применения шаблона можно отредактировать любой приз.

### Косметика

При выборе скина/бейджа/рамки:
- Список загружается из базы данных
- Показывается редкость каждого предмета
- Можно создать новую косметику в таблицах `skin_definitions`, `badge_definitions`

### Эксклюзивные призы

- Отметь "Эксклюзивный" для уникальных призов (например, чемпионский скин)
- Эти призы больше не появятся в будущих сезонах
- Создаёт дополнительную мотивацию для игроков

## ❓ FAQ

### Нужно ли нажимать "Распределить призы" вручную?

**Нет!** Просто нажми **"Проверить сезоны"** — система автоматически найдёт все завершившиеся сезоны и распределит призы.

### Что если призы не настроены?

Система покажет красное предупреждение: "Призы не настроены". Настрой призы через редактор, затем распредели их.

### Можно ли изменить призы после создания?

Да! Открой редактор призов и нажми ✏️ на любом призе. Измени параметры и сохрани.

### Как создать призы для всех позиций сразу?

Используй шаблоны:
1. Нажми "Применить шаблон" для позиции 1
2. Нажми "Применить шаблон" для позиции 2
3. Нажми "Применить шаблон" для позиции 3
4. Нажми "Применить шаблон" для позиций 4-10 (создаст для всех сразу)

### Что делать, если призы не распределились?

1. Проверь, что призы настроены (зелёный/оранжевый статус)
2. Проверь, что сезон действительно завершился (`end_date <= NOW()`)
3. Нажми "Проверить сезоны" ещё раз
4. Если не помогло — нажми "Распределить призы" вручную для конкретного сезона
