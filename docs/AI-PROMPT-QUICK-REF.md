**Версия:** V5

## 🚀 Быстрый старт

### Обработать один тест
```bash
node scripts/enrich-batch-v2.js data/parsed/topic-01/topic-01_test-001.json
```

### Импортировать в Supabase
```bash
node scripts/import-golden-dgt.js data/parsed/topic-01/topic-01_test-001-enriched.json
```

## 🔧 Частые правки

### Добавить новый термин в словарь

**Файл:** `scripts/enrich-batch-v2.js` (строки ~140-150)

```javascript
Новый термин ES → Новый термин RU
```

⚠️ **Важно:** Проверить на 10 вопросах перед массовой обработкой!

---

### Усилить запрет на фразу

**Файл:** `scripts/enrich-batch-v2.js` (строка ~195 или ~245)

Добавить в список:
```javascript
  • "новая запрещенная фраза"
```

---

### Изменить температуру (осторожно!)

**Файл:** `scripts/enrich-batch-v2.js` (строка ~85)

```javascript
temperature: 0.2  // Чем ниже, тем строже к инструкциям
```

**Рекомендация:** НЕ поднимать выше 0.3

## 🐛 Частые проблемы

### AI ссылается на схемы
➡️ Усилить финальную проверку (строка ~260)

### Блок РФ "водянистый"
➡️ Добавить примеры критических отличий (строка ~210)

### IDs возвращаются как 1/2/3
➡️ Проверить enum в JSON Schema (строка ~47)

---

## 📊 Быстрая проверка качества

После обработки проверь:
- [ ] Нет слов "схема", "таблиц", "рисунок" в RU
- [ ] Блок РФ только там, где критическое отличие
- [ ] Мнемотехника не "капитанская"
- [ ] IDs = a, b, c
- [ ] Ровно 30 вопросов
