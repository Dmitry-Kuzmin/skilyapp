# 👥 Друзья из Telegram групп/чатов

## ✅ Реализовано

### Как это работает:

1. **Автоматическая синхронизация:**
   - Когда пользователь отправляет сообщение боту в группе
   - Бот автоматически сохраняет его в таблицу `telegram_chat_members`
   - Обновляет `last_seen_at` при каждом взаимодействии

2. **Определение друзей:**
   - Если два пользователя состоят в одной группе
   - И оба активны (`is_active = true`)
   - Они автоматически считаются друзьями

3. **В лидерборде:**
   - Фильтр "Друзья" теперь включает:
     - Друзья из реферальной системы
     - Друзья из дуэлей
     - **Друзья из Telegram групп** ✨

---

## 📋 Структура базы данных

### Таблица `telegram_chat_members`:

```sql
CREATE TABLE telegram_chat_members (
  id UUID PRIMARY KEY,
  chat_id BIGINT NOT NULL,  -- ID группы в Telegram
  chat_type TEXT,  -- 'group', 'supergroup', 'channel'
  chat_title TEXT,  -- Название группы
  user_id UUID REFERENCES profiles(id),
  telegram_id BIGINT,  -- Telegram ID пользователя
  is_active BOOLEAN DEFAULT true,
  last_seen_at TIMESTAMPTZ,
  joined_at TIMESTAMPTZ,
  UNIQUE(chat_id, user_id)
);
```

---

## 🔧 Функции

### 1. `upsert_chat_member()`
Добавляет или обновляет участника группы:
- Если пользователь уже есть → обновляет `last_seen_at`
- Если нового → добавляет запись

### 2. `get_chat_friends(user_id)`
Возвращает список друзей из общих групп:
- Находит все группы пользователя
- Находит всех участников этих групп
- Возвращает массив `user_id`

## 🔍 Как проверить

### 1. Проверь синхронизацию участников:

```sql
-- Посмотреть всех участников групп
SELECT 
  cm.chat_id,
  cm.chat_title,
  p.first_name,
  p.telegram_id,
  cm.last_seen_at
FROM telegram_chat_members cm
JOIN profiles p ON p.id = cm.user_id
WHERE cm.is_active = true
ORDER BY cm.chat_id, cm.last_seen_at DESC;
```

### 2. Проверь друзей из групп:

```sql
-- Друзья пользователя из групп
SELECT get_chat_friends('USER_ID'::uuid);
```

### 3. Проверь в приложении:

1. Добавь бота в Telegram группу
2. Отправь сообщение боту в группе (например, `/start`)
3. Открой лидерборд → фильтр "Друзья"
4. Должны появиться участники группы, которые тоже используют приложение

## ⚠️ Важные моменты

### 1. Бот должен быть в группе
- Бот должен быть добавлен в группу
- Бот должен иметь права на чтение сообщений
- Бот должен быть активен

### 2. Пользователь должен быть авторизован
- Пользователь должен иметь профиль в приложении
- `telegram_id` должен быть привязан к профилю
- Если пользователь не авторизован → не синхронизируется

### 3. Активность
- `is_active = false` для неактивных участников
- Можно добавить автоматическое отключение после N дней неактивности
