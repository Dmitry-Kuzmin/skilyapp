# SP vs XP в Duel Pass - Объяснение

## 📊 Две системы очков

### 1. **SP (Season Points)** - Сезонные очки
- **Хранятся в:** `user_season_progress.season_points`
- **Начисляются через:** Edge Function `season-sp`
- **Используются для:** Прогресса в Duel Pass сезоне
- **Источники:**
  - Дуэли (с ставками и без)
  - Челленджи (daily, weekly, seasonal)
  - Daily login бонусы

### 2. **XP (Experience Points)** - Очки опыта
- **Хранятся в:** `profiles.duel_pass_xp`
- **Начисляются через:** Edge Function `duel-pass-xp`
- **Используются для:** Прогресса в Duel Pass (уровень)
- **Источники:**
  - Тесты
  - Дуэли
  - Daily login

---

## ❓ Почему лидерборд сортируется по XP, а не по SP?

### Текущая реализация:
Лидерборд сортируется по:
```sql
ORDER BY duel_pass_level DESC, duel_pass_xp DESC
```

### Причины:
1. **XP хранятся в `profiles`** - проще и быстрее запрашивать
2. **Уровень Duel Pass зависит от XP** - `duel_pass_level` вычисляется на основе `duel_pass_xp`
3. **SP хранятся в `user_season_progress`** - требуют JOIN с таблицей сезонов

---

## 🔄 Связь между SP и XP

### Вариант 1: SP и XP - разные системы (текущая реализация)
- SP начисляются за дуэли и челленджи
- XP начисляются за тесты и дуэли
- Они **не синхронизируются** автоматически

### Вариант 2: SP конвертируются в XP (возможная реализация)
- SP начисляются за дуэли
- SP автоматически конвертируются в XP
- Уровень Duel Pass зависит от XP

---

## 💡 Рекомендация

**Для Duel Pass лидерборда правильнее сортировать по SP**, потому что:
1. SP - это **сезонные очки**, которые показывают активность в текущем сезоне
2. SP начисляются за дуэли и челленджи - основные активности Duel Pass
3. XP могут начисляться за тесты, которые не связаны с Duel Pass

### Что нужно изменить:
1. Обновить Edge Function `duel-pass-leaderboard` для JOIN с `user_season_progress`
2. Сортировать по `season_points DESC` вместо `duel_pass_xp DESC`
3. Отображать SP вместо XP в лидерборде

---

## 📝 Текущий статус

- ✅ Лидерборд сортируется по `duel_pass_level DESC, duel_pass_xp DESC`
- ⚠️ SP хранятся отдельно в `user_season_progress.season_points`
- ⚠️ Нет автоматической синхронизации между SP и XP

**Рекомендация:** Изменить сортировку на SP для более точного отражения активности в Duel Pass сезоне.
