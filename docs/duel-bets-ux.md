# Duel Bets UX Flow

## 1. Экран создания дуэли (Host)
1. Host выбирает соперника → открывается модал `Ставка и страховка`.
2. Компоненты:
   - Слайдер/шаги для суммы монет (50–maxBet) с подсказкой уровня.
   - Тоггл “Страховка +15% (вернём 60% при поражении)”.
   - Просчёт: “Ты внесёшь 200 монет + 30 монет страховой премии. Потенциальный выигрыш: 400 монет + 45 SP.”
3. CTA `Забронировать дуэль`. Кнопка disabled, если:
   - уровень < порога;
   - баланс недостаточен;
   - превышен дневной лимит ставок.

## 2. Экран подтверждения дуэли (Opponent)
- Карточка оппонента, сумма, потенциальный выигрыш.
- Возможность выбрать свою страховку (если Host включил/выключил — это независимая настройка).
- Плашка анти-абуза: “Не более 3 ставок с этим соперником за сутки”.
- CTA `Принять ставку` / `Контр-предложить` (опционально).

## 3. Главный экран дуэли
Layout (top → bottom):
1. **Safe-area header**: Telegram back, статус “Ставка подтверждена”, иконка меню.
2. **Progress Line**:
   - Таймер слева, прогресс по центру, справа иконки bookmark/settings.
3. **Score + Bank Row**:
   - Блок счета игроков.
   - Справа карточка “Банк”: сумма, ваш вклад, статус страховки (значок щита).
4. **Question Card**:
   - Текст вопроса, иконки сложности.
   - Подсказка, если осталось <5 с.
5. **Answers Grid**:
   - Кнопки с анимацией press.
6. **Boost & Utility Row**:
   - Кнопки бустов (в линию с точными иконками).
   - Кнопка FAQ (всплывающее окно с правилами ставок и страховки).

### Состояния во время игры
- Banner events:
  - “Оппонент использовал страховку”.
  - “Банк увеличен до 1200 монет”.
  - “Подозрительная активность – ставка проверяется” (обводка жёлтого цвета).
- Если дуэль уходит в режим ожидания:
  - Блок “ход соперника” показывает прогресс и оставшееся время, плюс текущее состояние банка.

## 4. Завершение дуэли
### Победа
- Анимация “coin burst” + всплывающее окно: “+600 монет и +45 SP” и детали страховки (“страховка не понадобилась”).
- Кнопки: “Реванш”, “Поделиться”.
### Поражение
- Красный баннер: “−200 монет, страховка вернула 120 монет”.
- Ссылка на историю и совет “Попробуй страховку в следующий раз”.
### Ничья / тех. ISSUE
- Оранжевый баннер: “Возврат ставки, страховка возвращена”.
- Кнопка “Сообщить о проблеме” при тех. причине.

## 5. История и лидерборды
- Во вкладке профиля/сезона:
  - Карточки последних ставок (иконки победы/поражения, сумма, страховка, статус проверки).
  - Фильтры: все, страхованные, под review.
- Лидерборд:
  - Таблицы: “Чистый выигрыш”, “Серии побед”, “Страховые успехи”.
  - Карточка игрока → просмотр деталей дуэлей.

## 6. Уведомления
- Push/веб-уведомления:
  - “Ставка подтверждена, дуэль начнётся через 30 секунд”.
  - “Страховка вернула 180 SP”.
  - “Ставка отправлена на проверку – ожидайте до 24 часов”.

## 7. Специфические сценарии
- **Нет страховки**: показываем tooltip “Добавь страховку, чтобы вернуть 60% при поражении”.
- **Новичок**: бэйдж “Учебный режим” + сниженные лимиты, упрощённый UI без контр-предложений.
- **High Roller**: при ставке ≥ 600 монет активируем спец-анимацию и подсказку о рисках.

## 8. Точки анимаций
- Вход Boost Row (slide-up).
- Обновление банка (count-up + glow).
- Срабатывание страховки (щит закрывает ставку, затем частичный возврат).
- Победа/поражение (coin rain / shattered chips).

Док служит базовым гайдом для фронтенд-реализации и согласования дизайна.
