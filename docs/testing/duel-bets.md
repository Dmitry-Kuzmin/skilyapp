# Тестирование системы ставок и страховки

## 1. Создание дуэли
- [x] Создать дуэль без ставки — проверено, монеты не списываются, страховка не доступна.
- [x] Создать дуэль со ставкой 200 монет и включённой страховкой — списываются 230 монет, в `duel_bets` фиксируется премия и статус `pending`.
- [x] При отмене дуэли хосту возвращается ставка и страховая премия, транзакции записаны как `refund` и `insurance_refund`.

## 2. Присоединение к дуэли
- [x] Попытка присоединиться без достаточного баланса блокируется и показывает точную сумму с учётом страховки.
- [x] Успешное присоединение со страховкой — списывается ставка+премия, в `duel_bets` заполняются поля оппонента и статус `confirmed`.
- [x] Переключатель страховки на экране присоединения обновляет требуемую сумму и блокируется при недостатке монет.

## 3. Проведение боя
- [x] Во время дуэли в `DuelBattleFullscreen` отображается блок банка: общая сумма, сезонный бонус, статусы страховки.
- [x] Таймер на прогресс-баре работает штатно, отступы в Telegram корректны после обновления safe-area.

## 4. Завершение и выплаты
- [x] Победа без страховки: победитель получает `(банк - 10%)`, проигравший не получает возврат, в `duel_transactions` пишутся `win` и `commission`.
- [x] Победа со страховкой у проигравшего: начисляется `coverageRate * ставка`, фиксируется `insurance_refund`.
- [x] Ничья: обоим возвращается ставка и премии, статус дуэли = `finished`, в истории записан `draw`.
- [x] Значения `season_sp_host/opponent` сохраняются в `duel_bets` и используются для аналитики/лидерборда.

## 5. Season Points и Duel Pass XP
- [x] `season-sp` выдаёт 30 SP за победу без ставки, 80 SP за ставку 600+, 5 SP за поражение (независимо от ставки), 15 SP за ничью.
- [x] `duel-pass-xp` начисляет 40 XP за победу со ставкой, 30 XP без ставки, 25 XP за ничью.
- [x] Клиент (`DuelResult`) не инициирует начисление монет — всё делает серверная логика (SP/XR + челленджи остаются).

## 6. Лидерборд
- [x] Edge Function `duel-leaderboard` агрегирует данные по всем пользователям, сортирует по прибыли и SP.
- [x] Новая страница `/duel-leaderboard` отображает карточки и таблицу лидеров, показывает сумму страховых возвратов и статистику W/L/D.
- [x] При отсутствии данных выводится заглушка “Пока нет дуэлей со ставками”.

## 7. Документация и /help
- [x] `docs/duel-bets-spec.md` и `duel-bets-ux.md` обновлены под новую экономику.
- [x] Страница `/help` содержит новый раздел про ставки/страховку и обновлённые разделы про монеты.

## 8. Регрессия
- [x] Обычные дуэли без ставок проходят как раньше.
- [x] Boost’ы, таймеры, переход к ожиданию/результатам работают как в предыдущих фикcах переходов.

> Все проверки проводились на локальном окружении с обновлённой базой (новые миграции и функции). При деплое необходимо прогнать `supabase db push` и обновить edge functions `duel-manager` и `duel-leaderboard`.
