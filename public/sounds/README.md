# Звуковые эффекты для Digital Cockpit

Эта папка содержит звуковые эффекты для интерфейса кокпита.

## Необходимые файлы:

- `startup.mp3` - звук запуска системы (System Boot Screen)
- `click.mp3` - звук клика (Ignition Button, модули)

## Рекомендуемые характеристики:

- Формат: MP3 или WAV
- Качество: 44.1kHz, 16-bit или выше
- Громкость: Нормализованная до ~-3dB
- Длительность: 
  - startup.mp3: 1-3 секунды
  - click.mp3: 0.1-0.3 секунды

## Если файлы отсутствуют:

Приложение будет работать без звуков - код автоматически обрабатывает отсутствие файлов.

## Примеры генерации звуков:

Можно использовать бесплатные библиотеки звуков:
- https://freesound.org/
- https://mixkit.co/
- https://pixabay.com/music/

Или создать звуки самостоятельно с помощью:
- Audacity (бесплатно)
- GarageBand (Mac)
- Online генераторы звуков
