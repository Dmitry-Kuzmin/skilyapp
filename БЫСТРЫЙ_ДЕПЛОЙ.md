# ⚡ БЫСТРАЯ ИНСТРУКЦИЯ: Задеплоить функцию

## 🎯 Способ 1: Через Dashboard (РЕКОМЕНДУЮ - самый простой!)

### 1. Открой Supabase Dashboard
👉 https://supabase.com/dashboard → выбери проект

### 2. Открой Edge Functions
👉 В меню слева: **Edge Functions** → найди **ai-chat**

### 3. Открой код функции
👉 Нажми на функцию **ai-chat** → найди кнопку **"Edit"** или **"View Code"**

### 4. Скопируй код
👉 Открой файл **`AI_CHAT_CODE.txt`** в проекте
👉 Скопируй ВЕСЬ код (Ctrl+A, Ctrl+C)

### 5. Вставь код
👉 В Supabase Dashboard выдели весь старый код
👉 Вставь новый код (Ctrl+V)

### 6. Задеплой
👉 Нажми кнопку **"Deploy"** или **"Save"**
👉 Дождись сообщения об успехе

### 7. Проверь
👉 Убедись, что секрет `GROQ_API_KEY` добавлен в **Settings** → **Secrets**
👉 Открой приложение и проверь работу AI помощника


### 2. Авторизуйся
```bash
supabase login
```

### 3. Свяжи проект
```bash
# Найди project-ref в настройках проекта в Supabase Dashboard
supabase link --project-ref ваш-project-ref
```

### 4. Задеплой функцию
```bash
supabase functions deploy ai-chat
```

## ❓ Проблемы?

### "Function not found"
→ Создай новую функцию в Dashboard: **Create Function** → название `ai-chat`

### "Deploy failed"
→ Проверь, что код полностью скопирован
→ Посмотри ошибки в консоли браузера (F12)

### "AI service temporarily unavailable"
→ Проверь, что секрет `GROQ_API_KEY` добавлен в Supabase Secrets
→ Название должно быть точно `GROQ_API_KEY`

### CLI не устанавливается
→ Используй Способ 1 (через Dashboard) - он проще!
