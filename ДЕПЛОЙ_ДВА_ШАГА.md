## ШАГ 1: Обнови функцию поиска в базе

### 1. Открой Supabase Dashboard
https://supabase.com/dashboard

### 2. Перейди в SQL Editor
Левое меню → **SQL Editor** (иконка </>)

### 3. Скопируй миграцию
Открой файл:
```
supabase/migrations/20251110_update_search_function.sql
```

Скопируй **весь код** (Cmd+A → Cmd+C)

### 4. Выполни в SQL Editor
- Вставь код в редактор (Cmd+V)
- Нажми **"Run"** или **F5**
- Дождись "Success"

**Зачем:** Функция поиска теперь работает с **русским И испанским** языками!

---

## ШАГ 2: Задеплой Edge Function

### 1. Перейди в Edge Functions
**Supabase** → **Edge Functions** → **ai-chat** → вкладка **"Code"**

### 2. Скопируй обновлённый код
Открой файл:
```
AI_CHAT_FINAL.txt
```

Скопируй **весь код** (Cmd+A → Cmd+C)

### 3. Замени код в Supabase
- В редакторе: **Cmd+A** (выделить всё)
- Вставь: **Cmd+V**
- Нажми **"Deploy updates"**

**Зачем:** 
- Извлечение ключевых терминов (estrechamiento, prioridad)
- Явная инструкция использовать учебник
- Улучшенный промпт (5 блоков)
- Меньше ошибок (temperature 0.4)
