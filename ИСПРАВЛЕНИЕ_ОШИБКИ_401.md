# 🔧 Исправление ошибки 401 Unauthorized

## Проблема

Ошибка `401 Unauthorized` при запросе к AI chat функции возникала потому, что:
1. Код использовал ANON ключ вместо токена пользователя
2. Функция требовала обязательную аутентификацию пользователя
3. Но в конфигурации было `verify_jwt = false`, что создавало конфликт

## Решение

### 1. Обновлён AISearchWidget.tsx

**Изменения:**
- Добавлен импорт `supabase` клиента
- Получение токена сессии пользователя
- Fallback на ANON ключ, если сессии нет
- Улучшена обработка ошибок

**Код:**
```typescript
// Get user session token for authentication (optional)
const { data: { session } } = await supabase.auth.getSession();

// Use session token if available, otherwise use ANON key
const authToken = session?.access_token || import.meta.env.VITE_SUPABASE_PUBLISHABLE_KEY;
```

### 2. Обновлена функция ai-chat/index.ts

**Изменения:**
- Проверка пользователя стала опциональной
- Функция работает как с аутентификацией, так и без неё
- Улучшено логирование

**Код:**
```typescript
// Try to get user if auth header is provided
if (authHeader) {
  // Check user authentication (optional)
  // Continue even if auth fails
}
```

## Что нужно сделать

### Шаг 1: Задеплой обновлённую функцию

1. Открой **Supabase Dashboard**
2. Перейди в **Edge Functions** → **ai-chat**
3. Скопируй обновлённый код из `supabase/functions/ai-chat/index.ts`
4. Вставь в редактор функции
5. Нажми **"Deploy"**

### Шаг 2: Обнови приложение

Код в `src/components/AISearchWidget.tsx` уже обновлён. Если приложение запущено:
- Перезагрузи страницу (Ctrl+R или Cmd+R)
- Или перезапусти dev server: `npm run dev`

### Шаг 3: Проверь работу

1. Открой приложение в браузере
2. Задай вопрос AI помощнику
3. Должен прийти ответ без ошибок

## Проверка

### В консоли браузера должно быть:
- ✅ Нет ошибок `401 Unauthorized`
- ✅ Запросы проходят успешно

### В логах Supabase должно быть:
```
[AI Chat] Authenticated user: <user_id>
или
[AI Chat] No auth header provided, continuing without auth
[AI Chat] Trying Groq API...
[AI Chat] ✅ Groq API success
```

## Если ошибка сохраняется

### Проблема: Всё ещё 401 ошибка

**Решение:**
1. Проверь, что функция задеплоена
2. Проверь, что код обновлён в обоих местах:
   - `src/components/AISearchWidget.tsx`
   - `supabase/functions/ai-chat/index.ts`
3. Перезагрузи страницу
4. Проверь логи в Supabase Dashboard

### Проблема: "User not authenticated"

**Решение:**
- Это нормально, функция работает и без аутентификации
- Но лучше войти в систему для лучшей работы

### Проблема: Медленные ответы

**Решение:**
1. Проверь, что Groq API ключ добавлен
2. Проверь логи в Supabase
3. Убедись, что используется Groq (не Lovable API)

## Готово! 🎉

После исправления AI помощник должен работать без ошибок 401!
