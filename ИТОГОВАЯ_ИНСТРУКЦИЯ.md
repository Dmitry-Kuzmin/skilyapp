### Шаг 1: Обнови код функции ai-chat в Supabase

1. Открой **Supabase Dashboard**: https://supabase.com/dashboard
2. Выбери проект **sdadim-dgt-prep**
3. Перейди в **Edge Functions** → **ai-chat** → **Code**
4. **Выдели весь старый код** (Ctrl+A или Cmd+A)
5. **Скопируй код** из файла `AI_CHAT_CODE.txt` (в проекте)
6. **Вставь новый код** (Ctrl+V или Cmd+V)
7. Нажми кнопку **"Deploy updates"** (зелёная кнопка внизу справа)
8. Дождись успешного деплоя

## Если ошибки сохраняются

### Вариант 1: Добавь Gemini API (лучше для русского)
1. Получи ключ: https://aistudio.google.com/apikey
2. Добавь секрет `LOVABLE_API_KEY` в Supabase Secrets
3. Система автоматически переключится на Gemini

### Вариант 2: Сниз температуру
В коде замени:
```typescript
temperature: 0.3,  // Было 0.7
```

### Вариант 3: Используй только Gemini
Если Gemini работает лучше, можно изменить приоритет:
```typescript
// Сначала попробовать Lovable (Gemini), потом Groq
const geminiResponse = await tryLovable(messages);
if (geminiResponse) return geminiResponse;

const groqResponse = await tryGroqWithFallback(messages);
if (groqResponse) return groqResponse;
```
