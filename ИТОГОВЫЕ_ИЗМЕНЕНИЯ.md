### ✅ 4. Меньше "воды"

## Что нужно сделать

### Шаг 1: Обнови код в Supabase

1. Supabase Dashboard → Edge Functions → ai-chat → Code
2. Скопируй код из `AI_CHAT_CODE.txt`
3. Вставь в редактор (замени весь код)
4. Нажми "Deploy updates"

### Шаг 2: Проверь работу

1. Перезагрузи приложение
2. Пройди тест в режиме Практика
3. Нажми "Спросить AI"
4. Попробуй:
   - Автоматическое объяснение
   - Быстрые кнопки
   - Задай свой вопрос в чате
5. Проверь, что AI говорит про **Испанию**, не "некоторые страны"

## Про PDF учебники

Есть 3 варианта загрузки PDF:

### 🚀 Вариант 1: Быстрый (сегодня)
Добавить ключевые факты прямо в промпт:

- Штрафы в евро

- **Готов помочь сформировать**

### 🎯 Вариант 2: Средний (1-2 дня)
Загрузить тексты PDF в таблицу Supabase:
- Конвертируем PDF → текст
- Загружаем в базу по темам
- AI ищет нужную тему
- **Могу настроить автоматически**

### 🔥 Вариант 3: Продвинутый (неделя)
Полноценный RAG с векторами:
- PDF → чанки → embeddings → векторная база
- Семантический поиск по контексту
- Самый точный и умный
- **Настрою пошагово**
