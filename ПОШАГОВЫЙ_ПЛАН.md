# 📋 ПОШАГОВЫЙ ПЛАН: Загрузка PDF учебников

## Что делаем

Загружаем 10 PDF учебников DGT в базу данных, чтобы AI использовал их для точных ответов.

---

## ✅ ШАГ 1: Применить миграцию (создать таблицу)

### Через Supabase Dashboard:

1. Открой **Supabase Dashboard**: https://supabase.com/dashboard
2. Выбери проект **sdadim-dgt-prep**
3. Перейди в **SQL Editor** → **New query**
4. Скопируй содержимое файла `supabase/migrations/20251110_create_dgt_knowledge.sql`
5. Вставь в редактор
6. Нажми **"Run"** (или Ctrl+Enter)
7. Дождись сообщения об успехе

### Проверка:
- Table Editor → должна появиться таблица `dgt_knowledge`

---

## ✅ ШАГ 2: Подготовить PDF файлы

### 2.1. Положи PDF в папку `data/pdf/`

Файлы должны называться:
- `tema1.pdf`
- `tema2.pdf`
- `tema3.pdf`
- ...
- `tema10.pdf`

### 2.2. Проверь структуру

```
/Users/dimka/Desktop/Sdadim/sdadim-dgt-prep/
  data/
    pdf/
      tema1.pdf
      tema2.pdf
      ...
      tema10.pdf
```

---

## ✅ ШАГ 3: Установить зависимости

```bash
npm install pdf-parse
```

Эта библиотека нужна для извлечения текста из PDF.

---

## ✅ ШАГ 4: Получить Service Key

1. Supabase Dashboard → Settings → API
2. Найди **service_role key** (secret)
3. Скопируй ключ
4. Создай файл `.env` в корне проекта:
   ```
   SUPABASE_SERVICE_KEY=твой_service_key_здесь
   ```

⚠️ **ВАЖНО**: Не коммить `.env` в Git!

---

## ✅ ШАГ 5: Запустить скрипт обработки PDF

```bash
npm run process-pdf
```

### Что произойдёт:
1. Скрипт прочитает каждый PDF
2. Извлечёт текст
3. Разобьёт на разделы
4. Загрузит в таблицу `dgt_knowledge`

### Ожидаемый результат:
```
🚀 Начинаем обработку PDF учебников DGT
📚 Найдено PDF файлов: 10
📄 Обработка: tema1.pdf
✅ Извлечено 15234 символов
✅ Создано 8 разделов
✅ Загружено 8 разделов из 8
...
✅ ГОТОВО! Загружено 75 разделов в базу данных
```

---

## ✅ ШАГ 6: Проверить данные в Supabase

1. Supabase Dashboard → Table Editor → dgt_knowledge
2. Должны быть записи:
   - topic_number: 1, 2, 3...
   - section_title: названия разделов
   - content: текст из PDF
   - keywords: ключевые слова

## ❓ Возможные проблемы

### Проблема: PDF не читается
**Решение:** Конвертируй PDF → текст вручную и загрузи через Dashboard

### Проблема: Скрипт падает
**Решение:** Проверь логи, возможно нужно настроить парсинг под твой формат PDF

### Проблема: Данные не загружаются
**Решение:** Проверь `SUPABASE_SERVICE_KEY` в `.env`
