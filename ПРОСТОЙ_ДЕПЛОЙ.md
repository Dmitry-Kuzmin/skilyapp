# 🚀 ПРОСТАЯ инструкция: Как задеплоить функцию через Dashboard

## ✅ Способ через Dashboard (САМЫЙ ПРОСТОЙ!)

### Шаг 1: Открой Supabase Dashboard
1. Перейди на: **https://supabase.com/dashboard**
2. Войди в свой аккаунт
3. Выбери проект **sdadim-dgt-prep**

### Шаг 2: Открой Edge Functions
1. В меню слева найди **"Edge Functions"** (иконка молнии ⚡)
2. Нажми на него

### Шаг 3: Открой функцию ai-chat
1. В списке функций найди **"ai-chat"**
2. Нажми на неё

### Шаг 4: Обнови код функции

**Вариант А: Если есть кнопка "Edit" или "Edit Function"**
1. Нажми кнопку **"Edit"** или **"Edit Function"**
2. Скопируй ВЕСЬ код из файла `supabase/functions/ai-chat/index.ts`
3. Вставь код в редактор (замени весь старый код)
4. Нажми **"Deploy"** или **"Save"**

**Вариант Б: Если нужно создать функцию заново**
1. Нажми кнопку **"Create Function"** или **"New Function"**
2. Название: `ai-chat`
3. Скопируй ВЕСЬ код из файла `supabase/functions/ai-chat/index.ts`
4. Вставь в редактор
5. Нажми **"Deploy"** или **"Create"**

### Шаг 5: Проверь что секрет добавлен
1. Убедись, что секрет `GROQ_API_KEY` добавлен в **Settings** → **Edge Functions** → **Secrets**
2. Если нет - добавь его (см. инструкцию выше)

### Шаг 6: Проверь работу
1. Открой приложение
2. Задай вопрос AI помощнику
3. Должен ответить быстро!
4. Проверь логи: в функции ai-chat нажми **"Logs"**

## ❓ Проблемы?

### "Function not found" или функции ai-chat нет в списке
**Решение**: Создай новую функцию:
1. Нажми **"Create Function"** или **"New Function"**
2. Название: `ai-chat`
3. Вставь код
4. Нажми **"Deploy"**

### "Deploy failed" или ошибка при деплое
**Решение**:
1. Проверь, что код полностью скопирован
2. Убедись, что нет лишних символов
3. Посмотри ошибки в консоли браузера (F12)

### "Can't find Edit button"
**Решение**: 
- Некоторые проекты не позволяют редактировать через Dashboard
- В этом случае нужно установить CLI (см. альтернативный способ ниже)

---

## 🎯 Альтернативный способ: Установка CLI

Если Dashboard не работает, установи CLI:

### macOS (через Homebrew):
```bash
# Сначала установи Homebrew (если нет):
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Потом установи Supabase CLI:
brew install supabase/tap/supabase
```

### Или скачай готовый бинарник:
1. Перейди на: https://github.com/supabase/cli/releases
2. Скачай файл для macOS (например: `supabase_darwin_amd64.tar.gz`)
3. Распакуй
4. Перемести в `/usr/local/bin/`:
   ```bash
   sudo mv supabase /usr/local/bin/
   ```

### После установки CLI:
```bash
# Авторизуйся
supabase login

# Свяжи проект (project-ref найди в настройках проекта)
supabase link --project-ref ваш-project-ref

# Задеплой функцию
supabase functions deploy ai-chat
```
