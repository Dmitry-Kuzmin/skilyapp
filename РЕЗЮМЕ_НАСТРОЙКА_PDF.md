## Что делать СЕЙЧАС

### Вариант А: С PDF файлами (если есть)

1. **Положи PDF** в `data/pdf/` (tema1.pdf, tema2.pdf, ...)
2. **Примени миграцию** (SQL из файла в Supabase Dashboard)
3. **Установи библиотеку**: `npm install pdf-parse`
4. **Получи Service Key** (Supabase Dashboard → Settings → API)
5. **Создай `.env`** с ключом
6. **Запусти**: `npm run process-pdf`
7. **Обнови функцию** (код из `AI_CHAT_WITH_KNOWLEDGE.txt`)

Подробно: `БЫСТРЫЙ_СТАРТ_PDF.md`

### Вариант Б: Без PDF (на потом)

1. **Обнови промпт сейчас** (код из `AI_CHAT_CODE.txt`)
   - AI уже знает, что это про Испанию
   - Не будет писать "в некоторых странах"
2. **Загрузишь PDF позже** когда будут готовы
