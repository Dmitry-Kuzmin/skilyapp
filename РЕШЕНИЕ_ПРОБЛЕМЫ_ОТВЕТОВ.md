# Решение проблемы с ответами на вопросы

## Проблема:
Скрипт переключается между вопросами, но вопрос не меняется после клика на кнопку навигации.

## Причина:
На сайте PracticaVial нужно **обязательно ответить на вопрос** перед переходом к следующему. Без ответа кнопка навигации может быть неактивна или не работать.

## Решение:

### 1. Автоматический выбор ответа
Скрипт теперь **автоматически выбирает первый ответ (A)** перед каждым переключением:
- На первом вопросе
- Перед переходом к следующему вопросу
- Если вопрос не изменился после клика

### 2. Множественные способы выбора ответа
Скрипт пробует несколько способов:
1. **Клик по радио-кнопке** через Puppeteer
2. **JavaScript выбор** (`radio.checked = true` + события)
3. **Проверка выбора** - убеждается, что ответ действительно выбран

### 3. Множественные способы навигации
Если кнопка неактивна, скрипт пробует:
1. **Клик по кнопке навигации** через Puppeteer
2. **JavaScript клик** по кнопке
3. **Клавиши стрелок** (`ArrowRight`) для навигации

### 4. Увеличенные таймауты
- После выбора ответа: **1.5 секунды**
- После клика по кнопке: **4 секунды**
- После навигации через клавиши: **2-3 секунды**

## Что вы увидите в терминале:

```
✅ Вопрос 1: 1 La velocidad máxima...
💡 Выбираю первый ответ на первом вопросе...
✅ Ответ выбран
💡 Выбираю первый ответ перед переходом...
✅ Ответ выбран
🔘 Нажимаю кнопку навигации...
✅ Вопрос 2: 2 En una vía...
```

## Если проблема сохраняется:

1. **Проверьте браузер** - убедитесь, что вопросы действительно переключаются вручную
2. **Проверьте логи** - посмотрите, что именно происходит
3. **Увеличьте таймауты** - возможно, нужно больше времени для загрузки
4. **Проверьте структуру HTML** - возможно, структура страницы изменилась

## Отладка:

Если скрипт не работает:
1. Откройте DevTools в браузере (F12)
2. Проверьте, есть ли радио-кнопки: `document.querySelectorAll('input[type="radio"]')`
3. Проверьте, есть ли кнопка навигации: `document.querySelectorAll('button, a')`
4. Попробуйте вручную выбрать ответ и переключиться
5. Посмотрите, какие события срабатывают при переключении
