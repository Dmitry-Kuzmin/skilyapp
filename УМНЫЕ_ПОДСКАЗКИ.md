## 🚀 Как это работает

1. Пользователь задаёт вопрос
2. AI даёт основное объяснение
3. **Автоматически** запрашиваются умные подсказки
4. AI анализирует контекст и генерирует 3-4 релевантных вопроса
5. Подсказки появляются с красивой анимацией
6. Пользователь кликает на подсказку → AI расширяет ответ

## 📊 Технические детали

### Добавлено:
- `generateSmartSuggestions()` — функция генерации подсказок
- `smartSuggestions` — состояние для хранения подсказок
- `isGeneratingSuggestions` — индикатор загрузки
- Автоматический вызов после первого ответа AI

### Промпт для AI:
```
На основе этого вопроса, предложи 3-4 КРАТКИХ вопроса-подсказки 
(не более 5-7 слов каждая), которые помогут пользователю лучше понять тему.

Примеры хороших подсказок:
- Что если нарушу это правило?
- Какой штраф за это?
- Где ещё применяется?
- В чём суть этого правила?
```

### Парсинг ответа:
- AI возвращает список
- Код парсит строки (удаляет нумерацию)
- Фильтрует по длине (< 50 символов)
- Берёт первые 4 подсказки
