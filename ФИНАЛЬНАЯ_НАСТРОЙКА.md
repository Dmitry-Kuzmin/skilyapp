## ЧТО СДЕЛАТЬ СЕЙЧАС (2 шага)

### Шаг 1: Получить Gemini API ключ

1. Открой: **https://aistudio.google.com/apikey**
2. Войди через Google
3. Нажми **"Create API Key"**
4. **Скопируй ключ** (начинается с `AIza...`)

---

### Шаг 2: Добавить ключи в Supabase

**Supabase Dashboard** → **Settings** → **Edge Functions** → **Secrets**

Добавь **ДВА** секрета:

#### Секрет 1: GEMINI_API_KEY
- **Name**: `GEMINI_API_KEY`
- **Value**: ключ из шага 1
- **Нажми "Save"**

#### Секрет 2: GROQ_API_KEY (если ещё не добавлен)
- **Name**: `GROQ_API_KEY`
- **Value**: ключ из Groq Console
- **Нажми "Save"**

---

### Шаг 3: Обновить Edge Function

**Supabase Dashboard** → **Edge Functions** → **ai-chat** → **Code**

**Скопируй код из файла:**
- `AI_CHAT_WITH_KNOWLEDGE.txt` — с учебниками (рекомендую)
- `AI_CHAT_CODE.txt` — без учебников

**Нажми** "Deploy updates"
