### 2. 🖼️ Умная работа с изображениями

**Добавлено:**
```
🖼️ ПРО ИЗОБРАЖЕНИЯ:
- Упоминай элементы изображения ТОЛЬКО когда это помогает
- Например: "На изображении видны знаки R-5 и R-6"
- НЕ описывай очевидное: "на картинке дорога"
- Используй для УСИЛЕНИЯ объяснения
```

**Как работает:**
- Если есть картинка → AI получает подсказку: "📷 К вопросу есть изображение"
- AI **сам решает** упоминать его или нет
- Упоминает **только если полезно** для объяснения

**Примеры:**

**Вопрос про estrechamiento с картинкой:**
AI: "На изображении ты видишь автобус и легковую с прицепом в узком проходе. Согласно иерархии транспорта, автобус имеет приоритет..."
✅ Полезно!

**Вопрос про скорость с обычной дорогой:**
AI: "Максимальная скорость на autovía — 100 км/ч..."
✅ Не упоминает картинку — она не добавляет информации.
