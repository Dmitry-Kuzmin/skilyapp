# 🚀 Что делать прямо сейчас (для новичков)

## Проблема

Скрипт находит тесты, но не может извлечь из них вопросы. Это происходит потому, что структура страницы может отличаться от ожидаемой.

## Решение

Давайте сделаем более простой подход - скрипт будет извлекать вопросы напрямую из текста страницы.

## 📝 Пошаговая инструкция:

### 1. Откройте терминал в VS Code

Нажмите `Ctrl + `` (обратная кавычка) или выберите "Терминал" → "Новый терминал" в меню.

### 2. Перейдите в папку проекта

В терминале введите:

```bash
cd /Users/dimka/Desktop/Sdadim/sdadim-dgt-prep
npm run scrape:simple -- --topics=1
kuzmin.public@gmail.com
yVSnp
```

Нажмите Enter.

### 4. Дождитесь открытия браузера

Скрипт откроет браузер Chrome автоматически. Подождите, пока страница загрузится.

### 5. Авторизуйтесь в браузере

1. В браузере введите ваш логин (email)
2. Введите пароль
3. Нажмите "Iniciar sesión"

### 6. Перейдите на страницу тестов

В адресной строке браузера введите:

```
https://teorica.practicavial.com/permiso/b/tests/tema/1
```

Или найдите в меню "Test por temas" → выберите тему 1.

Вы должны увидеть список тестов (Test Tema 1 Nº: 001, 002, 003, и т.д.)

### 7. Вернитесь в терминал и нажмите Enter

1. Переключитесь на окно терминала (или VS Code)
2. **Нажмите Enter** в терминале
3. Скрипт начнет работу

### 8. Наблюдайте за процессом

В терминале вы увидите:
- Какие тесты обрабатываются
- Сколько вопросов найдено в каждом тесте
- Общий прогресс

### 9. Если вопросы не находятся

Если вы видите "Найдено вопросов: 0", это означает, что скрипт не может извлечь вопросы из страницы.

**Что сделать:**
1. Откройте один из тестов в браузере вручную (например, Test Tema 1 Nº: 001)
2. Посмотрите, как выглядит страница с вопросом
3. Сделайте скриншот страницы
4. Пришлите скриншот - я помогу улучшить скрипт

## 🔍 Как проверить, что скрипт работает:

1. **Скрипт должен найти тесты:** Вы увидите "✅ Найдено тестов: 17" (или другое число)
2. **Скрипт должен открыть каждый тест:** Вы увидите "[1/17] Тест 1", "[2/17] Тест 2", и т.д.
3. **Скрипт должен найти вопросы:** Вы увидите "✅ Найдено вопросов: X" (где X > 0)

## ❓ Что делать, если ничего не работает:

1. **Проверьте авторизацию:** Убедитесь, что вы авторизованы в браузере
2. **Проверьте URL:** Убедитесь, что вы на странице `/permiso/b/tests/tema/1`
3. **Проверьте список тестов:** Вы должны видеть список тестов на странице
4. **Проверьте консоль браузера:** Откройте DevTools (F12) и посмотрите, нет ли ошибок

## 💡 Важные моменты:

- **Не закрывайте браузер** - скрипт использует его для работы
- **Не закрывайте терминал** - там видны все сообщения
- **Следите за сообщениями** - они подскажут, что делать дальше
- **Если скрипт завис** - нажмите `Ctrl + C` в терминале, чтобы остановить

## 🎯 Следующие шаги:

После того, как скрипт завершит работу:

1. Проверьте файл `practicavial-questions.xlsx` - там должны быть вопросы
2. Проверьте базу данных - вопросы должны быть сохранены
3. Если вопросы не найдены - пришлите скриншот страницы с вопросом

2. Сделайте скриншот страницы с вопросом в браузере
3. Пришлите оба - я помогу разобраться
