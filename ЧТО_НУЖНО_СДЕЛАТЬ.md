# 🚀 Что нужно сделать для настройки Groq AI

## ✅ Шаг 1: Получить API ключ Groq (5 минут)

1. **Открой сайт**: https://console.groq.com
2. **Зарегистрируйся** через Google, GitHub или email
3. **Перейди в раздел "API Keys"** (в меню слева)
4. **Нажми "Create API Key"**
5. **Придумай название** (например: "Sdadim DGT App")
6. **СКОПИРУЙ ключ** (он выглядит так: `gsk_xxxxxxxxxxxxx`)
   - ⚠️ **ВАЖНО**: Ключ показывается только один раз! Сохрани его сразу!

---

## ✅ Шаг 2: Добавить ключ в Supabase (3 минуты)

1. **Открой Supabase Dashboard**: https://supabase.com/dashboard
2. **Выбери свой проект** (sdadim-dgt-prep)
3. **Перейди в Settings** → **Edge Functions** → **Secrets**
4. **Нажми "Add a new secret"**
5. **Заполни форму**:
   - **Name**: `GROQ_API_KEY` (точное название, без пробелов!)
   - **Value**: вставь скопированный ключ из Groq
6. **Нажми "Save"**

---

## ✅ Шаг 3: Деплой Edge Function (автоматически)

Код уже обновлён! Теперь нужно задеплоить функцию:

### Вариант А: Через Supabase CLI (если установлен)

```bash
# В терминале в папке проекта
supabase functions deploy ai-chat
```

### Вариант Б: Через Supabase Dashboard (проще)

1. Открой Supabase Dashboard
2. Перейди в **Edge Functions** → **ai-chat**
3. Нажми **"Redeploy"** или **"Deploy"**
4. Если нужно - обнови код вручную, скопировав содержимое файла `supabase/functions/ai-chat/index.ts`

## ✅ Шаг 4: Проверка работы

## ❓ Проблемы?

### Проблема: "AI service temporarily unavailable"

**Решение**:
1. Проверь, что ключ правильно добавлен в Supabase Secrets
2. Название должно быть точно `GROQ_API_KEY` (без пробелов, заглавными буквами)
3. Перезапусти Edge Function (redeploy)

### Проблема: "Groq API error"

**Решение**:
1. Проверь, что ключ активен в Groq Console
2. Проверь лимиты (бесплатно: 14,400 запросов/день)
3. Система автоматически переключится на Lovable API

### Проблема: Медленные ответы

**Решение**:
- Groq должен работать очень быстро (< 1 сек)
- Если медленно - проверь интернет-соединение
- Посмотри логи в Supabase Dashboard
